        虽然，童筱颖在念大学期间有勤工俭学，在双休节假寒暑假日帮着傅昼景在他所在的公司里打工赚钱，每个学期都有拿奖学金，但是两人开销所需的钱大部分还是来自于傅昼景的工资。
<br/>
<br/>        所以，开销太大与相处时间太少，这才是童筱颖不愿意再念硕士学位的根本原因。
<br/>
<br/>        “嗯，我们是该结束这‘  Love（柏拉图式精神恋爱）’了！”傅昼景意味深长地应道。
<br/>
<br/>        顿时，童筱颖羞涩地红了脸。
<br/>
<br/>        而和傅昼景的通话结束后，家里的门铃响了。
<br/>
<br/>        童筱颖放下了手中的水果沙拉盘，只身朝大门走去，她先是看了一眼猫眼，发现是熟人后，心里冷不丁地打了个寒战，若有所思地开了门。
<br/>
<br/>        “筱颖，好久不见。”
<br/>
<br/>        季思妍穿着一件墨紫色连衣裙，身形高挑纤细，一手拖着行李箱，臂弯上还挂着一个手提包，另一只手则牵着一个戴着鸭舌帽，穿着牛仔背带裤，苏格兰风格子衬衫的小男孩。
<br/>
<br/>        小男孩抬起头来，看向童筱颖，非常有礼貌地问候道：“阿姨，好！”
<br/>
<br/>        而就在这个小男孩抬起头来的一瞬，童筱颖看清了这小家伙的样貌。
<br/>
<br/>        小男孩那小巧又圆嘟的五官，犹如粉雕玉琢般可爱，乌黑的大眼睛，高挺的鼻子，粉嘟嘟的小嘴儿，除了皮肤过于苍白，显得有些病态以外，这孩子跟傅昼景还真有几分神似。
<br/>
<br/>        “你们……”童筱颖欲言又止，完全无法理解，季思妍为什么会带着一个看起来才两岁左右的孩子突然出现在自己面前。
<br/>
<br/>        “筱颖，可以让我们进屋谈谈吗？”季思妍声音深沉而哽咽。
<br/>
<br/>        童筱颖抿了抿唇，又低头看了一眼季思妍身边的这个小男孩一眼后，让了道，向季思妍和这小家伙做了一个“迎进”的手势。
<br/>
<br/>        季思妍和那小男孩一大一小，端端正正地坐在了客厅的沙发上。
<br/>
<br/>        小男孩出奇的安静，不像同龄的孩子那般又吵又闹。
<br/>
<br/>        童筱颖重新切了水果拼盘，又给季思妍泡了茶，还很贴心地给小男孩倒了一杯鲜牛乳。
<br/>
<br/>        季思妍随之从自己的手提包里，拿出了一叠资料，递给了童筱颖。
<br/>
<br/>        童筱颖犹豫了一下，才接过季思妍递过来的资料，心里带着疑惑细细地翻阅了起来。
<br/>
<br/>        资料里，从第一张孕检B超单，到孩子现在的身体状况的所有检查报告全都保留着。
<br/>
<br/>        最令人感到痛心的是，这孩子才两岁零三个月，却患有血红蛋白病，又称海洋性贫血。
<br/>
<br/>        童筱颖怜惜这孩子的同时，也疑惑这孩子的名字为什么要叫“季小景”。
<br/>
<br/>        特别是这小男孩，瞪着大大的眼睛，看着她的时候，和她的小卫越来越像。
<br/>
<br/>        看到这里，童筱颖开始坐立不安，目光找不到焦点，眼神闪烁不定，甚至放在膝盖上的双手有些抑制不住的颤抖起来。
<br/>
<br/>        季思妍眼睁睁地看着童筱颖那脸色渐渐地变得惨白起来，于是不打算再拐弯抹角，直接开门见山地说道：“没错，如你所料，小景，就是我和阿景的孩子。”
<br/>
<br/>        这一瞬间，童筱颖觉得自己就像听到晴天霹雳的噩耗一般，震惊、难以相信又不得不相信。
<br/>
<br/>        “茶、茶凉了！我去给你重新泡一杯过来！”童筱颖双唇微微抖动，话语支吾地从发紫的唇瓣间挤出。
<br/>
<br/>        她就像逃命似的，从季思妍面前起身疾步离开去了厨房。
<br/>
<br/>        走进厨房里，站在案板前，童筱颖只觉自己好像瞬间被万箭穿心，鲜血源源不断地从心脏里面往外冒。
<br/>
<br/>        紧接着，她的胸腔好像生出无数只蚂蚁，撕咬着她的心肝脾肺肾，刺痛感一阵一阵、一波又一波地涌上来，吞噬着她全身的痛感神经。
<br/>
<br/>        季思妍见童筱颖异常的平静，心里顿时一阵焦虑，按捺不住地从沙发上起身，追去了厨房门口。
<br/>
<br/>        此时此刻，她已经顾不上童筱颖是何种心情了，因为，她要救她的儿子。
<br/>
<br/>        “筱颖，小景需要脐带血里的造血干细胞治病。我一直联系不上阿景，所以才苦苦寻到这儿来的。”季思妍走到童筱颖身后，带着哭腔说道。
<br/>
<br/>        童筱颖深呼吸着，手里却紧紧地握着玻璃杯，五指的指腹已经红白分明，咬牙切齿地问道：“那孩子……你是怎么得来的？”
<br/>
<br/>        “……”季思妍身心一怔，忽然间沉默了。
<br/>
<br/>        “怎么？你都主动找上门来了，到了此时此刻，你还打算敢做却不敢认吗？”童筱颖忽然转过身来，目光犀利地瞪着季思妍，冷冷地质问。
