　　子午眼睛盯着刚才举牌参加竞拍的鸭舌帽男子的方向，眼睛眯了眯，对身边的罗昊问道：“看到刚才举牌的鸭舌帽了吗？”
<br/>　　“注意到了。”罗昊点头回答道。
<br/>　　“盯住他。”子午吩咐道。
<br/>　　“明白！”
<br/>　　拍卖会依旧在继续，而后面出现的拍卖品无疑要比之前所出现的拍卖品都要出色，当然竞拍的价格也要搞出许多。从第二十号拍卖品开始，起拍价格就不再仅仅是钱了，而是在钱的基础上再加上源代码，正如子午在来这里之前所说的那样，在这个源代码拍卖会上，最不值钱的就是钱！
<br/>　　拍卖会越是往后，越是那些大公司之间的竞争，比如紫色卡片区的人已经失去了竞拍的资格，因为他们连起拍价所需要的东西都拿不出来，谈何参加竞拍？
<br/>　　罗昊、子午来参加这次拍卖会就是想要从这里找到一些关于SHADOW的线索，现在找到了他们想要找的人，原本对于罗昊和子午来说，他们此行的任务已经完成，可是毒天使的现身却是让六个人不敢放松警惕，必须要弄清楚毒天使出现在这里的目的是什么。罗昊绝对不会相信毒天使来这里只是因为无聊而过来逛逛的，“暗月”的十二主天使之一出现，必然是有目的的。
<br/>　　拍卖很快就进入了尾声，但是在场所有人的热情并没有因此而降低，即便是没有竞拍的资格，但是他们也想要见识一下今年最后的压轴拍品是什么，毕竟每一年，这都是一个值得期待的焦点。
<br/>　　“接下来，是本次拍卖会的最后一件拍品。”
<br/>　　奥帕拿着话筒，对着舞台下的人群大声说道：“熟悉我们拍卖会规则的人都清楚，凡是持有黑色卡片的竞拍者，可以在拍卖会正式开始前得知拍卖会中的所有拍卖品的信息。也许，这里有新来的人觉得这样很不公平，质疑我们的做法有失公平，公平都是相对而言的，拿着黑色卡片的人背后有你们羡慕都羡慕不来的背景，所以他们有资格提前知道拍品，而你们不行！”
<br/>　　奥帕的话说得很不客气，但却是实情，这个世界本来就没有那么多的公平可言。王子和平民，从他们出生的那一刻起，就注定他们的人生轨迹会是不一样的。
<br/>　　奥帕那双嵌在脸上肥肉中的小眼睛中，闪烁着锐利之色，拿着话筒缓缓说道：“但是这一次，这最后一项拍品是什么就连黑色卡片的持有者也不知道，因为他们手里那本下册子最后一页是空白的。可以这么说，在这里除了这件拍品的主人外，只有我才知道这件压轴拍品是什么。”
<br/>　　看到舞台下的人胃口都被自己吊足之后，奥帕终于是揭晓了这次拍卖会最后一件拍卖品的神秘面纱，被推上舞台的，还是一个玻璃展示柜，里面还是摆放着一个U盘，只不过现场所有人都知道，这个U盘里的东西可不一般。
<br/>　　“当我刚知道这个U盘中的东西时，我被吓了一跳。”奥帕站在玻璃展示柜前，说道。
<br/>　　舞台下，黑色卡片区内，洛菲克和拉塞尔相互对视了一眼，他们觉得，P病毒配方会被拿出来进行拍卖的传言或许是真的。
<br/>　　而知道这个传言的，并非只有洛菲克和拉塞尔两个人，就连紫色卡片区内都是有人知道了这个消息。消息自然不可能是拍卖会这边的人走漏的，他们做拍卖也不是一年两年了，这个消息是有心之人故意放出去的。
<br/>　　但凡是知道这个消息的人，目光都是在闪烁着，心中做着盘算。如果这个U盘中所装的东西，真的是P病毒的融合配方，在场每一个人都想要把它拿到手；如果U盘中装得东西真的是P病毒的融合配方，那么接下来的拍卖将会这次拍卖会最激烈的一次，同样的，交易金额恐怕也将会是整场拍卖会中最高的。
<br/>　　“本次拍卖会最后一件压轴拍品是……”
<br/>　　奥帕故意拖了一个长音，说道：“‘灭亡’病毒的融合配方！”
<br/>　　当奥帕说出最后半句话的时候，舞台下顿时一阵哗然，就连罗昊也是差点从椅子上站起身来。
<br/>　　洛菲克和拉塞尔眼中露出锐色，心中已经是有了决定！
<br/>　　原来如此！
<br/>　　罗昊看向毒天使的方向，发现他居然还在低头玩着他手里的手机，对舞台上所发生的事情毫不在意，毒天使出现在这里的原因，竟然是拍卖P病毒的融合配方！
<br/>　　“‘暗月’是疯了吗？它居然把P病毒的融合配方拿出来进行拍卖？！”周柏诧异地说道：“这不等于是把自己的底牌全都交到别人手上了吗？”
<br/>　　“‘暗月’才不会做傻事。”李然淡淡地说道：“我们都是跟P病毒打过不止一次交道，我问你们，P病毒是靠什么病毒融合在一起得到的？”
<br/>　　“天花和埃博拉。”周柏回答道。
<br/>　　“准确的说，是天花Ⅱ型病毒和埃博拉病毒进行融合在一起得到的。”李然纠正道：“但是这个世界上，除了‘暗月’手中掌握了当年竹中逸夫所留下的天花Ⅱ型病毒的培养方法，谁还知道天花Ⅱ型病毒的培养方法？只要不把天花Ⅱ型病毒的培养方法泄露出去，就算是外面的人得到了P病毒的融合配方又有什么用？他们连开启P病毒这扇大门的钥匙都没有拿到手，如何来融合P病毒？而且我们都知道P病毒具有变异性，可以进行突变，退一万步讲，‘暗月’即便是把P病毒的融合配方和天花Ⅱ病毒的培养方法公布出去，他们自己手里依旧掌握着杀手锏。”
<br/>　　“原来如此。”周柏点了点头，说道：“‘暗月’为什么要把P病毒融合配方拿出来？”
<br/>　　显然，周柏是对“暗月”的这种做法感到非常不解。
<br/>　　“我先问你，这个世界上谁是最想要得到P病毒融合配方的？”李然没有直接回答周柏的问题，反倒是向周柏进行了提问。
