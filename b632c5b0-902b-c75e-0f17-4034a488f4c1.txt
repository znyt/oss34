夏夕绾跟吴妈去了隔壁的卧室，这个卧室真的好大啊，跟主卧一样，低调里透着奢华。
<br/>
<br/>        而且这个卧室的设计都是冷色调的，黑灰蓝居多，这种沉稳内敛的色调一看就很陆寒霆风。
<br/>
<br/>        “吴妈，我们是不是走错房间了，我看这个房间挺像陆先生的。”夏夕绾狐疑道。
<br/>
<br/>        吴妈一愣，迅速呵呵笑了，“少奶奶，我们没有走错房间，这就是客房，这个西苑是少爷名下的私人房产，所以每个房间的设计都差不多，是少爷喜欢的那种品味。”
<br/>
<br/>        “哦。”夏夕绾打消了疑虑，明白的点了点头。
<br/>
<br/>        吴妈暗中松了一口气，她感觉自己跟了老夫人后，这撒谎也撒的贼溜了。
<br/>
<br/>        “对了吴妈，你以后不要叫我少奶奶了，免得穿帮了，你就叫我的名字，叫我绾绾吧。”
<br/>
<br/>        “好，绾绾，那你早点休息，有事就叫我。”
<br/>
<br/>        “恩吴妈，你也去休息吧。”
<br/>
<br/>        吴妈走了，夏夕绾进了沐浴间，先冲了一个热水澡，她要早睡早起，明早就给奶奶准备营养餐。
<br/>
<br/>        洗过澡后，夏夕绾发现了一个问题，那就是她没有睡衣。
<br/>
<br/>        好尴尬啊。
<br/>
<br/>        夏夕绾准备拿一块浴巾裹住自己，但是这时她看到衣架上挂了一件男士的白色衬衫。
<br/>
<br/>        看衬衫的size，应该是陆寒霆的。
<br/>
<br/>        他是这里的男主人，这些东西当然是他的。
<br/>
<br/>        她借穿一晚，他不会知道的吧？
<br/>
<br/>        肯定不会知道啊，这是她的房间，等明天一早她就脱下，然后偷偷告诉吴妈帮忙处理一下。
<br/>
<br/>        夏夕绾伸出纤白的小手拿了他的白色衬衫，然后穿在了自己的身上，将潮湿的头发吹干，她掀开被子上床。
<br/>
<br/>        不知为何，她觉得她身上盖的这条蚕丝被都是他的气息，他身上那股干净清冽的男人气息，特别好闻。
<br/>
<br/>        在酒吧里，她还撞到他怀里过。
<br/>
<br/>        夏夕绾将小脸埋在了蚕丝被里，任由他的气息将自己给包围，露在外面的一双澄眸黑漉漉的转了一下，然后她闭上眼睛睡觉。
<br/>
<br/>        很快，她就进入了梦乡。
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆将厉嫣然送了回去，厉家别墅的草坪上，厉嫣然解开了身上的安全带，她看向身边的男人，“寒霆哥哥，你要不要进去坐坐？我爹地还在国外，没有回来，奶奶去山上的寺庙拜佛了，家里没有人。”
<br/>
<br/>        陆寒霆英俊的五官镀在路灯里，低调中越显迷人的清贵，他扭头，深邃的狭眸看了厉嫣然一眼。
<br/>
<br/>        厉嫣然的心扑通扑通的乱跳，在他成熟的目光里，她漂亮的小脸通红，支支吾吾道，“我…我不是那个意思…”
<br/>
<br/>        家里没有人，她邀请他进去坐坐，容易让人想入非非的。
<br/>
<br/>        “我不进去了，你回家吧。”这时陆寒霆嗓音低沉的开腔道。
<br/>
<br/>        厉嫣然红扑扑的脸蛋僵了一下，他拒绝的毫不犹豫，他拒绝一切可能性的暧昧进展。
<br/>
<br/>        厉嫣然看着他，他已经低下了英俊的眼睑从香烟包里拿出了一根烟叼在薄唇上，戴着名贵腕表的右手握在方向盘上，他左手用打火机点烟，猩红的火苗跳跃他紧蹙着剑眉抽了一口烟。
