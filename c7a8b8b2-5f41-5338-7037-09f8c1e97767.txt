        所以，从此以后，就让他一个人孤独终老吧！
<br/>
<br/>        就像他母亲那样，这一辈子只爱他父亲一人，父亲过世时，母亲那时还年轻，以母亲的姿色和家世背景，其实是可以改嫁另觅更好的婆家。
<br/>
<br/>        但是，他的母亲并未改嫁，只为守着那颗深爱着父亲的初心。
<br/>
<br/>        或许，这种钟情、专情、痴情，是会遗传的吧……
<br/>
<br/>        这一辈子，他都不会让她知道，他只爱她一人。
<br/>
<br/>        傅夜沉苦笑，只身离去。
<br/>
<br/>        童筱颖完全沉浸在自己的计划中，丝毫没察觉到傅夜沉对她的异样苦心。
<br/>
<br/>        在朋友最需要关怀的时候，童筱颖在陪老公和陪朋友之间，还是先选择了陪朋友。
<br/>
<br/>        当她联系上季雅姗后，季雅姗给她发了家庭住址。
<br/>
<br/>        童筱颖回了设计部，办理了请假手续后，离开了傅氏集团，直接去了季雅姗的家中。
<br/>
<br/>        季雅姗的家住的是滨江城里的高档别墅区，进出别墅区还得登记并通知被拜访的主人。
<br/>
<br/>        童筱颖得到核实后，才被门卫的保安放行进入。
<br/>
<br/>        到了季雅姗的家中，雅姗告知童筱颖，这房子是十爷买的，现在分手了，房子也一并送给她了。
<br/>
<br/>        童筱颖陪着季雅姗在飘窗上坐着，季雅姗双手抱，黯然神伤地看着窗外。
<br/>
<br/>        “阿十真的对我很好，我要什么，他就给我买什么！而我只需要在每个周末的晚上，陪他睡一晚即可。他真的只是抱着我单纯的睡觉噢！他从未碰过我！起初，我怕他。慢慢地，我好像开始贪恋他了怎么办？”季雅姗忧伤道。
<br/>
<br/>        十爷的身份是个迷，就连季雅姗也不知道他具体是个什么背景。
<br/>
<br/>        唯一知道的是，十爷是“夜惑”的股东之一。
<br/>
<br/>        童筱颖沉默了一会儿，忽然开口问道：“十爷他……是不是无间道上的？”
<br/>
<br/>        季雅姗看了童筱颖一眼，微微摇头：“我不知道。”
<br/>
<br/>        “雅姗，我知道，你喜欢上了十爷。但是，十爷与我们不是一个世界的人。我感觉，他的身份不简单。既然他愿意对你放手，还你自由。说明，他是为了你好，才做的这样的决定。所以，从今以后，雅姗你要开心地过好每一天，不要辜负了十爷的一番苦心。”童筱颖抓住了季雅姗的手，安慰道。
<br/>
<br/>        季雅姗抬眸看着童筱颖，突然张开双臂，抱着童筱颖说道：“筱颖，能遇到你，真好。”
<br/>
<br/>        “我也是！”童筱颖抬起手来，轻轻地拍抚着季雅姗的背脊，欣慰地微笑着。
<br/>
<br/>        十爷走了以后，季雅姗辞退了家里的保姆。
<br/>
<br/>        她和她的母亲并不住一块，母亲住在另一个小区里，十爷请了专人照顾。
<br/>
<br/>        季雅姗起初以为，十爷对她放手了，就不会再管她和她的母亲了。
<br/>
<br/>        岂料，她母亲那边，听照顾母亲的保姆说，十爷会继续赡养她的母亲。
<br/>
<br/>        “我妈妈嗜赌，但她怕阿十。因为，有一次，她去赌，被阿十的手下抓住了，差点砍掉了手指。我妈妈怕了，就再也没去赌了。老老实实地在家里，做一些手工活，可以拿去给阿十的手下，让他手下的人帮忙去卖。”季雅姗感慨道。
<br/>
<br/>        童筱颖第一次发现十爷的另一面，虽然暴戾恣睢，却是一个对带自己身边的人心地十分善良的男人。
<br/>
<br/>        回想起自己那次被十爷抓，还差点被十爷拿去喂狗，童筱颖至今都心有余悸。
<br/>
<br/>        十爷是一个什么样的男人，现在差不多已经明了了。
<br/>
<br/>        他会对自己中意的人特别好，不中意的人，只怕他像掐死一只蚂蚁一样简单。
<br/>
<br/>        “筱颖……”季雅姗忽然一声柔柔的呼唤，拉回了童筱颖的思绪。
<br/>
<br/>        “嗯？”童筱颖回神看着季雅姗。
<br/>
<br/>        季雅姗咧嘴一笑：“我决定了！”
<br/>
<br/>        “决定什么呢？”童筱颖怔愣一下，一种不祥的预感漫上心头。
<br/>
<br/>        “我要去找阿十！”季雅姗笑容灿烂。
<br/>
<br/>        童筱颖一时间不知道该说什么是好，愣了一会儿，才问道：“你是认真的？！”
<br/>
<br/>        “嗯！不管阿十是什么人，我都想去他的身边！”季雅姗信誓旦旦地说。
<br/>
<br/>        在这种情况下，作为她的朋友，她该说什么是好？
<br/>
<br/>        十爷的身世背景一定不简单，突然玩失踪了一般不再见雅姗，一定是遇上了什么麻烦事，又或者他是真的为了雅姗好。
<br/>
<br/>        童筱颖总觉得十爷对雅姗放手的这个决定是对的，但是雅姗似乎却在为这件事情而苦恼。
<br/>
<br/>        万一，这件事情，一边是天堂，一边是地狱，雅姗选错了路，步入了地狱，怎么办？
<br/>
<br/>        童筱颖心里非常纠结。
