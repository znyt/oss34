　　第251章
<br/>　　这天早上，城门下起了一场大雾，铺天盖地，每个人都如身在仙境，也如同进入鬼蜮。
<br/>　　宫符的心提了起来，因为这场大雾，目力所及不过百丈，以骑兵的冲锋速度，几个呼吸都能到阵前。
<br/>　　今日如果马匪冲过来，情况并不乐观。
<br/>　　宫符抽了抽鼻子，空气中有股怪味，吸入鼻子中，喉部感觉火辣辣的疼。
<br/>　　“不好，雾里有毒。”罗酒第一个反应过来，口中大吼。
<br/>　　“大家用布遮住口鼻，别吸雾里的烟。”宫符立刻下达命令。
<br/>　　这是鱼龙卫，他们应变能力极其惊人，可是肖竹率领的城防军就没有这样的运气了，有人被烟呛得直咳嗽，有人弯着腰呕吐不止。
<br/>　　远处，点着几个火堆，火堆中不知加了什么东西，顺着风吹到这里。
<br/>　　火堆后的马匪脸上都蒙着黑布，娃娃脸少年骑在马上，微微眯眼，口中吐出一个字。
<br/>　　“杀。”
<br/>　　噢！
<br/>　　如同狼吼一样的啸声，马蹄声如同擂鼓，大军乌压压扑过来。
<br/>　　鬼见愁身为边城十三家马匪头目，为这次行动，出动了五千余人。
<br/>　　不是鬼见愁如此兴师动众，奈何这次绿林道都传开了，都要抢和亲的明玉公主。某种程度上，这成为绿林道的一次盛事，就像长安城的武科举一样，谁能夺了公主，谁就可以笑傲群雄。
<br/>　　鬼见愁也好个面子嘛。而且他离得又近，出了夕照关就是他的地盘，怎么可能让别人抢了先。
<br/>　　羽箭从白雾中射出，给冲上来的马匪迎头痛击。鱼龙卫的反应能力绝对出乎鬼见愁预料。
<br/>　　而白雾中的宫符心情也不好，白雾影响了视力，这样的防御，只是无目标的乱射，杀伤有限，但对弓箭的消耗是巨大的。
<br/>　　宫符的布置不可谓不妥当，反应不可谓不快，某种程度上，他已经做到他能力范围的最好。
<br/>　　可是，在对方兵力占优的情况下，己方胜算不大。
<br/>　　天时，地利，人和，己方一样不占，若是输了，似乎也输的很有理由。
<br/>　　可宫符还不想认输，因为赢这个字，对一个男人来讲，很重要。
<br/>　　宫符在等，在等浓雾散去，然后向马匪亮出鱼龙卫的刀锋。
<br/>　　“虎爷，请你闯出去，去夕照关搬兵，就说我们已经将鬼见愁吸引过来，只要大兵过来，就可以将他们一网打尽。”
<br/>　　一个虎背熊腰的汉子无奈收回自己的钢刀，嘴里骂咧咧道了一声：“嘿，我还想杀个痛快。”
<br/>　　面对马匪的骑兵冲锋，宫符展开反冲锋，以骑对骑。
<br/>　　在帝国与戎族的战争中，帝国一直被戎族的骑兵压制。多少帝国人绞尽脑汁，思考克制戎族骑兵的办法，就有人提出，要以骑对骑，培养帝国的骑兵。
<br/>　　不过，这个法子实施起来难度太大，第一帝国没有好马，第二，帝国没有培养骑兵的土壤，骑术是戎族的血统天赋。
<br/>　　可今天情况不同，第一马匪不是戎族，第二，鱼龙卫不差钱。
<br/>　　他们有展开骑兵对冲的资格。
<br/>　　像两股洪流对撞，马匪在鱼龙卫的冲锋下溃散。这次护送和亲队伍，明帝派出十名龙卫，莫看只有十人，每个人都有与青叶佛相仿的实力，他们在这次战斗中发挥了很大作用。
<br/>　　马匪的第一次攻击，被压了下去，而晨雾也在这个过程中渐渐散去。
<br/>　　宫符的心情不是很好，虽然侥幸挡住第一次，但未必能撑到援兵赶到。
<br/>　　他悄悄讲罗酒几名龙卫叫到一起，如果在黄昏援兵还未赶到，那么就有他们带着明玉公主离开，宫符带人阻挡追兵。
<br/>　　明玉公主站在废城的城楼上，目睹着城下战斗过后的狼藉，那些尸体，受伤的士兵，都没能在她脸上掀起任何波澜。
<br/>　　如果是以前，她会吓得大叫吧，但现在不对了。慢慢，她已经能体会父王的心情，无论遇到什么情况，都只做正确的事情。
<br/>　　该牺牲的牺牲，该舍弃的舍弃，不要被情绪所影响。
<br/>　　她带着太监宫女下城楼，给战士包扎伤口。宫符以感激的目光看着她，虽然她什么都没说，但士气已经提了起来。
<br/>　　靠着这一点，宫符又抵住两波攻击，战斗一直持续到下午。
<br/>　　然而，惨烈的战斗必然带来惨烈的牺牲，在黄昏时马匪中突破了鱼龙卫的防线。
<br/>　　然后就是面对面的白刃战。
<br/>　　“守住城楼，守住城楼，莫让贼人伤到公主。”
<br/>　　宫符提枪捅死面前一个马匪，扯着嗓子大吼。
<br/>　　戎族已经围住城楼，城楼成了孤城，陷入马匪的汪洋大海中。
<br/>　　终于还是没坚持到援兵到来，宫符知道已经守不住了，只能做最坏打算，让罗酒等人护送着明玉公主先走。
<br/>　　现在宫符就是为罗酒争取时间，至于那些太监宫女则完全顾不上了，因为宫符的命已经准备好牺牲。
<br/>　　罗酒保护着明玉公主下得城楼，挥舞着铁鞭将马匪驱散，如此把明玉送上马车。
<br/>　　可是，此时周围都是密密麻麻的马匪，便是罗酒也是一身寒意，想要冲出去，怕不太容易。
<br/>　　“劈脑袋喽！”
<br/>　　“掏耳朵喽！”
<br/>　　“剔牙齿啊！”
<br/>　　伴随着一声声大吼，外围的马匪竟被杀出一条路来。宫符心中一喜：难道最关键时刻，自己等的援军到了。
<br/>　　可随即，他便发现自己错了，因为来的只有一个人。
<br/>　　只见他骑着一只白毛骆驼，好俊的骆驼，浑身上下没有一丝杂毛，四蹄踏雪，身若滚雪，此时赫赫冲过来，当真是俊逸非常。
<br/>　　至于骆驼上的人……因为平平无奇，不提也罢。
<br/>　　而马车上的明玉看到这人模样时，眼中闪过一道亮光，这一刻明玉公主重新成为李婉儿。
<br/>　　马车周围挤满马匪，这骆驼出场方式又太招恨，几乎所有马匪都向他砍去。
<br/>　　骆驼上的人点脚跃起，背后如生双翼，在空中踏风而行，跃过重重包围圈，落在马车前。
<br/>　　也落在千军万马正前方。
<br/>　　此时，他一个人就是一支队伍。
<br/>　　罗酒松口气，道：“兄弟，你是哪儿的，我以前怎么没见过……”
<br/>　　话没说话，他就被踹飞了。
<br/>
