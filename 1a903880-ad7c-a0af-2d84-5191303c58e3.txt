　　张无生冷笑，收起九天元辰尺，手持玉清拂尘，上前迎击，两人即将碰到的一刻，叶少阳一抬手打出八张灵符，脚下一个错步，滑到他身后去，单手结印，对着他后脑勺拍过去。
<br/>　　张无生反应极快，猛然转身，拂尘刷下来，叶少阳变幻手印，硬对了一记，两股罡气相撞，两人各自退了一步。
<br/>　　张无生皱起眉头，暗暗吃惊于眼前这个少年的手段。
<br/>　　叶少阳双手结印，发动了新一轮的攻击，张无生也沉下心来，一招一式地跟他拆解起来。
<br/>　　两个道门领袖，一老一少，就在两军阵前展开了斗法。
<br/>　　不管是茅山道术还是龙虎山道术，主要都是针对邪物的，哪怕是星月奴这样的轩辕山大佬，本质上不是人类，也是吃法术的，但人类跟人类之间，很多法术都没法用，法师之间过招，就像武术一样，比的一个是身手，一个是法力。法器的作用并不大。
<br/>　　在旁人眼里，这样的斗法打起来并不好看，一点也没有惊天动地的那种效果，就看两人左右游走，突然击掌，接着立刻分开……一刻钟下来，场面不分高下。
<br/>　　小马就捏着板砖在边上晃悠，盯着张无生那些手下，免得他们上去帮阵。包子坐在他肩膀上，一直盯着战场，喃喃道：“这个龙虎山掌教好厉害啊！”
<br/>　　“废话！”
<br/>　　好歹是龙虎山掌教，道门魁首，虽然张无生做人一向狡猾，极少出头，也好多年没跟人打过，但没人怀疑他的实力。
<br/>　　“啪啪啪啪啪！”
<br/>　　两人不断变换手印，用力拍打在一起，最后一记碰撞发出的声音尤其巨大，两人都使出了全力。叶少阳后退了半步，张无生却是连退几步。
<br/>　　叶少阳没有给他机会，一个纵身过去，趁着他没站稳，倏然出招，张无生一口气没提上来，仓促应战，被叶少阳在眉心处弹了一下，只觉得魂魄震撼，眼花缭乱的，凭着本能反击，但叶少阳已经完全掌握了局面，几个回合下来，寻到了他一处大的破绽，再狠狠来了一下，张无生单膝跪地，叶少阳绕到他身后，对着他后脖颈用力来了一下。
<br/>　　“啊！”张无生惨叫一声，身子一软，倒在地上。
<br/>　　“将军！！”
<br/>　　张无生手下之中，几个军官立刻就要冲上来。叶少阳一只手按着张无生的脖子，呵斥道：“都别过来啊！”
<br/>　　这帮人立刻就不敢动了。
<br/>　　小马和包子立刻飞到叶少阳身边，站在左右保护他。
<br/>　　叶少阳把张无生翻过身来，发现他额角磕在地上，皮肤蹭破了，但是破损的皮肤看上去有些怪异，于是伸手抹了一把，发现皮肤有些卷曲，再用力一抹，血当场就流下来了。
<br/>　　三人都吓坏了，然而血只流了一会就停了，再看张无生，一张面皮迅速萎缩，出现了很多皱纹。
<br/>　　“啊呀小叶子，你给他弄死了！”小马惊叫。
<br/>　　“滚！”
<br/>　　叶少阳突发奇想，一只手按在张无生额头上，用力往下蹭，结果将整张面皮都撕下来，叶少阳用袖子把他脸上的血抹掉，这才露出了那张熟悉的张无生自己的脸。
<br/>　　再看那张脱落的脸皮，因为失血已经萎缩了。
<br/>　　“原来脸皮是假的！圣灵会干的吧，为什么要这么干？”小马一脸懵。
<br/>　　“我哪知道。可能是怕被熟人认出来吧。”
<br/>　　“啊，小叶子，你说，和尚会不会也是这样，被换了脸皮，所以咱们没能认出他来？”
<br/>　　叶少阳心中一动。这时候张无生开始抽搐了，叶少阳赶紧画了一张定身符贴在他额头上，张无生很快醒来，像那些刚被唤醒的人一样，陷入狂暴之中，不过因为魂魄被封，浑身动弹不得。
<br/>　　他那些部下虽然着急，但统帅被他们控制着，不敢轻举妄动，只好高声叫骂。
<br/>　　叶少阳又把发生的事说了一遍（已经不知道说了几遍了，台词背得贼溜），然后又跟张无生说了他被灌输的记忆和新的身份。
<br/>　　“话说，您老完全都听懂了没有？听懂了眨眼。”
<br/>　　结果张无生给了他一个大大的白眼。
<br/>　　叶少阳吐了吐舌头，“不眨眼不给你解咒啊。”
<br/>　　张无生拿眼珠子狠狠瞪他。
<br/>　　叶少阳哈哈一笑，把灵符撕掉。
<br/>　　“咳咳咳……”张无生弯腰咳嗽起来。
<br/>　　叶少阳给他拍了拍后背，道：“老爷子，你不会被我打伤了吧。”
<br/>　　“凭你？”
<br/>　　张无生咳了一会好了，坐直身子，朝四周望去，低声道：“这些都是我的手下？”
<br/>　　“对，您是将军。”
<br/>　　“哼。”张无生轻蔑又无奈地哼了一声，思考了一下，道：“星月奴为什么不杀我？”
<br/>　　“她闭关了，根本没工夫处理这些，我估计她都不知道你闯进来。”
<br/>　　“那是谁给我安排的这身份？”
<br/>　　“是她弟子吧。我也不知道，这里所有人都被分配了新的身份。”
<br/>　　张无生果断地说道：“不可能，他们不会这么稀里糊涂的，自然是知道我们的身份，没杀我们，必然是有目的的。”
<br/>　　“也许吧。老爷子先别说这个，你先安抚一下你这些手下。”
<br/>　　张无生站起来，朝自己手下那帮士兵望过去。
<br/>　　所有人都睁大眼睛、用不可思议的目光望着他，不明白他为什么换了一张脸。但也不噶问。
<br/>　　“我搞错了，他们不是反贼，是朝中出了奸臣，他们是来清君侧的，是自己人。”
<br/>　　将士更加震惊。
<br/>　　“你们都在这等着吧，不要妄动，我……进去面圣。”
<br/>　　跟着叶少阳他们一起进宫。
<br/>　　“姜还是老的辣，师叔你这么快就适应身份了。”叶少阳冲他竖了竖大拇指。
<br/>　　张无生哼了一声，叹道：“还不是着了他们的道，不然也不会到这里来。”
<br/>　　“这个不管你，九星叠气阵实在可怕，我们不也被吸进来了吗。”
<br/>　　“九星叠气阵……”张无生停住脚步，感慨了一下，才接着迈步进去。
<br/>　　张无生刚清醒，跟他们之前一样，对这里的一切都感到新奇，叶少阳在一旁给他讲解，一行人进了皇宫，张无生见到皇帝和文武百官，更加吃惊，不过也没有多说。
