　　“怎么，莫非你还要留下我们打劫不成？”
<br/>　　林峰一眼看出这家伙的实力不过一阶罢了。
<br/>　　“呦呵，还拿着武士刀和长枪？小子，你们莫非是表演杂技的不成？不过这话你说对了，我们就是要打劫！”
<br/>　　小弟冷笑道。
<br/>　　“广哥，这边有好武器，您过来看看！”
<br/>　　小弟立即道。
<br/>　　“哦？是吗？”
<br/>　　广哥眼睛一亮，他现在最缺的就是武器了。
<br/>　　立即朝林峰走了过去。
<br/>　　“广哥，就是这小崽子，武士刀和长枪！也不知道哪里搞来的！“
<br/>　　小弟立即道。
<br/>　　“广哥，你说这两个家伙是不是杂剧团的，不如让他给我们表演表演。”小弟嘿嘿道。
<br/>　　“这是个好主意。”
<br/>　　被叫做广哥的人也是颇为赞同的点了点头。
<br/>　　“看来真有人找死。”
<br/>　　林峰无奈的拔出了武士刀。
<br/>　　瞬间那广哥表情就变了。
<br/>　　然后一巴掌就是对身边的小弟抽了过去。
<br/>　　这小弟瞬间就被一巴掌抽飞了。
<br/>　　嘴巴都被打肿了，牙齿都断了好几根，广哥三阶的存在，这一巴掌没有任何手下留情，差点一巴掌给他打死了。
<br/>　　“广哥，你干嘛呢。”此时此刻他一脸懵逼，广哥不打这个小子，对他怎么忽然出手了。
<br/>　　“跪下。”
<br/>　　广哥忽然道了句。
<br/>　　小弟完全不知道发生什么了。
<br/>　　但是下一刻他更加愕然。
<br/>　　因为广哥居然主动跪下了。
<br/>　　“大哥，我这小弟有眼无珠！不知道您是四阶的大哥！我错了！别杀我们！放我们一马！”
<br/>　　广哥认真的说道。
<br/>　　“什，什么四阶？”
<br/>　　小弟之前还一肚子委屈，但此时此刻，全都烟消云散了，只剩下了无尽的恐惧。
<br/>　　噗咚。
<br/>　　旋即他也跪下了。
<br/>　　“爸爸，别杀我，我错了，爸爸。”
<br/>　　林峰和陈勾愣了，有意思，他的刀刚刚拔出来，这群人就全都吓跪了。
<br/>　　“怎么，现在知道错了？这拔出去的刀可不好收回来。”
<br/>　　林峰笑呵呵道。
<br/>　　“不知道两个祖宗在此，还望祖宗们大人不记小人过。”
<br/>　　小弟欲哭无泪。
<br/>　　他此时此刻身子都在颤抖，居然是四阶的，自己之前怎么就没有察觉到。
<br/>　　“林峰，看来你是动了王八之气，这群家伙立即感觉到了。”
<br/>　　陈勾道。
<br/>　　“十分有可能，不过不是王八之气，而是某种气势，我刚才都准备拔刀灭了他们，可能是杀气之类的，被他们察觉到了，所以就一下子看出我实力来了。”
<br/>　　林峰道。
<br/>　　“大哥，这都是个误会，真的是个误会。”
<br/>　　广哥在那里干笑道。
<br/>　　黑铁战士多厉害他自己也是自然清楚。
<br/>　　彼此之间，等级森严，除非有枪，或者其中一方学习过武术学习过搏击术，不然基本上根据等级，就已经高低立分了。
<br/>　　这也是为什么广哥一个三阶高手，二话不说跪下的缘故。
<br/>　　“一群白痴，都给我过来给大哥跪下。”
<br/>　　广哥立即把其余小弟都叫过来了。
<br/>　　一群小弟不知道发生了什么，但是看到广哥都跪了，二话不说也都跪了。
<br/>　　而那群幸存者趁这个机会都跑了。
<br/>　　“都给我拜见祖宗！”
<br/>　　“拜见祖宗！”
<br/>　　十几个年轻人都给林峰和陈勾齐刷刷磕头。这一幕就好像是黑社会砍人之前拜关公一样。
<br/>　　林峰和陈勾两个高一学生，一辈子还是第一次受人如此大礼。
<br/>　　“林峰，我们两个现在是不是比那曹豹还要牛逼，他之前也没有这么多社会人给他磕头吧？”陈勾笑道。
<br/>　　“和曹豹比什么，他死人一个。不过这群人你觉得怎么办。”
<br/>　　林峰道。
<br/>　　“要不当小弟算了？这群人比学校里那群家伙要战斗力强多了，你总不可能一直带着那群家伙吧，而且我看那群家伙一个个都想着回家，说不定安全了后他们第一时间就回家去了。”陈勾道。
<br/>　　林峰闻言，陈勾说的很对，学校里的老师和学生都要回家，学生要去看自己爸爸妈妈，老师担心自己的孩子。
<br/>　　都有家庭的负担。
<br/>　　而他和苏云孤儿，无父无母，陈勾一个人在江城租房子，家里也没有姐姐妹妹，父母都在乡下，担心也没有。唯独一个开网吧的表哥。
<br/>　　所以，到最后，能够带在身边的就陈勾和苏云了，而且他们也肯定愿意跟林峰走，这也是为什么，林峰一直对他们很不错的缘故。
<br/>　　“那个，大哥，怎么称呼啊，小的我叫李广，曾经是混天安街那边的，道上兄弟给脸面的就叫我一声广哥，当然了，在大哥您面前，我就是个弟弟。”
<br/>　　李广拿了包中华递上。
<br/>　　林峰和陈勾都会抽烟，点燃了烟后眯着眼看着他。
<br/>　　“挺会来事。”
<br/>　　“那是必须的，必须给大哥点上。”李广笑道。
<br/>　　“两个大哥看上去很年轻，混哪里的。”
<br/>　　“我们还是学生，对了，你知不知道曹龙这号人物？”
<br/>　　林峰忽然问道。
<br/>　　“知道，龙哥，翻江龙龙哥。”李广道。
<br/>　　“很牛逼？”林峰问道。
<br/>　　他杀了曹豹，这个曹龙若是没死肯定要来报仇的。他要了解下这个人的过往。
<br/>　　“是挺牛逼的，不过也就十几年前牛逼，那时候整个江城都知道他，不过现在人老了，五十岁人了，体力什么的跟不上了，最主要的是，当大老板了，人怕死，就开了几家赌场，不怎么出面。再加上现在这朝阳群众无所不能，拍个小视频放微博上，一下子甭管多大的黑社会只要曝光了，都得给抓进去，所以大家都不得不怂。”李广道。
<br/>　　“你怕不怕他？”
<br/>　　林峰笑呵呵道。
<br/>　　“怕个毛，我干嘛要怕他？大家都是人，都一条小命，他手下有人，不过都是些三十岁的大叔，而我手下也有人，还是不怕死的年轻人，真干起来，胜负难分，当然了，我听说他有枪，就这一点，我要怂一下，当然了，我和他也没有什么矛盾。井水不犯河水吧。这年头赚钱的来路太多了，不像以前打打杀杀见面就砍，现在和气生财。”李广道。

<br/>　　“服气，一万个服气，你们呢，服不服气？”
<br/>　　李广看着几个小弟道。
<br/>　　一群小弟面面相觑。
<br/>　　让他们给几个学生当马仔？就算是四阶又如何。一群小屁孩罢了。
