　　萧逸的位置，是在洞玄境妖兽这边。
<br/>　　并不与裂天剑派的弟子一道。
<br/>　　在人类武者的刻意之下，战场早就被分割开一片片。
<br/>　　妖兽不懂排兵布阵，各个级别的实力，泾渭分明地被分开了出来。
<br/>　　这也是它们死伤得这么快的原因之一。
<br/>　　本以为，这次的兽潮，胜利在即。
<br/>　　但，火岩巨兽和阴风蝠妖的出现，却再次让在场所有人类武者心头咯噔一声。
<br/>　　两头破玄九重实力的妖兽，是这次兽潮中，首次出现的最强妖兽。
<br/>　　“诸位莫慌。”三长老一掌震退自己的对手，而后大喝一声。
<br/>　　“四长老，前去相助弟子们，这里交给我。”
<br/>　　“好。”四长老，也就是剑堂长老，猛地点点头。
<br/>　　而后身影一闪，瞬间朝三百弟子那边赶去。
<br/>　　这些弟子，可都是剑派最出色的下一代，绝不能出意外。
<br/>　　再者，他的弟子顾长空，也在那里。
<br/>　　“孽畜，你的对手是我。”
<br/>　　剑堂长老本身的对手，是两头破玄八重实力的妖兽。
<br/>　　它们刚想追击剑堂长老，便被三长老拦了下来。
<br/>　　三长老本身也在牵制着两头破玄八重实力的妖兽。
<br/>　　现在，竟是一人力敌四头破玄八重，竟丝毫不落下风。
<br/>　　他的实力，可见绝不一般。
<br/>　　他是火之一道的武者，爆发强，擅长范围性战斗。
<br/>　　要拦下数头破玄八重妖兽，问题不大。
<br/>　　而剑堂长老，乃是剑道武者，最擅战斗。
<br/>　　单打独斗方面尤为见长。
<br/>　　由他去拦下火岩巨兽和阴风蝠妖最是合适。
<br/>　　……
<br/>　　火岩巨兽和阴风蝠妖的带领下，十万头半步破玄实力肆虐而来。
<br/>　　裂天剑派三百弟子、各大猎妖队队长，还有不少家族武者。
<br/>　　加起来，足有数千人。
<br/>　　他们乃是除了破玄境武者以及萧逸意外的，最强战力。
<br/>　　此刻，完全被包围。
<br/>　　只要火岩巨兽和阴风蝠妖随便一头出手，他们必死无疑。
<br/>　　“孽畜，拿命来。”剑堂长老及时赶到。
<br/>　　一人一剑，轻松拦下它们。
<br/>　　一众武者，立刻士气大振。
<br/>　　与这十万头刚刚进入战场的十万头半步破玄妖兽鏖战起来。
<br/>　　只不过，情况，并没有想象中的好。
<br/>　　除了十位内门首席，以及少数较强的内门弟子外。
<br/>　　其他武者，根本不是这数量庞大的半步破玄妖兽的对手。
<br/>　　他们只能苦苦支撑，根本不可能斩杀。
<br/>　　同一时间，剑堂长老，也陷入了下风。
<br/>　　火岩巨兽，浑身冒火，身躯高大且坚硬，极擅正面战斗。
<br/>　　阴风蝠妖，翅膀一抖，一缕缕幽深黑风，让人心神不宁，极为诡异。
<br/>　　两兽联手，就是剑堂长老都难以力敌许久。
<br/>　　另一边的三长老等人，显然也发现了这边的情况。
<br/>　　“四长老，带着弟子撤退，我尽快过去助你。”三长老大喝一声。
<br/>　　四长老怒喝道，“我拦下这两头孽畜已是尽力，如何还能带他们撤离。”
<br/>　　战况，由于这一批妖兽的加入，再次不妙了起来。
<br/>　　……
<br/>　　另一边，萧逸也发现了这个情况。
<br/>　　“不能在这边耗下去了。”
<br/>　　“必须到那边为他们打开缺口，让他们尽快撤离。”
<br/>　　萧逸一瞬间有了打算。
<br/>　　一道覆海斩打出，清理了身旁的妖兽。
<br/>　　而后身影一闪，直往被围困的众武者身边而去。
<br/>　　他的出现，立刻另得这近十万头半步破玄妖兽死伤惨重。
<br/>　　一路疾驰而来，所过之处，妖兽尽皆化为肉泥。
<br/>　　当他来到裂天剑派弟子身边时，一条由妖兽尸体铺成的血腥通道，硬生生被开了出来。
<br/>　　“叶铭，带人撤。”萧逸快速说道。
<br/>　　“两头破玄九重实力的妖兽，有剑堂长老拦着。”
<br/>　　“这十万头半步破玄，交给我就是。”
<br/>　　“好。”叶铭毫不犹豫地点了点头。
<br/>　　他刚才已经见识过萧逸秒杀洞玄境妖兽的快速和恐怖。
<br/>　　在他的带领下，一众武者，边打边退。
<br/>　　无数妖兽意图追击，都被萧逸一道覆海斩杀退。
<br/>　　当然，妖兽的数量，太多。
<br/>　　一众武者，毕竟在包围圈之内。
<br/>　　萧逸的力量也有限，只能大幅度帮他们减少压力。
<br/>　　一众武者，边打边退，还是有些吃力。
<br/>　　数分钟后，才算渐渐打出包围圈。
<br/>　　正当众人心头松一口气时。
<br/>　　不远处，那本在与剑堂长老激战的火岩巨兽和阴风蝠妖瞬间发狂。
<br/>　　“不好，这两头孽畜发疯了，快跑。”
<br/>　　剑堂长老大喝一声，并立刻后退。
<br/>　　这两头妖兽身上的气势，在不断激增。
<br/>　　如果没有猜错的话，它们在凝聚力量，恐怖的攻击，就要释放而出来。
<br/>　　连剑堂长老都要跑，更别说萧逸、叶铭等一众武者了。
<br/>　　“吼…”
<br/>　　“吼…”
<br/>　　两声震天兽吼出现。
<br/>　　火岩巨兽猛地张开了嘴巴，一层层炙热高温火焰激射而出。
<br/>　　阴风蝠妖猛地抖动翅膀，一股如龙卷风般的吸力陡然而出。
<br/>　　强悍风力，加上这层层高温火焰。
<br/>　　竟形成了一个恐怖的火龙卷。
<br/>　　数千武者，毫无反抗之力地被吸扯得不断后退。
<br/>　　“师尊，救我。”顾长空慌张地大喝了起来。
<br/>　　剑堂长老反应过来，一把将他拉住，而后全力劈出数道剑气。
<br/>　　生生挡了这火龙卷数秒。
<br/>　　“快跑。”剑堂长老大喝一声。
<br/>　　一众武者，撒脚便跑。
<br/>　　十大内门首席，实力过人，跑得飞快。
<br/>　　剑堂长老，不断劈出剑气，试图拦下这火焰龙卷。
<br/>　　忽然，顾长空急忙道，“师尊，我弟弟快撑不住了，先救他。”
<br/>　　顾长风那边，是内门弟子的方向。
<br/>　　他们的实力，比不上内门首席，自然没那么快逃脱。
<br/>　　嗖，剑堂长老竟停下挥动的剑气，瞬间消失在原地。
<br/>　　身影再次出现时，已在顾长风身边。
<br/>　　一手拎着顾长空，一手拎着顾长风，竟快速撤退起来。
<br/>　　“剑堂长老，你…”萧逸脸色大变。
<br/>　　这里，还有大量武者没有逃出包围圈。
<br/>　　以他自己的实力，要不是要帮林劲等人，以及其他武者打掩护，早就跑了。
<br/>　　绝对比十大内门首席还跑得快。
<br/>　　现在剑堂长老跑了。
<br/>　　那恐怖的火焰龙卷，毫无阻拦地再次往一众武者袭来。
<br/>　　“混账东西。”林劲等人脸色难看到极点。
<br/>　　剩下他们这一批武者，如何能在火岩巨兽和阴风蝠妖的联合攻击下逃生。
<br/>　　“秦飞扬，变雕。”萧逸大喝一声，而后猛地踏前几步。
<br/>　　“林劲，你们先走，能走一个是一个，我来断后。”
<br/>　　萧逸血戮剑出，体内三千多丈真气澎湃而出。
<br/>　　甚至没能来得及在半空中形成真气漩涡，已然立刻化为覆海斩的剑音。
<br/>　　轰…轰…轰…
<br/>　　火焰龙卷，生生被阻碍了两秒。
<br/>　　秦飞扬化雕下，驮着林劲、铁牛、柳嫣然，以及附近几个内门弟子快速飞离了包围圈。
<br/>　　其余内门弟子，身上也有剑派长辈赠予的防身之物。
<br/>　　一个个都毫不犹豫地用了出来，自身实力大涨下，逃出了这包围圈。
<br/>　　“呼…呼…”萧逸满头大汗。
<br/>　　一瞬间调动所有真气，他能做到，却不代表不吃力。
<br/>　　他可不觉得自己能打得过这两头破玄九重实力的霸主级别妖兽。
<br/>　　他只是拦个几秒罢了。
<br/>　　下一秒，感觉到体内真气差不多耗尽，也立马撤退。
<br/>　　显然，覆海斩的威力极强。
<br/>　　火岩巨兽和阴风蝠妖虽然没有被打伤，却也被剑音震得懵了一下。
<br/>　　萧逸的计算没有出错，他恰好可以够时间逃离。
<br/>　　嗖，身影一闪，快速撤退。
<br/>　　孰料，这时，一道澎湃剑气，竟猛地从他前方打来。
<br/>　　“嗯？”萧逸脸色一冷，前方，正被剑堂长老带着逃离的顾长空，竟在狞笑。
<br/>　　那道剑气，由他打出。
<br/>　　在冥空剑的增幅下，这道剑气，达到了破玄二重的威力。
<br/>　　同一时间，他手上一阵亮光。
<br/>　　一道更恐怖的剑气，直朝他打来。
<br/>　　“混蛋，你做什么。”率先逃离的叶铭等人大喝一声。
<br/>　　“那是剑堂长老给你防身的剑气，你打向萧逸做什么？”
<br/>　　他们的质问，并没有任何用。
<br/>　　因为，剑气已经打向萧逸。
<br/>　　萧逸挡得下顾长空的第一道剑气。
<br/>　　却挡不下这道不亚于剑堂长老全力一击的剑气。
<br/>　　“噗。”萧逸猛地吐出一口腥血，身影不断后退。
<br/>　　若非有五行铠甲护身，此刻已经重伤了。
<br/>　　一切都发生得太快，快到没人能反应过来。
<br/>　　而就是这短短时间，不足一秒。
<br/>　　火岩巨兽和阴风蝠妖已经回过神来。
<br/>　　那恐怖的火焰龙卷，再次袭来。
<br/>　　现在再撤退，已经来不及了。
<br/>　　萧逸眼神一冷。
<br/>　　他早就知道，顾长空此人，心狠手辣，不早日除掉，必会后患无穷。
<br/>　　今日，应了他的猜测。
<br/>　　但是，要他死，还没那么简单。
<br/>　　“孽畜，死…”萧逸只能使用易老留给他的剑气。
<br/>　　易老说过，他的剑气，无论在什么情况下，都必能保自己一命。
<br/>　　这道剑气，一个念头便能发挥。
<br/>　　然而，他却没有发现，阴风蝠妖的凶残目光上，邪魅的气息，陡然一闪。
<br/>　　一瞬间，一股莫名的力量，袭击了萧逸的心神。
<br/>　　萧逸的脑海，猛的一白，而后便失去了意识。
<br/>　　与他有相同遭遇的，还有数千名逃得慢的家族武者。
<br/>　　他们可不像裂天剑派弟子，有宝物防身。
<br/>　　他们的逃跑速度，远不如裂天剑派的弟子。
<br/>　　当他们失去意识后。
<br/>　　火焰龙卷，顷刻袭来。
<br/>　　包括萧逸在内的所有人，毫无反抗地被火焰龙卷吸摄。
<br/>　　最后，直入火岩巨兽那张开的巨大嘴巴中。
<br/>　　“咕。”火岩巨兽打了个饱嗝。
<br/>　　它那高如山岳的身躯，浑身冒火。
<br/>　　体内更是蕴含岩浆，炙热无比。
<br/>　　被它吸入腹中的一切，最后的结果，只会是在岩浆中融化，化为一阵轻烟。
<br/>　　“萧逸。”除了剑堂长老外，六大长老，纷纷色变。
<br/>　　毫不犹豫地舍下本身对付的妖兽，立刻朝火岩巨兽而去。
<br/>　　只是，五级妖兽，何等狡猾。
<br/>　　见六位强者疯了般攻来，毫不犹豫地逃离。
<br/>　　不过数个呼吸间，已经逃入妖兽森林，不见了踪影。
