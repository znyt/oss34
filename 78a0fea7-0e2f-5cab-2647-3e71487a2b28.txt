        诺筱颖觉得自己躺着也中枪，真的很冤！
<br/>
<br/>        她收拾东西，离开公司的时候，小组长杨阳好心过来给她践行。
<br/>
<br/>        “筱颍，我听说，我们公司新上任的女老板的名字叫苏漫雪！你说，我们这个女老板会不会就是你的那个闺蜜呀？”
<br/>
<br/>        临别前，小组长杨阳覆在她耳边，小声地八卦着。
<br/>
<br/>        诺筱颖怔了怔，淡然地笑了笑：“是不是，都已经与我无关了。”
<br/>
<br/>        “那你好自为之噢！别泄气！”小组长杨阳咧嘴一笑，握着拳头给诺筱颖做了一个加油的手势。
<br/>
<br/>        诺筱颖点点头，微微一笑。
<br/>
<br/>        这一刻，算是她最狼狈的时候。
<br/>
<br/>        好不容易才找到的一份带薪的实习工作，却没想到这么快就结束了。
<br/>
<br/>        诺筱颖回头看了看这家装饰装修公司，心里突然有丝不舍。
<br/>
<br/>        这是她正式步入社会，人生第一次奋斗的地方。
<br/>
<br/>        与此同时，在某个深山老林里扎营的傅夜沉，不顾自己身上有伤，仍旧在跟敌人周旋。
<br/>
<br/>        大少奶奶只喜欢一切高消费的东西，比如：高档化妆品、珠宝首饰、名牌衣服、名牌香包、名牌高跟鞋！——刘管家。
<br/>
<br/>        傅夜沉看完刘管家发过来的信息后，不由地压低了眉头。
<br/>
<br/>        难道，是他看走眼了？
<br/>
<br/>        哎，算了。走眼就走眼吧！再怎么说，他们已经有过肌肤之亲，这也是他唯一一个有感觉的女人，他又能怨谁去？
<br/>
<br/>        大不了，将来等他退役了，他便负责赚钱养家，苏漫雪负责貌美如花便是。
<br/>
<br/>        “四少，秃鹰他们下一步，会选择什么方式走私？”战友韩剑锋的问话，拉回了傅夜沉的思绪。
<br/>
<br/>        傅夜沉收好手机，单指点了点地图上的一条河，果断道：“秃鹰一定会选择这条河水运！但是，他还会选择这条旱路作为掩护运输。”
<br/>
<br/>        他的判断，从来都没有失误过。
<br/>
<br/>        唯独失误在对一个女人的判断上！
<br/>
<br/>        傅夜沉本想着等这次任务结束后，回去买件她喜欢的同时又比较特殊的礼物送给她，于是询问了刘管家关于苏漫雪的喜好。
<br/>
<br/>        既然苏漫雪喜欢的都是那些常见得不能再常见的奢侈品，傅夜沉觉得自己还不如给她苏漫雪一张刷不完的信用卡会更省事。
<br/>
<br/>        事实也确实如此，当刘管家接到傅夜沉的指令后，替苏漫雪办了一张信用卡交给苏漫雪时，苏漫雪拿着那张信用卡，兴奋得欢天喜地，尖叫了好几回。
<br/>
<br/>        甚至，她还喜极忘形地对刘管家又搂又抱，一点贵太的矜持也没有。
<br/>
<br/>        被苏漫雪弄得非常尴尬的刘管家，不禁嗤之以鼻，并莫名其妙地对苏漫雪心生厌恶。
<br/>
<br/>        这种拜金女，压根就配不上他家大少爷！
<br/>
<br/>        因为有了钱，苏漫雪很快就在临海城的名流圈里混出了头，还结交了不少千金名媛和贵族公子。
<br/>
<br/>        这人脉一广，苏漫雪挥金如土，做什么事都非常顺心。
<br/>
<br/>        但，她的心却一直轻松不下来，因为诺筱颖的这根刺，一直扎在她的心脏上。
<br/>
<br/>        她一定要想办法，让诺筱颖在临海城混不下去。。
<br/>
<br/>        被公司辞退后，诺筱颖消沉了几天，然后重整旗鼓的去找工作。
<br/>
<br/>        可谁知，她就像瘟疫一样，那些公司一听到她的名字，就说不要，到最后，她不仅没有找到工作，回到出租屋里，就连房东都拿涨房租来逼迫她。
<br/>
<br/>        可是诺筱颖哪有多余的钱来垫付房租费，无奈之下，只好退了出租房，然后去投奔男友何明旭。
<br/>
<br/>        临海城有市中心，也有县区。
<br/>
<br/>        何明旭的大学，就在县区里，她从市中心坐公交车过去，要转三趟车，花两个半小时才能到哪儿。
<br/>
<br/>        等到诺筱颖拖着行李箱下了公交车，站在“临海大学”的校门口时，天都已经黑了。
<br/>
<br/>        男生宿舍是十点钟关门，诺筱颖事先并未给何明旭打电话，而是直接找了过去。
<br/>
<br/>        何明旭的宿舍她知道在哪儿，以前，每个周末，她一有空就会过来帮他洗衣服，以至于这里的宿管阿姨都知道她了。
<br/>
<br/>        而何明旭的另外三个室友也都认识诺筱颖，并且对诺筱颖都非常地友好，把她当妹妹看待。
<br/>
<br/>        当诺筱颖拖着行李箱，站在何明旭寝室门口时，宿舍里的三个男生看到诺筱颖后，立即连游戏都不打了，一个个全都凑过来，十分热情地欢迎她过来。
<br/>
<br/>        因为每次过来，不仅仅只是帮何明旭洗了衣服，就连何明旭他这三个室友的衣服，在何明旭的要求下，她也一并洗了，而且还把他们的寝室里打扫得干干净净的，所以，他们都很喜欢她。
<br/>
<br/>        此时，剪了一个球头，皮肤稍微黝黑的章海昌，非常殷勤地问候：“筱颍，今天不是周末啊！怎么也过来了？还带着这么大一个行李箱！”
<br/>
<br/>        方浩博搬来凳子，让诺筱颖坐下歇一歇。
<br/>
<br/>        戴着黑框眼镜，文质彬彬的马智杰则给诺筱颖倒来了一杯凉开水：“筱颍，你喝口水。”
<br/>
<br/>        “谢谢啊！”诺筱颖端过水，微笑着坐下，目光却四下看了看，“怎么不见阿旭？”
<br/>
<br/>        一提起何明旭，三个人相互交换了眼神，停顿了一下，章海昌才笑嘻嘻地说：“他去上自习了！”
<br/>
<br/>        “是、是啊！”方浩博有些支支吾吾地接话。
<br/>
<br/>        马智杰却皱了下眉头，不吭声。
<br/>
<br/>        诺筱颖总觉得他们三好像有什么事情瞒着她，于是从自己手提包里掏出手机，准备给何明旭打电话的时候，马智杰突然走过来，将她的手机给夺走了。
<br/>
<br/>        “电话就不用打了，我直接带你去找明旭吧！”马智杰忽然冷冷地说。
