　　“那我干脆就要那一套，四百平的。我看打折后好像是两千五。”
<br/>　　林峰干脆要了套更大的。
<br/>　　毕竟他们人不少，一百一十平，这么多人住还真得够呛。
<br/>　　收起七阶的，只拿出十颗五阶的外加四百颗三阶的。
<br/>　　他身上五阶一共十几颗，剩下的就留着修炼，这七阶的也是。
<br/>　　“苏云老师，为了让你塞内裤的时候不被人偷走，我可是倾家荡产了。”
<br/>　　林峰把晶石给了销售人员后眼巴巴的看着苏云。
<br/>　　苏云翻了个白眼都不知道该说些什么了。
<br/>　　“不过林峰，我还真的很感谢你。我知道你在努力给大家一个家庭。所以老师今天可以考虑一些事情……比如说…………”

<br/>　　“亲自下厨！”苏云笑道。
<br/>　　“你个小机灵鬼，说得好像你之前不是亲自下厨一样。”林峰白高兴一场。
<br/>　　手续一切办理完。
<br/>　　从此。林峰也算是有房一族了。

<br/>　　“林峰，真佩服你，可以买得起这么大的房子。”刘峰羡慕极了。
<br/>　　他依然还只是五阶的实力。
<br/>　　“后生可畏啊。”其余几个战士会的干事一个个也都眼巴巴的羡慕嫉妒恨。
<br/>　　大家都是干事，这差距可越来越明显了。他们还要住在组织提供的宿舍，而林峰都可以搬进自己买的新房子了。
<br/>　　“林峰，你现在实力不只是五阶了吧？”
<br/>　　寒鸦笑道。
<br/>　　“嗯，前几天突破，六阶了。”林峰依然逢人留三分。
<br/>　　“六阶？我去，现在林峰是战士会的第四高手吧？“
<br/>　　其余干事都羡慕极了。
<br/>　　他们依然是五阶。
<br/>　　“六阶？”
<br/>　　寒鸦笑了笑没有多说什么。
<br/>　　寒鸦等人留下礼品后就离去了。
<br/>　　接下来就是林峰一家子人自己庆祝了。
<br/>　　一桌子的水果，一桌子的洽洽瓜子，一桌子的泡面，红烧的香辣的，还有一桌子的香飘飘奶茶，外加一些烤肉串猪排，想怎么吃就这么吃，想怎么喝就怎么喝。
<br/>　　虽然寒酸了点，但放在现在，绝对是满汉全席级别的，多少人也就晚上做梦想想罢了。
<br/>　　房子也买了，林峰的一个小目标也算是达成了。
<br/>　　然而第二天。恐慌在平民区内传播。
<br/>　　有不下于二十个幸存者死了。
<br/>　　这一次死不是病死饿死而是被吸血而死。
<br/>　　脖子上出现了被牙齿咬过的两个小洞。
<br/>　　“吸血蝙蝠？”
<br/>　　林峰听得这个消息时也是意外。
<br/>　　“这是怪物做的吗？”
<br/>　　陈勾等人都大感意外。
<br/>　　“会不会是有人故意杀了人，然后伪装成这样的。”苏云道。
<br/>　　林峰倒觉得，没有人这么无聊，十之八九就是吸血蝙蝠之类的玩意做的。
<br/>　　“这几天大晚上的还是不要出去，注意安全。”
<br/>　　林峰说了句就直接起身出去找空投了。
<br/>　　刚下楼就发现一个意外的人。

<br/>　　这个骚蹄子今天穿了个超短裙配一双长筒袜。这家伙，就从来没有看到过她穿过一次长裤的。

<br/>　　“你怎么知道的。”林峰皱起眉头。

<br/>　　“呵呵，那人家还真是没有误会错。”林峰笑呵呵道，这个古月的确怎么看都像是卖肉的小姐。特别的骚。

<br/>　　“你那个风哥哥呢。”
<br/>　　林峰没功夫和对方谈情说爱似的逗嘴。
<br/>　　“不知道，不过应该就在附近保护我吧。”古月道，“对了，烤肉小哥哥，你去哪里啊。”

<br/>　　林峰说完就离开了。
<br/>　　“真是的。”
<br/>　　古月跺了跺脚嘟囔了一句。

<br/>　　基地外，林峰捡走了今天的几个空投，几天下来，融合了两个C级空投，林峰存着没用，他准备搞一发大的，来一发B级空投。

<br/>　　有战士会的人认出了林峰顿时引起了人群之中许多人的惊呼。
