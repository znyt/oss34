　　不错，这是一门圣阶法诀。
<br/>　　一路看下来，萧逸大致明白了这些剑力法诀的存在。
<br/>　　说白了，便也是功法的一种罢了。
<br/>　　不过，正常功法是修炼各种元力；而剑力法诀，则是修炼剑力。
<br/>　　萧逸合上了这本剑力法诀。
<br/>　　随后，手中元力轻动。
<br/>　　空气中，一个个字体，凭空而现。
<br/>　　字体，自成篇幅。
<br/>　　肉眼观之，一抹冲天剑意扑面而来。
<br/>　　这是萧逸的剑意。
<br/>　　而这些字体，事实上是萧逸背下的丁家庄那本器谱的内容。
<br/>　　这些内容与字体，如今由他剑气凝聚而出。
<br/>　　“通明器谱。”萧逸看着这些字体，自语一声。
<br/>　　丁家庄的这本器谱，名唤通明器谱。
<br/>　　同样的，它也是一门功法，不过是修炼剑力的功法罢了。
<br/>　　萧逸看了几眼，摇了摇头。
<br/>　　随后大手一挥，散去了字体。
<br/>　　这本通明器谱，他是记下了内容。
<br/>　　但功法本身的韵味和玄奥，却是无法记下的。
<br/>　　一本功法，之所以需要特定用书籍记录下来；便是因为撰写这本书籍的人，本就掌握了这门功法。
<br/>　　由此写下，书籍上，自带武道玄妙，让得观看者修习起来更加容易。
<br/>　　当然了，这通明器谱，毕竟不是自己的东西。
<br/>　　故那时即便丁秋月热情地向将器谱给他，他也并未要，只是单纯记下内容。
<br/>　　事实上，没有书籍上拥有的武道玄妙力量，单纯记下内容，其效果本身极差。
<br/>　　不过无所谓了，萧逸只是大概了解一下剑力是什么存在。
<br/>　　真正能否修习出剑力，还是靠自己的本事，而非一本书籍能左右的。
<br/>　　只是，另外一点让萧逸惊讶的是，这门通明器谱，其品阶绝对不低。
<br/>　　他以前虽未修习过剑力法诀，但功法的强弱和品阶高低，他还是能分出来的。
<br/>　　这本通明器谱，其品阶远远高过了那位分殿主赠予他的那门剑力法诀。
<br/>　　萧逸甚至可以确定，以这本通明器谱的品阶，就算是圣皇境巅峰层次的武者，都休想拿得出手。
<br/>　　那么，很显然，这本通明器谱，原本绝非丁家庄这等小势力之物。
<br/>　　结合之前那六个老者的话，这通明器谱，很显然乃是器宗之物。
<br/>　　至于器宗。
<br/>　　其实，这也是萧逸特地要去风刹殿查一下情报的原因。
<br/>　　早在他看到通明器谱之时，他便已确定了这门法诀品阶极高。
<br/>　　那么丁家庄这么一个最强者不过是圣皇境一重的小势力，怎么可能会拥有这等功法？
<br/>　　而且，那六个老者，尽皆是武道大能层次。
<br/>　　丁秋月这么一个小丫头，怎么可能会卷入这等层次的纷争中？
<br/>　　在风刹殿大致查阅了一番情报后，萧逸才明白了过来。
<br/>　　剑域之内，有两大势力。
<br/>　　一为器宗，一为气宗。
<br/>　　二者的传承岁月，异常古老，绝不亚于中域十八府。
<br/>　　但同时，这两个势力，平日又极为低调，甚至是神秘；宗内武者，极少离开剑域。
<br/>　　故萧逸之前并未听说过这两个宗门。
<br/>　　当然，这两个宗门，在剑域可是如雷贯耳。
<br/>　　二者的区别，只在于一个修炼的是器剑力；另一个，则是修炼气剑力。
<br/>　　剑力，作为剑域特有的力量。
<br/>　　但器剑力一脉，与气剑力一脉，之间的争斗，却持续了无数岁月。
<br/>　　器宗，随便六位老者出来，便是六个武道大能。
<br/>　　这等势力，能拿出通明器谱这样的功法，倒也正常。
<br/>　　只是，萧逸始终想不明白，丁秋月这个小丫头，怎会卷入两脉纷争之中。
<br/>　　寒光府，乃是气剑力一脉的势力。
<br/>　　“罢了。”萧逸摇了摇头，也并未深究。
<br/>　　若是日后有缘再见，能帮的话，萧逸会帮一把。
<br/>　　摇了摇头，萧逸收敛了心神。
<br/>　　他现在，要开始尝试一番修炼剑力了。
<br/>　　两本剑力法诀，同门器谱，修的是器剑力；而另一本，那位分殿主所赠予的，修的是气剑力。
<br/>　　二者剑力，截然不同。
<br/>　　通明器谱，阶品更高。
<br/>　　但另一边，由于有原本在，故书籍上有武道力量，修习起来容易些。
<br/>　　萧逸思索着，也选择着。
<br/>　　半晌，笑笑。
<br/>　　他适合修炼哪种剑力，现在还不知晓呢。
<br/>　　据丁秋月所说，武者，会对特定一种剑力比较敏感，修炼起来也比较容易。
<br/>　　萧逸试了试，便率先从气剑力开始。
<br/>　　毕竟这本功法修炼起来比较容易。
<br/>　　气剑力，需得调动体内元力，感悟这天地之力。
<br/>　　待得体内元力与天地之力产生共鸣，那时，便有可能衍生出剑力。
<br/>　　萧逸按照功法，开始闭目修炼。
<br/>　　半个时辰后。
<br/>　　萧逸的额头，开始汗水密布，无奈下，只得睁开了眼睛。
<br/>　　“不行，丝毫无法感悟到这天地中有半分剑力的意味。”
<br/>　　“呼。”萧逸深呼吸一口气，擦了擦额头汗水，再次闭目。
<br/>　　这一次，他打算试试修习器剑力。
<br/>　　按照通明器谱的功法，萧逸再次开始了修炼。
<br/>　　又是半个时辰，萧逸再次汗水密布。
<br/>　　他明显感觉到，无论是修炼气剑力，还是修炼器剑力，当他尝试沟通天地之力，试图寻找那一抹剑力的意味时，都会存在一种异常强悍的压迫感。
<br/>　　这种压迫感，仿佛天地之威。
<br/>　　而他要感悟的剑力，仿佛无处不在，充斥天地，又仿佛天地之中，渺小如粟，万不存一。
<br/>　　“该死，我便不信了。”萧逸眼眸一冷，擦了擦额头汗水，再次闭目苦修。
<br/>　　一个时辰后。
<br/>　　“噗。”萧逸一口腥血喷出。
<br/>　　看了眼天空，脸色一白。
<br/>　　这天地之威，就当真这般可怕？
<br/>　　属于剑力的一丝武道存在，就真的这般难以感悟？
<br/>　　任何外来武者，妄图修习剑力者，无一成功？
<br/>　　萧逸咬了咬牙，无论是体内元力，还是他那强悍肉身，他进阶尝试过。
<br/>　　可无一例外的，丝毫无法感知到天地中的剑力意味。
<br/>　　强行感悟之，只是反噬受伤罢了。
<br/>　　“不对。”萧逸双眼一眯，忽然想起了什么。
<br/>　　他记得，丁秋月说过，一个纯粹的剑修，身体的任何一部分都可以修炼出剑力。
<br/>　　体内元力，以及肉体，他都尝试过了。
<br/>　　但…还有武魂。
<br/>　　武魂能否修炼出剑力，萧逸不知晓。
<br/>　　他也没有这方面的法诀。
<br/>　　他的两本法诀，尽是修炼气剑力与器剑力。
<br/>　　但，未尝不可一试。
<br/>　　毕竟，他需要做的，只是以武魂沟通天地之力，试图找出那一抹剑力的存在。
<br/>　　萧逸当即闭目，催动体内武魂。
<br/>　　几乎是这一瞬间。
<br/>　　轰…一股天地轰鸣，凭空而现。
<br/>　　“怎么可能。”萧逸猛地睁开了眼睛，瞳孔一缩。
<br/>　　指尖，一抹乳白色光芒悄然凝聚。
<br/>　　那，便是剑力。
