<br/>“迷幻草魔兽？”李振邦感觉一阵阵的无力。<br/>　　<br/>　　普通的迷幻草并不是药材，而是一种制作幻阵的材料。成熟的三百年迷幻草才有可能进化成三阶植物魔兽，虽然是三阶魔兽，但是智力已经相当于人类七八岁孩子的水平了。<br/>　　<br/>　　迷幻草魔兽的攻击并不像其他魔兽一样，是物理或者魔法攻击，而是单纯的精神攻击。<br/>　　<br/>　　迷幻草魔兽的本体很弱小，外形看起来和普通的小草没有什么区别，但是它释放的幻境却不容小觑。<br/>　　<br/>　　即便是强大的白银战士和四级魔法师在不小心的情况下，都可能中招。精神力却强大，抵抗力也就越强。李振邦虽然精神力比同级别人要强大很多，但还远远达不到四级魔法师的水平。<br/>　　<br/>　　迷幻草魔兽的幻阵不过是比较低级的，靠的是自身释放的气息和花粉来使侵入领地的人产生幻觉。虽然是低级的，但是对李振邦这帮人来说那绝对算是致命的，幸好迷幻草并没有要杀害他们的意思，只是将他们困住原地打转而已。<br/>　　<br/>　　“振邦，你是准备抓住它还是放过它？”<br/>　　<br/>　　“算了吧！上天有好生之德！它既然没有伤害我们的意思，我们又何必去伤害它呢？多多你带着我们出去吧！”李振邦犹豫了一下，叹了口气，放弃了抓取迷幻草魔兽的诱惑。<br/>　　<br/>　　要知道普通迷幻草是制造幻阵法阵时的必不可缺的材料之一，所以价格要比普通的药材高很多。<br/>　　<br/>　　迷幻草魔兽更是不可多得的好东西，从它身上采集的材料作为阵眼，那幻阵会强大很多。尤其是活着的迷幻草魔兽，那珍贵程度可是无法用金钱来衡量的了。<br/>　　<br/>　　多多从后天袋中钻了出来，站在李振邦肩膀的多多眼神复杂的看了他一眼，眼神中多出了一些说不清的东西。<br/>　　<br/>　　当多多从后天袋钻出来的一瞬间，李振邦清晰的感觉到幻境中有一股轻微的带着一丝恐惧的灵魂波动。李振邦知道，只是迷幻草魔兽看到多多后产生的恐惧，多多可是说是一切法阵的克星。<br/>　　<br/>　　李振邦带领着伙伴们在多多的指引下，走出了幻阵，当李振邦快要离开幻阵的时候，明显感觉到一股带着放松气息的精神波动，中间似乎还掺杂着一丝感激之情。<br/>　　<br/>　　走出幻境之后，众人感觉眼前一黑，刚刚幻境中还是大白天，出来的时候经天已经快要黑了。当李振邦跟大家解释完之后，众人明显松了一口气，那可是幻境，太危险了！<br/>　　<br/>　　不过紧接着大家就开始埋怨张大友的乌鸦嘴了，如果不是张大友的乌鸦嘴说要碰到一只植物魔兽娱乐一下，也许就不会出现这种情况了。<br/>　　<br/>　　张大友也一阵懊悔，打起来百分百的精神，一句话也不敢再说了。幸好刚才是有惊无险，迷幻草魔兽对他们也没有恶意，否则他可要愧疚一辈子了，虽然不一定和他有关，但是他还是佷自责。<br/>　　<br/>　　大家看到张大友情绪很低落，也不再开玩笑了，劝慰了他几句，并没有什么明显效果。<br/>　　<br/>　　之后两个小时的时间，大家都显得很沉默，同时也很紧张，生怕再遇到什么未知的危险。李振邦倒是不惧怕幻阵，但是也总要防范其他的特殊情况的。<br/>　　<br/>　　天很快黑了下来，精神紧绷两个多小时，众人都感到一阵疲惫，依然是沙含山和蓝乐水配合做好小土屋，众人钻了进去，想要早点休息一下，可是却没有人能睡着。<br/>　　<br/>　　大家都躺在地上，大眼瞪小眼的，连火都没有生。进入澳加森林一个多星期了，由于李振邦拥有着多多，他们并没有遇到太大的危险，这一次他们算是真正感到了森林的恐惧。<br/>　　<br/>　　如果不是史丹利和李振邦发现的早，谁能保证不会出现什么意外呢？<br/>　　<br/>　　不知道过了多久，众人都有些迷迷糊糊的时候，突然听到外面一阵杂乱的脚步声。<br/>　　<br/>　　七个人几乎同时睁开眼睛，互相对视着，没有一个人说话。所有人都看向了李振邦，此刻的李振邦早已经成为了这个小队的领袖。<br/>　　<br/>　　李振邦冲着大家做一个稍安勿躁的动作，悄悄的从土包上扒开一个小洞，偷偷看着外面的情况。<br/>　　<br/>　　外面应该是两拨人，一拨人衣着华丽，装备精良，六名男孩两名女孩，应该是贵族子弟队伍。另一拨人穿着朴素，身上有一些脏乱，呼吸急促，有七个人，其中有一名女孩，应该是平民队伍。<br/>　　<br/>　　“一百零八号小队是吧？赶紧把你们的空间戒指交出来。放心，我们只拿取我们需要的东西，其他的我们不会动的。”衣着华丽的一方，一名长的比较猥琐看着大概有十岁左右的小男孩微笑着说道。虽然他在笑，但是没有人觉得他的笑温暖，反而觉得浑身发冷。<br/>　　<br/>　　“你们跟踪我们一路，还突然对我们动手，你们到底想干什么？这可是学院的野外集训，你们不能乱来。”平民队伍中，一名头发火红色的少年一手捂着胸口剧烈的喘息着说道。<br/>　　<br/>　　很显然他受了一定的内伤，听语气很可能就是对方偷袭造成的。<br/>　　<br/>　　“不想干什么，就是想看看你们空间戒指中，有没有我们想要的材料。”猥琐的小男孩转动了一下手中的镶嵌着不知名宝石的匕首，嘿嘿笑着说道。<br/>　　<br/>　　“我们凭什么给你们看？这是我们的收获！”<br/>　　<br/>　　“凭什么？就凭谁拳头大，谁说话硬气！”猥琐男冷笑着说道。<br/>　　<br/>　　“哼！有本事你们放下你们的装备，凭借装备打赢我们算什么本事！咳咳！”少年气愤的叫道，由于激动引动了内伤，剧烈的咳嗽了几声，嘴角甚至泛起一些血沫。<br/>　　<br/>　　“哼！穷人就是穷人，输不起啊！实相的就痛快的把戒指交出来，不要逼我们自己动手。”<br/>　　<br/>　　“别做梦了！我们是不会把戒指交给你的！”平民队伍中唯一的女孩扶着受伤的男孩激动的叫道，虽然声音中充满了愤怒，但是声音清脆婉转，非常好听。<br/>　　<br/>　　不好意思章节打错了，应该是六十三打成六十四了，六十四章马上奉送。<br/>　　<br/>　　<br/><br/><br/>
