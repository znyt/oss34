　　第89章
<br/>　　秦蛮和杜茂打得难解难分，一时间谁也奈何不得谁。
<br/>　　此刻仓库的弓箭已经被取来，蛤蟆寨拥有的铁弓正好能武装二十人。
<br/>　　“秦蛮，退。”程大雷抬手指挥身后的弓箭手：“每队十人分开，准备。”
<br/>　　秦蛮连续强攻，逼退杜茂，纵身跃出战团。
<br/>　　“放！”程大雷的手重重放下。
<br/>　　“一队准备，二队……放。”
<br/>　　每十人一组，连续的羽箭射向杜茂和孟子云二人，没有一刻停歇。杜茂挥舞着手中的木板，挡住大部分羽箭，却是且战且退，不能前进半步。
<br/>　　“将他们逼到悬崖，不要让他们往山下冲。”
<br/>　　蛤蟆寨下山只有一条路可走，后山是一道峡谷，深不见底。
<br/>　　杜茂抓着孟子云，一直向后退，终退到后山的峡谷边缘。
<br/>　　这个时候，程大雷阻止手下继续射箭，只是用弓箭瞄准二人。
<br/>　　战斗到现在，杜茂和孟子云身上都已挂彩，幸运的是蛤蟆寨这边没有任何伤亡。
<br/>　　“杜老大，身后已无路，不如加入我蛤蟆寨，大家大碗吃肉，大碗喝酒，岂不快活。”程大雷还真不忍杀了这头懒大虫，假若他能加入蛤蟆寨，山寨的实力至少能上一个台阶。
<br/>　　“姓程的，你少来这一套，老子做鬼也不会放了你！”
<br/>　　程大雷一挥手：“放刘悲。”
<br/>　　刘悲双手掺在一起站出来，视线无悲无喜的扫过悬崖前的二人，眼睛中露出怜悯的目光。
<br/>　　“嗳，何苦呢……”
<br/>　　瞬间，二人就有一种被衰神盯上的感觉，从里到外泛起一股寒意。
<br/>　　杜茂又想起曾经被刘悲恐惧支配的日子，看看刘悲，再看看身后深不见底的悬崖。忽然一抓孟子云的肩膀，身子向后倒去。
<br/>　　“喂，走啦。”
<br/>　　二人直直的跌入悬崖，黑暗中，只有孟子云的悲惨的回声传来。
<br/>　　这一幕，令刘悲也愣住了。他不可思议的挠挠头：“这是怎么会事，我还什么都没说呢。”
<br/>　　“够了，完全够了。”
<br/>　　程大雷拍拍他的肩膀，身子站在悬崖上往下望。
<br/>　　“这样掉下去，恐怕要摔成肉酱吧。”徐神机道：“这悬崖可有上百丈。”
<br/>　　“一般来说……”程大雷顿了顿：“掉下悬崖都是死不了的，说不定还能触发奇遇什么的。”
<br/>　　“我带人下去，看看情况，一定杀了这二人。”秦蛮。
<br/>　　“不必了，所谓穷寇莫追，如果他们这样还能活着，那就是天数，日后有机会碰着再说吧。”
<br/>　　程大雷说完，忽然觉得脸上一凉，他昂起头，看到天空中纷纷雪花落下，铺天盖地，宛若谁在苍穹上撒下碎玉。
<br/>　　是的，下雪了。
<br/>　　这一年的第一场雪，比以往时候来得更早一些。
<br/>　　大雪下个不停，鹅毛大雪落在山脊、树干、屋顶。蛤蟆寨还真有几座屋子被压塌的，也只好趁着大雪抢修。
<br/>　　这样的天气什么也做不了，程大雷和徐神机等人吃了几顿火锅，余下时光也只好陪着柳芷在屋中有一句没一句的逗闷子。
<br/>　　这个世界的时光可是真闲呐。这样的天气不是程大雷想偷懒，而是不得不偷懒。程大雷只好窝在屋里看外面的雪景，无聊是真无聊，却也悠闲。
<br/>　　前生的世界动不动就996，日复一日，为了梦想为了自我实现，把自己忙得像车轱辘一般。而却是从未想过，生而为人，其实没必要那么累的。
<br/>　　直到三天后，大雪才算是停下来。程大雷走出屋子，积雪足有小腿厚，整个世界银装素裹，雪砌冰雕的相仿，当真是琉璃世界玉乾坤。
<br/>　　“好哇好！”程大雷忍不住赞道：“瑞雪兆丰年，明年是个好年景呐。”
<br/>　　“好什么好，有大半年没下雨了，庄稼旱得要死。如今种子刚撒入田，就来了这样一场雪，没出土的庄稼都要被冻死。”说到最后，徐神机不介意嘲笑程大雷的无知：“大当家，你不懂不要乱说好不好，在外面被人听到很容易挨揍的。”
<br/>　　程大雷不介意徐神机的嘲讽，却注意到他话中一条信息。
<br/>　　『大半年没下雨』
<br/>　　“平常年景……不这样嘛？”
<br/>　　“连雨季老天都不曾下过一滴雨，老天爷这是要饿死人呐。”
<br/>　　之前，程大雷从没有注意过天气，此刻才恍然想起，从来到这个世界开始，似乎就未碰上过一次下雨。
<br/>　　今天由徐神机提醒，程大雷才注意到这件事。
<br/>　　望着周围白茫茫的大雪，程大雷忽然心中一动，一个一直没有办法解决的问题，此刻竟有了主意。
<br/>　　之前，程大雷就想去黑石城收割一批恐惧值，只不过两地相距太远，万一碰到黑石城的猛人，想逃都来不及。
<br/>　　这场铺天盖地的大雪，岂不是为自己创造了机会。把马车改成雪橇，马的负重减轻，速度也更快些。
<br/>　　程大雷心心念便是凑够十万恐惧值，来个十一连抽，正好趁着冬闲把这件事完成了。
<br/>　　叫来黄三元，问清楚黑石城的情况。程大雷这是第一次去黑石城，人生地不熟，多做功课总是不错的。
<br/>　　接下来，便是确定这次行动的人选。黑石城对程大雷来说完全陌生，不知藏着怎样的高手，这次行动还是要选几个高手。
<br/>　　程大雷第一个确定下来的人选就是林少羽。林少羽是程大雷遇到的少数几个有隐藏属性的人，这可是一位立志成为大侠的少年，这次行动是惩善扬恶，劫富济贫怎么能少得了他。
<br/>　　由杜茂的事情，程大雷意识到人才的可贵，真有几个顶级人物，也不会让杜茂那样猖狂。刘关张要升级，山寨原有的人也不能耽误，这次让林少羽陪着一起去，说不得能刷些经验值。
<br/>　　之后便是秦蛮，徐神机，刘悲，张肥四人。秦蛮是武力保障，有他在，不管遇到多厉害的猛人，都算不上事。带上徐神机是为了防止他的毒奶祸害山寨，刘悲和张肥自然是为了给他们刷些经验值。至于关鱼……他留在山寨还有事，把无极刀传给大家，为程大雷培养一支大刀队出来。
<br/>　　准备好一切后，一行六人用四匹马拉雪橇，向着黑石城出发。苍茫雪地，出现四道蚂蚁般渺小的黑影。
<br/>　　嘟，触发任务，抢劫城中富户十次。
<br/>　　
<br/>(本章完)
