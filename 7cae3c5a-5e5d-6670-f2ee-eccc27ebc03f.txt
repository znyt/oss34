
郭德江女儿受伤其实挺无语的，这点背的孩子是想去车库取车，卷帘门升起来的时候她刚走进去，门突然就砸了下来，直接把她就给压到了底下，要不是临倒地的时候那一声惊呼被郭德江给听见了的话，这倒霉催的姑娘，可能就得一直被压在下面了，要命到不至于，可挺遭罪的，但饶是如此她身上也被挤的青一块紫一块，然后头骨还砸破了。<br/>　　<br/>　　“郭小姐这是怎么了，这个月可没少出事啊……”旁边的司机和佣人无心的嘀咕了一句，郭德江听着脸色都青了，脑袋里又再次想起了昨天那青年的一番话。<br/>　　<br/>　　“明日，你家孩子若是有什么异样，你摆好姿势登门来找我！”<br/>　　<br/>　　郭德江安慰了女儿几句，告诉她不要出门在家里老实的呆着，他脸色阴晴不定的琢磨了半天，心里始终不太托底。<br/>　　<br/>　　算上这次，他女儿已经连续四次犯险了，偶尔一次你还能不当回事，两次可以算是巧合，但四五次都这样，你就没办法解释了。<br/>　　<br/>　　半晌后，郭德江犹豫了半天，拿出手机翻出了张兵的电话，一开口就挺客气的说道：“小兵啊，呵呵，我是你郭叔……”<br/>　　<br/>　　电话那头，张兵听见郭德江的动静有点发愣的说道：“啊？你，你好郭叔”<br/>　　<br/>　　“那个我问你个事情，昨天吃饭的时候你带的那个年轻人，你知道他在哪么？”<br/>　　<br/>　　张兵顿时又“啊”了一声，郭德江接着说道：“你把他的地址告诉我行么，我找他有点事情”<br/>　　<br/>　　张兵这时候无疑是很懵的，猛然想起昨天临走之前，自己问王惊蛰的那句话，对方告诉他，今天郭德江会主动找过来的。<br/>　　<br/>　　张兵说了下王惊蛰住的酒店，郭德江客气的说了声谢谢，就让司机开车送自己过去了，同时好奇的张兵也往酒店那边去了。<br/>　　<br/>　　中午十一点，王惊蛰的房间外面，就响起了敲门声，他叼着烟头一点都不以外的打开房门，就看见了门口的郭德江。<br/>　　<br/>　　“你怎么知道我女儿今天会出事的？”郭德江僵着脸问道。<br/>　　<br/>　　王惊蛰夹着烟头，淡淡的说道：“姿势不对，重来”<br/>　　<br/>　　郭德江咬了咬牙，捏着拳头语气稍微客气了点：“王先生，您是怎么知道我女儿会出事的”<br/>　　<br/>　　“还是不对，再来！”<br/>　　<br/>　　“你……”郭德江脸色顿时阴了，他得有二十来年了吧，除了有数的几个人，他哪里对人这么客气过。<br/>　　<br/>　　“唰”郭德江的脾气上来了，直接转身就走。<br/>　　<br/>　　王惊蛰弹了弹烟灰，说道：“送你句话，过了你女儿的生日，三天之后她还好不了，到时候我可就不在双城了”<br/>　　<br/>　　郭德江刚走了几步，两腿就顿住了，他缓缓转过身子，惊愕的说道：“你说什么？”<br/>　　<br/>　　“我说你摆的姿势不对……”<br/>　　<br/>　　郭德江僵硬着身子站在原地足足半分钟都没有动弹，然后才又走了回来，有一种礼贤下士的方式，鞠了一躬说道：“王先生，请见谅”<br/>　　<br/>　　王惊蛰手指点着他胸口淡淡的说道：“有钱确实挺牛逼，但有些事你有钱也未必办的了，沈万三都不敢有你这姿态啊，朱元璋问他猪蹄子这盘菜叫什么，他也只敢说一句万三蹄而已罢了，猪蹄这两字是万万不敢说的”<br/>　　<br/>　　郭德江皱眉说道：“你也不是明太祖”<br/>　　<br/>　　“但没我，你女儿也好不了……”<br/>　　<br/>　　郭德江顿时无言以对。<br/>　　<br/>　　郭德江叹了口气，以他的阅历也知道女儿的问题不是偶然的巧合了，他就拱手说道：“王先生，我女儿的问题你要是能解了，你要的那三件东西，我无偿给你都可以”<br/>　　<br/>　　“走吧！”王惊蛰关上房门，跟着郭德江往酒店外走，张兵也正好迎了过来，看见他们两人一前一后的出来，眼神略有惊异的压下了疑问。<br/>　　<br/>　　王惊蛰笑道：“来了兵哥”<br/>　　<br/>　　“唉，唉，你这是……”<br/>　　<br/>　　“没事，过去看看，走吧”<br/>　　<br/>　　出来后王惊蛰没上张兵的车，跟郭德江坐在了一起，他来双城无非就是要那三件古董的，有拿回东西的机会，他就没心思在这浪费时间了。<br/>　　<br/>　　“王先生，我女儿的问题不知道是出在哪里了，之前我也让人看过祖坟和风水，都没什么异样”<br/>　　<br/>　　“你家风水不对，祖坟不稳，你早就倒了，还能轮得到你女儿？你现在问我，我也不太清楚，先去看看她吧”昨日里，见到郭德江王惊蛰从他的面相上，就品出来对方的子女有祸事了，但细节在哪他也不可能未卜先知。<br/>　　<br/>　　车一路往郭德江家方向走，路上的时候，他就跟王惊蛰打听道：“不知道王先生是在哪发财的，晋中的一些行里人我多少也认识几个，但似乎没听说过王先生呢？”<br/>　　<br/>　　“郭老板，我和你之间仅限于这一次，多的你还是别打听了”王惊蛰不软不硬的回了一句。<br/>　　<br/>　　这时候的郭德江其实隐约对自己昨天的态度有点懊悔了，王惊蛰这类人他是挺愿意打交道的，平常有朋友带个风水，阴阳先生去他那，也都是好好款待的，他没想到自己这次似乎是有点走眼了？<br/>　　<br/>　　到了郭德江的家里，他们几人一进门，就看见客厅沙发蜷缩着个苦着脸，手里拿着一堆零食正往嘴里塞的姑娘，那吨位把沙发都给压得陷进去了，整个人都跟沙发完美的融合在了一起。<br/>　　<br/>　　“爸，我要过生日呢，你能不能让我出去啊……”郭德江女儿见他回来了，就从沙发上蹦了起来，拉着他胳膊说道：“人家都在等着我呢”<br/>　　<br/>　　“老实点，你今天哪都不能动”郭德江呵斥了一句，扭头问王惊蛰道：“您看……”<br/>　　<br/>　　王惊蛰扫了眼这姑娘的脸，眼神瞬间古怪起来，这倒霉催的孩子面相颇为怪异，脑袋上面紫气压顶，两边颧骨过高异常尖锐，嘴唇略薄现青紫色……笼统的讲就是，这是一幅受不起的面相。
