<br/>				    那修士没有防备，整个人猛的后退了数步才停住了倒退的脚步，他眯着眼看着那前面冷着一张脸的绝美女子，有些意外的笑了起来：“姑娘好本事。”<br/>　　<br/>　　    凤九冷冷地扫了他一眼，收回目光，将脚下的沙子扫干净后，穿上了靴子。<br/>　　<br/>　　    那修士好歹也是见世过面的，以他仙圣巅峰的修为，自然不会像色鬼一般看见美人便失了理智的猛凑上前，尤其是，在对方衣袖拂出的那一下之后，更是谨慎了几分。<br/>　　<br/>　　    如果说先前还有几分猎艳的心思，那么现在，那份心思便被他压了下来，重新审视着眼前的这名绝美的红衣女子。<br/>　　<br/>　　    这人的实力修为难道是隐藏了？<br/>　　<br/>　　    这个念头一起，他觉得有些不可思议。这女子的岁数不大，骨龄也摆在眼前，这样年轻的女子，实力难道已经逆天？说什么他却是不相信的。<br/>　　<br/>　　    只不过，因刚才那一下，他眼下也没有对她动什么心思的念头。<br/>　　<br/>　　    凤九穿好靴子后，便朝海边走去，手中一翻，那艘小船出现在手掌之中，她用银针刺破手指滴入鲜血认主后，便将那小船往海面一抛。<br/>　　<br/>　　    只见那小船瞬间变大，浮动在那海面之上迎着水波起伏。就在她脚尖轻点要跃上那小船之时，老者的声音也在这时急急传来。<br/>　　<br/>　　    “小姑娘，小姑娘，你这是要走了？”老者从树林中快步走出来，看着站在海边的凤九，以及那艘在海面停留着的小船，不禁有些不舍。<br/>　　<br/>　　    她走了，他可就很难再吃到这些好吃的东西了。<br/>　　<br/>　　    “老伯，我要走了，那些东西就留给你吧！好好保重，告辞！”她拱手一礼说着，脚尖一点，跃上那小船。<br/>　　<br/>　　    那一旁的修士一见，目光微闪，视线落在那小船上面。他回过头来，看了老者一眼：“阁下就是守海者？”<br/>　　<br/>　　    老者瞥了那修士一眼，连话也懒得跟他多说，转而来到那小灶边，看着那些东西，再看着那已经远去的身影，心下不由的浮上一股说不出的感觉。<br/>　　<br/>　　    他在这里守着这片海这么多年，还从没有人给他做过吃的，这个小姑娘，是唯一的一个。<br/>　　<br/>　　    想到这，他看着那渐渐远去的小船和那红色身影，当下，神识一动，隔空传音给她。<br/>　　<br/>　　    “小姑娘，蓬莱仙岛就在这片海域之中，只不过，它的出现却似海市蜃楼，飘渺不似人间所有，据我所知，这蓬莱仙岛每半个月会出现一次，离上次出现也就还有几天的时间了，你要好好把握，另外，这片海域的海怪是会攻击小船的，你也要小心了。”<br/>　　<br/>　　    听着那传音入耳的话语，凤九微讶，她站在小船上回头朝那岸边看去，见那老者站在海边注视着她，不由的露出一抹笑容来。<br/>　　<br/>　　    她伸出手，朝那老者挥了挥手，喊着：“老伯，待我回来，再请你吃好吃的。”<br/>　　<br/>　　    听到这话，老者也不禁笑了起来。<br/>　　<br/>　　    一旁的修士见此，便快步往那海边而去，提气一跃，准备追上凤九的小船……<br/>　　<br/>
