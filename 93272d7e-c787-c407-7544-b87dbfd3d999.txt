　　另一边，楚风眠跟五行刀祖的交手，已经陷入了僵局。
<br/>　　碰！
<br/>　　刀剑碰撞，这五行刀祖，被暂时的逼退了出去。
<br/>　　但是他身上的气息，却是丝毫没有紊乱，纵然是楚风眠打出了无数剑术，这五行刀祖，都足以应对。
<br/>　　双剑齐出，楚风眠已经是暂时的压制住了五行刀祖，占据了上风。
<br/>　　想要压制追这五行容易，但是想要击败这五行刀祖，却是无比困难，轰杀五行刀祖，更是几乎没有可能。
<br/>　　如今楚风眠明白了想要轰杀九劫古帝的难度，要比起轰杀一位八劫古帝，难上百倍。
<br/>　　除非是现在楚风眠的实力再进一步，不然的话，根本杀不了这五行刀祖。
<br/>　　“这样僵持下去，不是办法。”
<br/>　　楚风眠目光闪烁，他心中已经没有在这跟五行刀祖战下去的想法了。
<br/>　　如此僵持下去，只会一直拖延下去，回头要是这宫殿之中的宝藏，被其他人得到，可就麻烦了。
<br/>　　而且一旦是这样拖延下去，也会还会引来其他的九劫古帝。
<br/>　　一名九劫古帝，现在楚风眠都难以对付了，要是再来一个，那楚风眠要考虑的可不是击败了，而是应该考虑如何才能全身而退了。
<br/>　　楚风眠心中思考这，突然之间一股天地之力，降临了。
<br/>　　楚风眠的脸色猛然一变，这天地之力的降临，代表又是一名九劫古帝，来到了独桥之内。
<br/>　　“是齐林山的九劫古帝，麻烦了！”
<br/>　　楚风眠感受着天地之力，从中察觉到了其中的功法，正是齐林山的功法。
<br/>　　窥神镜也向楚风眠发出了警告。
<br/>　　“好，果然有人来了，小子，你的末日到了！”
<br/>　　那五行刀祖兴奋的喊道，他也同时感觉到了有人到来，脸色变得无比的兴奋，现在齐林山，五行天，沧海都三方势力，在暗地里已经是联合在了一起。
<br/>　　这齐林山古帝的到来，正是给他了一大助力。
<br/>　　他一个人，正好是难以轰杀楚风眠，但是在加上一名九劫古帝的话，情况可就截然不同了。
<br/>　　现在他只要在拖延住楚风眠，让楚风眠无法脱身，等到这齐林山的九劫古帝到来，他们二人就可以联手轰杀。
<br/>　　“五行封锁，天地牢笼！”
<br/>　　五行刀祖猛然一抓，这五行之力，便是化为了五根铁索，向着楚风眠四面八方包围而来。
<br/>　　这五根铁索，扎根于虚空之中，将周围的空间，都化为了一道牢笼，死死的困住了楚风眠。
<br/>　　五行之力，弥漫在这牢笼之上，几乎是以这五行刀祖的全力，将这天地化为了一座牢笼。
<br/>　　“不好！”
<br/>　　楚风眠心中也清楚无比，这五行老祖的想法。
<br/>　　现在对付这一个五行老祖，楚风眠还可以从容应对，要是一旦是在来一名九劫古帝的话，那可就无比麻烦了。
<br/>　　尤其是在这独桥之上，想要逃走，都无比困难。
<br/>　　楚风眠已经是感觉得到，那齐林山九劫古帝的步步逼近。
<br/>　　瞬息之间，楚风眠便是不再犹豫，突然手中剑锋，指向天空之上。
<br/>　　漫天星辰，仿佛在这一刻被引动了一般，一股股的星辰之力，凝聚在了双剑之上。
<br/>　　“贪狼，巨门，禄存，天机！”
<br/>　　刹那之间，四道剑光同时打出，随即而出的，便是剑气化为了一道星河，冲着周围五行铁索化为的牢笼，轰击而去。
<br/>　　这九玄剑术的前四式，楚风眠同时斩杀而出。
<br/>　　在楚风眠踏入到了八劫古帝之后，在施展这九玄剑术，也耗费的力气，也轻松了许多。
<br/>　　四剑齐出，星河之力凝聚，万千剑气轰击在了那五行铁索后化为了牢笼之上。
<br/>　　轰隆隆！
<br/>　　一阵阵轰鸣之下，这五行铁索化为的牢笼，被当即打出了一个缺口，剑气席卷。
<br/>　　这其中的一道铁索，率先破碎，随即便是连锁反应，其他的四道铁索，一同蹦碎。
<br/>　　“仙术！”
<br/>　　五行刀祖看到这剑光星河的一刻，脑海之中也蹦出了这两个字。
<br/>　　能够爆发出如此威力的，已经不是剑术可以做到的了，只有仙术才行。
<br/>　　“星辰之力，九玄仙术！九玄仙尊的洞府，居然是被此子得到了？”
<br/>　　五行老祖的脑海之中，突然想到了之前葬仙大陆开启的事，上一次葬仙大陆开启最大的一件事，便是九玄仙尊洞府开启。
<br/>　　五行天的古帝，也深入其中的去抢夺，但是最终进入其中的五行天古帝，都被斩杀，没有一人能够活着离开。
<br/>　　以至于最后是谁得到了九玄仙尊洞府，得到了九玄仙术，一直都是一个谜团，看到今日楚风眠施展出来的剑术。
<br/>　　五行老祖一下子明白了过来，这绝对是被楚风眠所得到。
<br/>　　“不好，这小子要逃！”
<br/>　　刹那之间，五行老祖在惊讶之中醒悟过来，一股灵力猛然打出，打算去加固这五行牢笼。
<br/>　　但是随着武道铁索的破碎，这五行牢笼的崩塌，已经是无法阻拦。
<br/>　　楚风眠的身影，从这五行牢笼之中，当即是飞了出去，向着那宫殿的方向，直接冲了进去。
<br/>　　“怎么回事？五行刀祖？你在跟谁交手？”
<br/>　　突然之间，一道身影从独桥之中当即飞了出来，这是一名中年男子，皮肤已经是呈现出了一种碧绿的颜色，跟寻常人，已经是截然不同。
<br/>　　浑身上下，这中年男子，都散发出了一种雄厚的灵力，一身力量，丝毫不弱于那五行刀祖。
<br/>　　“碧螺老祖，来的正好，跟我联手杀了此人，此人就是圣龙宗的那楚风眠！”
<br/>　　五行刀祖看到这来人，脸色大喜，这就是他一直在等待的援兵。
<br/>　　齐林山的九劫古帝，碧螺老祖。
<br/>　　“什么？楚风眠？就是那圣龙宗的新人？八劫古帝？怎么可能，此人不是古帝吗？”
<br/>　　这碧螺老祖，听到五行老祖的话，猛然看向楚风眠，打量了一眼楚风眠的境界，突然吃了一惊道。
<br/>　　五行天，齐林山，沧海都三方势力，打算联手要毁灭了圣龙秘境，毁灭了圣龙宗，对所有圣龙宗弟子，自然都有着研究。
<br/>　　楚风眠的名字，他也是听闻过的，不过在他看来，楚风眠只是一个准帝，所以根本没有放在眼中。
