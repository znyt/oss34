        &lt;h3&gt;第三卷第八十七章  寒伯安和郝景文商讨&lt;/h3&gt;
<br/>
<br/>        郝景文西城之旅收获满满，看到了大家昂扬的斗志，他觉得形势一片大好，心中一种自豪感和责任感油然而生，他必须而且有能力带动他们走向更高的境界。
<br/>
<br/>        他在他的qq签名上这样写着：我觉得我现在已经跳脱了，从一个境界升华到了另一个境界。
<br/>
<br/>        寒伯安将安男送回家后，让江俞轩帮着找另外一只手镯的下落，便回了上海，一回到寒家，忙得不知道所踪，安男也开始进行她的科研，俩个人两个城市，一份牵挂。
<br/>
<br/>        他的签名：
<br/>
<br/>        “男：虽然外面阳光灿烂，可是我还是没有感觉到温暖，因为你不在我身边，今天又是想你的一天。”
<br/>
<br/>        安男艾特他：“琪：空气因为有你而甜，生活因为有你而美满，每分每秒都是想你的天！”
<br/>
<br/>        陈子昂将她的思路发给寒伯安兄弟三人之后，便着手耗材的事情，元月份将整个耗材推向市场，这将是耗材市场的一次重大的改革。
<br/>
<br/>        她计划铺满整个市场，实行全店联动，一炮打响。
<br/>
<br/>        这一段时间她发动所有的人的积极性，亲自上阵，会同各地区的店长及经理做着上市前的准备。
<br/>
<br/>        有些地方他们有经销商，有些地方自己投资开店，有经销商的地方是轻车熟路，自己开店的地方要带着新的店长去从头做起，陈子昂忙得就像是一只飞旋的陀螺，停不下来。
<br/>
<br/>        寒伯安他们都看到了陈子昂的方案，但是因为陈子昂到处飞，一直时间确定不下来，他们也只能等着陈子昂。
<br/>
<br/>        郝景文到了上海。
<br/>
<br/>        在寒伯安那豪华的有着各种收藏的办公室里见到了寒伯安，除了原来的格局没有变之外，多了一张丽人照。
<br/>
<br/>        郝景文看到丽人照片会心一笑，大哥这是抱得美人归了。
<br/>
<br/>        “大哥这样放着美人图，是想天天看着安男嫂子吗？”郝景文笑着说。
<br/>
<br/>        寒大总裁的办公室是内外两层，外间是会客的，正常处理文件的地方，内间是只有像郝景文这样的人才能进。
<br/>
<br/>        “看着她我就更有工作的动力了。”寒伯安毫不顾忌郝景文还是一只单身汪。
<br/>
<br/>        “如果是我，我就会让安男嫂子直接辞职陪着我。”郝景文不怀好意的说。
<br/>
<br/>        “别激将我，没用，我就喜欢和你安男嫂子搞个小资，下班开车去给她一个惊喜。看着我的如花美眷舞动着三千秀发飞奔向我的情景，那将是世界上最美好的画面。”
<br/>
<br/>        “我，不是来吃狗粮的。”郝景文酸酸的。
<br/>
<br/>        “没事，你还小，你也会有的。如果你有喜欢的，你可以告诉我。”寒伯安笑着说。
<br/>
<br/>        “告诉你干嘛？”郝景文不解。
<br/>
<br/>        “我帮你追啊。”
<br/>
<br/>        “当真？”
<br/>
<br/>        “当然是真的。”
<br/>
<br/>        “那，大哥，我喜欢安男嫂子。”
<br/>
<br/>        郝景文看着寒伯安幸福的模样，刺激一下寒伯安。看来还是得找个女朋友了，被人经常这样喂狗粮，心里多少还是有点羡慕的。
<br/>
<br/>        “眼光不错，不过我已经帮你追到手了。哈哈哈。”寒伯安笑。
<br/>
<br/>        “不开玩笑了，说，有什么事情？”寒伯安知道郝景文是有事情特意来找他。
<br/>
<br/>        “陈子昂给你发的邮件你看到了吗？”
<br/>
<br/>        “看到了，不过这一段时间轩轩和陈子昂不是都很忙吗？大家坐一起的机会不多，等他们忙完这一时期我们几个人坐下再谈。”
<br/>
<br/>        “那你看陈子昂的方案如何？”
<br/>
<br/>        “结合现在的情况，能满足客户的需求，但是陈子昂现在的身份和地位也不一样了，我们虽然定位的起点不一样，但是也需要相应的不同的档次，就像你去餐馆一样，可以有选择的套餐。”寒伯安说。
<br/>
<br/>        “大哥的意思还需要有套餐？分档次？”
<br/>
<br/>        “我也就是设想，能不能实现还需要陈子昂他们的研发团队来进行研发。”
<br/>
<br/>        “我们可以再提我们的需求，我这次去西城收获颇丰，尤其是从哲的服务，让我刮目相看。大哥你觉得呢？”郝景文问寒伯安。
<br/>
<br/>        “从哲的服务是从心底发出来的，好像他本来就这么做，这个是一种服务态度。而且他对人的态度不亢不卑。”寒伯安接着说：“我出国去考察了一下市场，那边的服务态度也很好，司机都比较绅士，但是商业化气息太浓。”
<br/>
<br/>        “我在想，我们的第二个事业拓展开之后，让从哲给大家培训一下，然后给客户服务。”郝景文说。
<br/>
<br/>        “你说的很对，我们要区别于传统的行业，其实更重要的一点是在服务上，现在人们的生活水平提高了，消费水平也提高了，各自生活需求也会提高，当人们出门享受到不一样的待遇，虽然是自己花了钱，但是心里是舒坦的。我们要人家愿意花钱，花舒服钱。”寒伯安笃定的说。
<br/>
<br/>        “文文，公司的大小事务都是你在打理，很辛苦吧？”寒伯安问郝景文。
<br/>
<br/>        “辛苦是有点，但是看着事业红红火火，哈哈哈，我还是很有成就感的。”
<br/>
<br/>        “来，让我看看你的地图。”
<br/>
<br/>        “不用看了，全满了。”
<br/>
<br/>        “哈哈哈，你厉害，比我预期的要早了三分之二的时间。”
<br/>
<br/>        “其实，这个跟轩轩有很大的关系，陈子昂他们的项目在全国大部分城市都有，轩轩的人脉关系很广，他每到一个地方就会推动一个地方的进度，比我们安排人去跑点轻松很多。”郝景文想到了自己在家乡拓展业务的时候，微微苦笑了一下。
<br/>
<br/>        人家说熟人好办事，这几句好像也是要分情况的，有些人看着表面一团和气，你好我好，还不知道在后面怎么给你捅，刀子呢。
<br/>
<br/>        “轩轩回家了，怕也不轻松吧？”
<br/>
<br/>        “房地产业现在呈上升趋势，但是每个行业的竞争也很激烈，不过他们好像走的是国际化，而且他们的品牌已经打出来了，相对来说能轻松一点吧，况且俞琪姐姐已经独挡一面很多年了。”
<br/>
<br/>        “我前几天还让他帮我找一个物件，到现在也没有回我。”
<br/>
<br/>        “我从西城回来不久，他最近忙得一塌糊涂，他们公司要参与西城一个城区的规划建设。”
<br/>
<br/>        “去西城有什么感觉？”
<br/>
<br/>        “文化底蕴很浓厚，毕竟是十三朝古都，但是市容市貌不敢恭维，不过西城现在正在建设中，据说要建设成国际化大都市。哈哈，轩轩也算是为西城的发展做出贡献了。给我印象最深刻的还是西城的人，憨厚、淳朴，粗犷、豪爽。”
<br/>
<br/>        “我对他们的感觉是：精明！不过我接触的人不多。”
<br/>
<br/>        “嗨嗨，被你带偏了，我是来和你商量什么时候软件开发的事情，陈子昂估计在等我们的消息。”
<br/>
<br/>        “据我所做，陈子昂准备将耗材推向市场，等那一天我们去给她助个阵，顺便讨论一下软件的事情。”
<br/>
<br/>        “大哥原来早有打算。不过陈子昂如果在同一天将耗材推出去，是需要大量的资金的，他们目前已经有很多项目在进行中。”
<br/>
<br/>        “你怎么知道这么多？”
<br/>
<br/>        “陈思宇告诉我的。”
<br/>
<br/>        “陈思宇？”
<br/>
<br/>        “他们公司的财务副总。”
<br/>
<br/>        “文文，你厉害啊，手都升到陈子昂的公司内部去了。”
<br/>
<br/>        “不是，我们有工作上的来往。”
<br/>
<br/>        “这么重要的信息告诉你了？”
<br/>
<br/>        “我们app的软件到了第三次收费期了，虽然我们是股东，但是还是得支付一定的费用，陈思宇来找我，说最近资金紧缺，陈子昂忙得到处飞，江俞轩又不见了人影，她和凤盛夏忙得焦头烂额了。”
<br/>
<br/>        “问一下轩轩看看这事情怎么办？”寒伯安说。
<br/>
<br/>        “好的。”郝景文心里也有和大哥一样的打算，只是他没有说出来。
