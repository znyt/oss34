　　这中年男子看着楚风眠走了过来，开口道。
<br/>　　“这老骗子卖的东西，都奇贵无比，而且几乎都是假的，之前已经有好几个人，上当受骗了，你可不要去买了。”
<br/>　　那躺着的老者，缓缓的睁开了眼睛，听到这中年男子的话，冷哼了一声说道。
<br/>　　“哼，老夫的东西，可都是好东西，难得一见的宝物，怎么说是骗人。”
<br/>　　“你还说你不是骗人？之前荒古门的一位准帝，从你这里买了一件灵器，带了回去，都说是假的，难道荒古门的鉴定大师，还有错？”
<br/>　　中年男子也同样一声冷哼说道。
<br/>　　说完随即转过头来，看向楚风眠劝解道。
<br/>　　“这老骗子的东西，就连荒古门的鉴定大师都说是假的，你可千万别买。”
<br/>　　“我先看看。”
<br/>　　楚风眠随意笑了笑道。
<br/>　　这中年男子的话，的确也是一片好意，不过楚风眠可不觉得，这一本古籍，是假的。
<br/>　　毕竟这一本古籍，可是对楚风眠的内心都有着极大的吸引力。
<br/>　　这一点从楚风眠走过来之后，他感觉得便是更加明显了，无论如何他也要得到这古籍。
<br/>　　说完楚风眠便是走了过去，离近了，仔细的打量着这一本古籍。
<br/>　　“哎。”
<br/>　　中年男子看到这一幕，叹了一声气。
<br/>　　但是他跟楚风眠无亲无故，提醒就好了，至于阻止楚风眠，他也不浪费力气，继续是吆喝着，卖着他摊位上的东西了。
<br/>　　“这一本古籍，怎么卖？”
<br/>　　楚风眠仔细的打量了一眼这一本古籍。
<br/>　　以他的见识，都看不出这古籍到底是什么来历，可是这古籍至少是经历了数十万年，甚至是上百万年，都没有腐朽。
<br/>　　单单是从这一点，楚风眠就能感觉出这古籍的不凡。
<br/>　　“又有人要买老骗子的东西了。”
<br/>　　“还真有人信这老骗子的话？”
<br/>　　“又一个撞破南墙不回头的。”
<br/>　　“这老骗子的东西倒是古怪，就连帝尊都看不出真假。”
<br/>　　“没准这小子只是好奇问问，要知道这老骗子的东西，可都是天价啊，这小子可都未必买得起。”
<br/>　　楚风眠站在了这老者的摊位前，不少往来的武者妖神，都是目光看了过来，互相交谈着。
<br/>　　显然是对着一幕，已经是不陌生了，这老者卖的东西，价格奇高，而且几乎都是没有任何一点用处的东西。
<br/>　　一开始还有人光临他的摊位，到后来几乎就没有人愿意停留。
<br/>　　这老者也就躺在哪里，时而拿出一件东西摆放在摊位前，没有人的时候，就躺着摊位后面睡觉。
<br/>　　在这归墟城内，不允许战斗，不然就是对整个归墟城的挑衅，所以这老者一直就在这里，也无人敢让他走。
<br/>　　久而久之，这老者到也算是归墟城的一个名人了，人人都知道这样一个老骗子。
<br/>　　就连七大宗门的一些强者，都从他这里买走了不少的东西，可最终却是一无所获，不是扔了，就是都贱卖了。
<br/>　　谁从这老者手中买东西，就是冤大头，这一点便是整个归墟城的共时。
<br/>　　现在看到楚风眠又一个冤大头来，自然围观起来了不少的人。
<br/>　　“这古籍，怎么卖。”
<br/>　　楚风眠又重复道。
<br/>　　老者听到楚风眠接连询问，才缓缓的抬起头来，看了楚风眠一眼，开口道。“十万皇品灵石，一口价。”
<br/>　　嘶！
<br/>　　听到这老者的报价，在场围观的众人都是屏住了呼吸。
<br/>　　十万皇品灵石，这简直是一个天文数字，只怕是许多不朽之境的无上老祖，都拿不出来这个数字。
<br/>　　只有准帝，甚至是真正的帝尊，才有可能拿得出来，绝对是一个天价。
<br/>　　这个价格，都足以在通天阁之类的商会里，买到许多宝物了。
<br/>　　“漫天要价，这老骗子疯了？”
<br/>　　“这么一本破古籍，十万皇品灵石？十万皇品灵石，都购买天级武技了，还要你这破古籍？”
<br/>　　几名武者更是毫不客气的说了出来。
<br/>　　这个价格，的确是太高了，就算是让一名准帝，甚至是真帝来，一口气都难以拿出如此之多的财富来。
<br/>　　“老骗子，你的价格怎么又高了？之前不还是一万，现怎么就十万了？”
<br/>　　人群之中，一名武者看着老者不由的说道。
<br/>　　上一次他见人询问的时候，价格还是一万皇品灵石，现在却已经是十万了。
<br/>　　“爱买就买，不买就走，不要打扰老夫睡觉。”
<br/>　　老者仿佛是没有听到众人的议论，盯着楚风眠，挥了挥手道。
<br/>　　“不要着急。”
<br/>　　听到这老人赶人的话，也没有在意，笑着说道。
<br/>　　“十万皇品灵石是吗？我买。”
<br/>　　“买？”
<br/>　　“十万皇品灵石，买这样一本破古籍？”
<br/>　　听到楚风眠的话，一群众人都是目瞪口呆。
<br/>　　这简直太疯狂了，这样一本破古籍，就连鉴定大师都认为是假的，现在楚风眠居然要买？
<br/>　　而且可是十万皇品灵石啊，就算是十万极品灵石，都算是一笔大钱了，十万皇品灵石，许多准帝都难以积攒出的财富来。
<br/>　　这样的挥霍财富，就算是从七大宗门之中出来的弟子，都不敢这样的大手笔。
<br/>　　冤大头！
<br/>　　无数目光看向楚风眠，就如同在看向冤大头一样。
<br/>　　敢花如此之大的代价，买人尽皆知骗子的东西，就算是在愚蠢的人，都不会这么做。
<br/>　　“十万皇品灵石，这小子拿得出来吗？”
<br/>　　一名年轻男子有些嫉妒的说道。
<br/>　　说楚风眠是冤大头，可他们可就连做冤大头的机会都没有，十万皇品灵石，围观众人几乎无人可以拿的出来。
<br/>　　就在众人的议论之中，楚风眠手心一动，一大团灵石对着那老者飞了出去，一共十万枚皇品灵石，一枚不少。
<br/>　　看着眼前呈现晶莹的皇品灵石，人群之中不少人的目光都是露出了几分贪婪。
<br/>　　这么多的皇品灵石，如果他们得到，购买丹药，足以提升好几重的境界了。
<br/>　　不过归墟城纪律森严，还没有人敢光天化日之下动手抢夺。
<br/>　　那老者看到这么多的皇品灵石，脸色都没有任何的变化，只是清点了一下，便是将这些灵石，放入到了空戒之中，随后将古籍扔向了楚风眠。
