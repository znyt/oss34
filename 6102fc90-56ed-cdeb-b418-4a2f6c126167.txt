　　“Marcus，Alicia，Kim。”
<br/>　　躺在破屋地板上，满脸血污，重伤的K国学员对破屋中三个同伴说道：“把我留下，你们快走吧！”
<br/>　　“不行！”Marcus立刻拒绝道：“我们不能丢下你不管！”
<br/>　　Marcus是想要带着重伤的Riggs一起离开，但是旁边的Alicia和Kim目光却是不断闪烁着。
<br/>　　带着重伤的Riggs一起离开显然会拖累到他们，他们只有三个人，两把膛线都快要磨平了的AK，两把枪的子弹加起来只有五十多发，就这样的装备，只是三个人想要在人多势众、弹药充足而且还有猎狗帮助的反政府武装势力手下逃脱都是一件很困难的事情，更别提带着重伤、连路都走不了的Riggs。
<br/>　　Riggs如何能不明白Alicia和Kim的沉默代表什么意思，Alicia和Kim闪烁的目光如何能瞒过Riggs？
<br/>　　他们几个人一起进入“炼狱”接受这个世界上最残酷的军事训练，他们平时训练在一起、吃住在一起，Riggs如何能看不穿Alicia和Kim目光闪烁的背后代表着什么？
<br/>　　“Marcus，不要犹豫了。”Riggs对Marcus说道：“你们带着我，一个都逃不了，我留在这里至少还能吸引猎狗的注意，给你们争取逃跑的机会，带着我就是给猎狗一个追踪源，根本不可能逃得掉！我不知道你们最后是不是能够从这片‘无政府区域’逃出去，不知道你们是不是能够活下去，但是活着至少还有希望不是吗？”
<br/>　　Riggs所说的道理，Marcus如何能不明白，只是他实在难以接受把Riggs丢在这里！
<br/>　　“汪汪汪！”，“汪汪汪！”
<br/>　　猎狗的叫声更近了，这也就意味着反政府武装人员也距离他们更近了。
<br/>　　“快走！”Riggs对着Marcus说道：“赶紧走，再不走，我们都走不了了！”
<br/>　　Marcus摘下挂在他身上的AK想要塞给Riggs，但是却被Riggs拒绝了，“你们比我更需要它，你把枪留给我这个将死之人有什么用？”
<br/>　　对于破屋中的四个人来说，食物他们总有办法可以找到，相对于食物，他们更紧缺的是武器弹药和药品。没有武器弹药，他们无法跟盘踞在这片“无政府区域”的武装势力抗衡；缺少药品，他们身上伤口会感染引发各种并发症，消耗他们的体力和战斗力，最终死在逃亡的路上。
<br/>　　Marcus拿回塞给Riggs的AK重新挂在身上。Marcus此时最想要做的事情，就是回到训练基地，然后询问缘由，当然前提是他们能够活着从“无政府区域”离开。
<br/>　　砰！砰！砰！砰！砰！砰！
<br/>　　“嗷呜！”
<br/>　　破屋外的街道上，突然传来一阵急促的手枪声，急促的手枪声中还夹杂着猎狗被子弹击中倒地时发出的哀嚎声。
<br/>　　突兀的枪声令得破屋内的四个人感觉到诧异，从枪声和外面交战的动静判断，这急促的手枪声不是来自反政府武装那边，如果是反政府武装那边，子弹的运动轨迹应该是破屋这边，而现在他们所藏匿的这间破屋根本没有遭到子弹的攻击。
<br/>　　Marcus拿着AK悄然移动到破屋墙壁边，透过墙壁上的缝隙往外看去，看到了令Marcus一辈子都难以忘记的一幕。
<br/>　　街道上，一个穿着风衣，带着牛仔帽的白人男子，手中握着两把银色的手枪在那些反政府武装人员中穿梭，他身形脚步无比敏捷，就像是一道幽灵一样，让人难以捕捉到他的运动轨迹，每一声枪响都会带起一片血花。
<br/>　　而在在风衣牛仔帽男子身边还跟着一个人，这个人全身包裹在一件宽大的长袍中，长袍遮住了他的头，看不清他的脸，他的手中也没有枪，有的只是一把长柄镰刀，就跟网络上很多描绘死神的图片中，死神手里所拿的那把镰刀一样。
<br/>　　看似笨重的长柄镰刀在这个长袍男手中显得十分轻盈，被他舞的密不透风，每一刀挥下，就会有一个反政府武装人员倒地。他们或是胸口被镰刀划开内脏流了一地，或是直接被镰刀一刀斩在脖子上，脑袋从脖子上掉落下来，滚到街道边，然后鲜血呈箭状从脖子平整的切口处喷射而出。
<br/>　　比起风衣牛仔帽男人优雅的杀人方式，握着镰刀的长袍男所过之处都会刮起血雨，宛如地狱修罗场，而长袍男就是修罗场中收割着他人生命的修罗！
<br/>　　即便是Marcus在看到街道上这无比血腥的一幕都觉得自己空荡荡胃部，胃酸在不断的翻涌着，想要涌上他的喉咙。
<br/>　　街道上的反政府武装人员数量是风衣牛仔帽男人和长袍男的几十倍，但是却被他们两个人杀得毫无还手之力。
<br/>　　一名反政府武装人员惊惧的想要对着风衣牛仔帽男人扣动扳机，可是还没等他扣下扳机，一道寒光从他面前闪过，长袍男的镰刀落在他手中的AK上，旧AK枪管被镰刀整齐的削断，紧接着旧AK发生了炸膛，把这名反政府武装人员的双手炸得血肉模糊，还没等到这名反政府武装人员因为炸膛炸毁了双手而尖叫，又是一道寒光从他眼前闪过，只是这道寒光却成了他看到的最后的东西，因为在下一秒，他的脖子上就多了一条平整的血线……
<br/>　　几十个反政府武装人员片刻后只剩下了那个戴着一顶满是污渍红色贝雷帽的指挥官，风衣牛仔帽男人手中银色手枪顶在他的眉心处，让这个在几分钟前还凶残无比的反政府武装势力指挥官连连后退，直到被逼到墙边，无路可退。
<br/>　　“你、你们到底是谁？”反政府武装势力指挥官颤抖着声音，眼中充满着恐惧看着自己眼前这个皮肤白皙，手指就跟钢琴家一样纤长漂亮的男人。
<br/>　　“‘暗月’！”
<br/>　　风衣牛仔帽男人眼睛中没有任何感情色彩的看了眼被吓得已经尿裤子的反政府武装势指挥官，潇洒的转身，说道：“堕天使！”
<br/>　　反政府武装势力指挥官正以为自己捡回了一条命的时候，堕天使手中的银色手枪响了，反政府武装势力指挥官眉心多了一个暗红色的弹孔，子弹从他后脑勺穿出钻进他身后的墙壁中，墙壁上留下一蓬夹杂着骨头碎屑的血浆。
