　　听到电话中司徒空的话，佟千雪一下子就从床上坐了起来，神情也是变得激动起来。这些天佟千雪的情绪一直很低落，也正是因为这件事情，尽管医生没有在她面前多说什么，可是正如赵天宇所说得那样，佟千雪自己就是医生，医生对病患隐瞒病情那套手段她又如何能不了解？
<br/>　　佟千雪早就熟悉的不能再熟悉了，因为她也做过同样的事情，所以只根据医生的只字片语以及说话时的语气神态就能猜个八九不离十。
<br/>　　“你先别激动。”
<br/>　　司徒空在电话那头耐心的对佟千雪说道：“司徒家的祖先曾经在御医院供职，也留下过相关的方子，但是我现在只是根据目前了解的情况做出初步判断，无法确定你的情况，而且我在中医药理上学到的也只是皮毛。你最好还是去东海市一趟，让我家里人帮你确诊，我家里那边我已经打过招呼，你直接按照我说得地址过去，再说我的名字就可以。”
<br/>　　佟千雪用笔记下了司徒空所说的地址后，好一会儿才抚平了内心的激动，用自己手机订了两张今天晚上去东海市的高铁票。
<br/>　　因为当天杭山市飞往东海市的航班已经没有了，要不然佟千雪绝对会选择时间最近的航班，直接飞到东海市去。
<br/>　　慕容飞没有多问佟千雪什么，没有人告诉他关于佟千雪的情况，可是他心中又何尝会不清楚，只是不愿意提及，在佟千雪的伤口上再撒一把盐罢了。
<br/>　　两个人收拾了简单的行李，当天晚上就做高铁去了东海市，抵达东海市的时候已经是晚上八点，这个时间点也不适合再去司徒空所说的地方去找他家里人，所以慕容飞和佟千雪就在东海市找了一家便捷酒店住了下来。
<br/>　　因为压在心中的事情放下了一些，再加上两个多小时的高铁，本身身体还没完全恢复的佟千雪在快捷酒店住下后，洗了个澡就睡着了。
<br/>　　慕容飞从电视柜上拿了一瓶酒店赠送的瓶装饮用水，拧开瓶盖刚喝了一口，他放在床头柜上的手机就是震动了一下，提示有一条未读短信。
<br/>　　放下手中的瓶子，走到床头柜边，拿起手机打开短信。短信是一个未知号码发来的，上面只有一句话，让慕容飞到短信上所说的地址见面。
<br/>　　慕容飞试着按照短信上的号码回拨回去，可是对方直接选择了拒接。慕容飞看着手机中的短信微微皱眉，看了眼躺在床上已经睡着的佟千雪，最后拿起衣服还是走出了酒店房间。
<br/>　　慕容飞对东海市不了解，所以选择在便捷酒店外打了一辆车，前往短信上所说的地方。
<br/>　　短信上的所提及的地上是一家网咖，距离慕容飞所居住的便捷酒店不算远，也就是一个出租车起步价的路程。
<br/>　　网咖网吧只有一字之差，但是档次却是差了好多，这里没有网吧烟雾缭绕，也没有网吧中打游戏时歇斯底里的吼叫声，而且以女性顾客居多，当然这里的消费也要比网吧贵很多。
<br/>　　大厅一个小时要8块钱，包厢的价格就直接翻了个倍。
<br/>　　慕容飞按照短信上所提及的包厢号，找到了对方要跟自己见面的包厢，包厢中的灯光不算明亮，只能看到里面有一个戴着鸭舌帽的男子正在电脑前玩着游戏。
<br/>　　推门走进包厢，听到从音响中传来的声音，慕容飞才这个坐在电脑前的男人正在玩的游戏居然是CS，这款曾经在网吧中风靡一时的游戏。
<br/>　　一个小时16块钱，用这么豪华配置的电脑玩CS，慕容飞实在是不知道这个坐在电脑前的男人是怎么想的。
<br/>　　“哎呀，又死了。”男人推开键盘，叹了口气说道：“到底是老了，反应跟不上了。”
<br/>　　男人抬头看到站在包厢门口的慕容飞，微微一笑，指了指自己对面的位置说道：“来了？坐！”
<br/>　　借着包厢中昏暗的灯光，看清楚坐在电脑前戴着鸭舌帽的男人的脸，慕容飞差点没吓得跌倒在地上，“野……”
<br/>　　慕容飞刚开口，鸭舌帽男人就示意他噤声，并且让他在自己对面的位置坐下，说道：“来，先陪我玩两把，过过瘾。”
<br/>　　慕容飞无奈，只能打开游戏，进入鸭舌帽男人建的局域网，陪着他玩起了自己早就已经从电脑中删除掉的CS。慕容飞现在的确是不玩游戏了，可是游戏的功底还在，有了慕容飞的助阵，鸭舌帽男人很快就连赢了好几盘，过足瘾后，满意的说道：“让你们这些小子欺负老人家，我找了帮手还不打得你们屁滚尿流！”
<br/>　　“野……，赵先生，您特意来东海市并且选了这么个地方跟我见面，搞得跟地下党似得，是为了什么事情？”慕容飞对坐在自己对面的鸭舌帽男人问道。
<br/>　　“无常，我们认识有多少年了？”鸭舌帽男人问道。
<br/>　　“我18岁入伍参军，19岁进入师部侦察连，您也是在师部侦察连发现的我，今年我已经34了，我们认识15年了。”慕容飞回答道。
<br/>　　“准确的说，是15年零45天。”鸭舌帽男人纠正道：“想想时间过得还真快，一转眼你都已经到了从作战部队强制退役的年纪了。当年我见到你的时候，你还是冒头小伙子，一心想要进特种部队，对特种部队充满了憧憬。现在进了比军区特种部队更牛的特种作战部队，有什么想法？”
<br/>　　“想听真话？”慕容飞问道。
<br/>　　“屁话！”鸭舌帽男人骂道：“要听假话我还要问你？”
<br/>　　“真话就是，如果可以让我重新选择，我当年肯定不会跟你走，甚至都不会想着要去特种部队，就在部队混个两年，然后选择退役，读完大学回家娶妻生子，平平凡凡的过日子。”慕容飞回答道：“这个世界上也‘无常’这个狙击手，没有他的狙杀记录。”
<br/>　　鸭舌帽男人对慕容飞的回答丝毫不感到意外，仿佛从一开始就知道了慕容飞会这么回答自己的问题一样。
