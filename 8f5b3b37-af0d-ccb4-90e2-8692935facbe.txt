　　第525章
<br/>　　牛背上的大汉表情凶恶，模样恐怖。说话像破锣一般，刺耳难听。
<br/>　　但在此刻，所有人的目光都聚在他身上，周围雅雀无声，落针可闻。
<br/>　　“姓程……”
<br/>　　那年轻商人皱起眉头，看看他的黑牛，看看他的斧头，一下睁大了眼睛。一张脸变得苍白如纸，似乎三魂七魄都在一瞬间被抓走。
<br/>　　刚才他看到黑河九鬼，像大山中逢到猛虎。但此刻的表情，却像是看见惨死的尸体又打开棺材爬了出来。
<br/>　　周围其他人的表情也没有好到那里去。
<br/>　　而这个时候，程大雷已经牵着黑牛，不吭不响的走到一个角落里。
<br/>　　独角鬼皱起眉头，道：“莫非你是……”
<br/>　　大汉脸上的表情瞬间收在一起，冷声喝问：“你既然已经知道了，何必再问！”
<br/>　　就是这一声喝，独角鬼手下的弟兄人人惶恐，握刀的手都在颤抖。
<br/>　　“不，不可能！”独角鬼捧起拳头冲天空摆了三摆，道：“天下谁不知道，那一位在凉州，怎么可能会出现在这里？”
<br/>　　大汉用手扶着胡须，道：“本当家要去哪里，难道还要事先通知你一声？”
<br/>　　独角鬼心里一激灵，若这大汉开口解释，他反而会有些怀疑。因为传说中，那一位是从来不屑给人解释的。
<br/>　　上三眼下三眼将大汉打量一遍，看看他座下的黑牛，看看他手中的斧头。独角鬼心里自然没有完全相信，但是，他不敢冒险。
<br/>　　“原来是程当家到了。”独角鬼赶忙行了一礼：“程当家大人大量，恕我有眼无珠，不识英雄。”
<br/>　　“好啦，好啦。”大汉摆摆手：“不知者不怪。”
<br/>　　“来人，将路与程当家让开，请程当家过桥。”
<br/>　　人群中，程大雷睁大眼睛，难以的看着这一幕，心想：这样也行！
<br/>　　早知道，本大家就早早报名了，何必候在人群里，混一个平平无奇。
<br/>　　那大汉骑在牛背上，大摇大摆的准备过桥。眼睛恨不得长在脑袋上，连看一眼弯腰行礼的独角鬼都不肯。
<br/>　　这时候，大汉身周呼啦啦跪倒一大片人，尤其是那年轻商人，跪在地上道：“求程当家救我们一救，我家老母病重，我赶回家是为见老母最后一面。”
<br/>　　幸亏程大雷躲得远，假若他靠得近，如果不跪下的话，就有些太惹眼了。
<br/>　　此刻他睁大眼睛，查看着这大汉的信息，看到底是何方神圣，冒充我的名头。
<br/>　　姓名：刘发财（招摇撞骗的顶级游侠）
<br/>　　技能：连环刀
<br/>　　隐藏属性：惑众
<br/>　　啧……
<br/>　　程大雷叹了一声，没想到在此地，还能碰见个有隐藏属性的人。这明显是选错了职业呐，有这种属性，为何不当个骗子呢。
<br/>　　刘发财骑在马上，面对那年轻商人的苦苦哀求，连垂下头也不肯。这也的确符合他的人设，想那程当家，杀人如麻，怎么会在乎几条人命。
<br/>　　他就骑着黑牛，大摇大摆的过桥，任身后连声哀求，不肯回头看上一眼。而独角鬼的手下都低着头，但也有人大着胆子抬起头，看上那么一两眼。
<br/>　　在山贼这个行当，程大雷的确不用自谦，说他是天下第一山贼，一丁点也不过分。如果这世上有追星族，程大雷毫无疑问是所有山贼的偶像。如果有可能的话，要个签名合个影也是完全可能的么。
<br/>　　独角鬼同样低着头，眼珠却在骨碌碌乱转。突然，他眼睛一亮，道：“狗头山周当家向程当家问好，他感谢程当家上次出手相助？”
<br/>　　大汉挥挥手，道：“告诉他，我知道啦。老周这个人，还真是……”
<br/>　　独角鬼的头猛地抬起，嘴角浮现一抹冷笑：“混蛋，你露馅了！狗头山根本没有什么周当家。”
<br/>　　那大汉一激灵，脑袋转得极快，已经知道事情不妙，骑着黑牛就往前冲。
<br/>　　“留下他！”独角鬼大喊：“他是假冒的。”
<br/>　　独角鬼的手下反应虽然有些慢，但并不是傻瓜，至少不全部都是傻瓜。一听独角鬼这么喊，那还不明白是怎么回事。
<br/>　　于是所有人一拥而上，将那大汉围在当众，那大汉虽然反应快，但也没能冲过石桥。
<br/>　　程大雷撇撇嘴：瞧瞧，我的名头就是那么好冒充的么。纵然你有隐藏属性加成，但现在不还是露馅了么。
<br/>　　当然，在巨大压力下，刘发财能做到这一步，也算是不错了。换成程大雷，未必会做得比他更好。
<br/>　　不过，程大雷也不用发展这方面能力，这次他虽然没带斧头，但他有剑啊。
<br/>　　路遇不平，自然可一剑斩之，用得着招摇撞骗么。
<br/>　　倒是要看看这刘发财如何应对。
<br/>　　刘发财被拦在桥中央，抡起大斧头，哈哈大笑。
<br/>　　“没胆的狗贼，竟然敢害本大家，你们不认得本当家，那就与这条大斧说话。”
<br/>　　大斧挥过，就是人头滚滚落。到此时此刻，他还嘴硬，倒也是个人物。
<br/>　　程大雷刚才已经确认过，这货是顶级实力，在江湖上也不算弱手。所以一时半刻，程大雷并没有出手。
<br/>　　是的，程大雷已经决定要出手。他这次出山，一是为了替关鱼访找大夫，二也是为了访一访江湖上的高手，收到山寨中，壮大蛤蟆城的实力。
<br/>　　这刘发财明显是个人才么，武艺还算过得去，倒不值得程大雷看重，关键是他的隐藏属性。将他拉到山寨，完全可以做个宣传委员什么的，帮自己聚拢人心。
<br/>　　刘发财本事不弱，可他仍旧没能力闯过眼前的石桥。因为黑河九鬼的老大独角鬼，也同样是个顶级人物。
<br/>　　独角鬼拎起一条大棍，跳起来砸刘发财的脑袋。只是一招，刘发财就在众人围攻下，被击落牛背。他就地一滚，抛下手中的斧头，抓出了腰上的腰刀。
<br/>　　这货，本就是个使刀的。
<br/>　　好汉难敌四手，何况他的本事虽然能算不错，但实实在在称不上强。连环刀使出来，将身周人逼退，他纵身跃入大河中。
<br/>　　水性倒是不错，在激流中还能逆流而上，几下就爬到了岸上。
<br/>　　不偏不倚，爬出来的位置，正好是程大雷的脚下。二人互相看着彼此，大眼瞪小眼。
<br/>(本章完)
