　　“看来这龙魔龙，在九域之中是得到大机缘了，真是运气好。”
<br/>　　“龙摩罗，的确是有着几分希望，踏入半步天人境了。”
<br/>　　“半步天人境，那有那么容易，在这百年间，唯有少帝才踏入了半步天人境，这龙摩罗，还差得远。”
<br/>　　一尊尊魔族，目光闪烁，四处交谈起来，他们也无人敢上前，插入到这战斗之中。
<br/>　　对他们来说，龙摩罗跟夜魔王等人，打到两败俱伤才最好。
<br/>　　少了两方强敌那个时候，他们才可以有机会得到更多的宝藏。
<br/>　　“龙摩罗，你太狂妄了！”
<br/>　　夜魔王听到楚风眠狂傲无比的话，脸色剧变，他本来是想要给楚风眠一个台阶下，他们双方，都可以免生事端。
<br/>　　进入净土佛寺之中，必然会有大战，大战将近，他也不愿意跟楚风眠冲突。
<br/>　　可没想到，楚风眠居然是丝毫不给情面。
<br/>　　他堂堂夜魔王，也是魔族之中有名的强者，这让他怎么能够忍受。
<br/>　　“龙摩罗，既然你狂妄自傲，那今日我们就来领教一下你的实力，看看你到底有没有这个狂妄的本事！”
<br/>　　夜魔王看向了旁边的虚魔王，血魔王，长啸一声。
<br/>　　“动手！”
<br/>　　轰！
<br/>　　夜魔王，虚魔王，血魔王，这三尊魔族王者，同时出手。
<br/>　　夜魔王也知晓龙摩罗的实力，刚刚一招之下，便是击退了血魔王，换做是他，也是一样的结果。
<br/>　　所以一旦出手，决定撕破脸皮，便是三人一同出手，向着楚风眠轰击而来。
<br/>　　三人联手的灵力，从天空之上凝聚，化为了一把魔意大手，对着楚风眠一把抓了过来。
<br/>　　一招打出，周围的群山都在震动，这三尊魔王，可都是九劫古帝巅峰的人物。
<br/>　　联手之下的威力，虽然无法达到半步天人境，但是如此威势，在九劫古帝之中，无人能敌。
<br/>　　魔意大手，铺天盖地，笼罩在魔龙身躯之上，要一把将这魔龙，猛然抓起。
<br/>　　“一群蝼蚁，联手之下，也是废物，给本帝破！”
<br/>　　楚风眠冷哼一声，魔龙身躯，猛然冲天而起，旋转舞动，滚滚龙威降临。
<br/>　　那一道魔意大手，在接近到楚风眠身躯周围的一刻，被龙威无情的碾碎。
<br/>　　“战龙舞苍穹！”
<br/>　　轰隆！
<br/>　　下一刻，澎湃的龙威，更是凝聚成为一条远古战龙的模样，向着那夜魔王三人咆哮而去。
<br/>　　“什么？”
<br/>　　“怎么可能！”
<br/>　　夜魔王三人，脸色同时大变，看着眼前这一幕，惊慌无比，纷纷凝聚灵力，化为了一道屏障，准备抵挡这一条远古战龙的轰击！
<br/>　　咔嚓！
<br/>　　远古战龙猛冲过去。
<br/>　　这凝聚而出的屏障，都是纷纷破碎，夜魔王三人，都是吐出了一大口精血，被逼退了出去。
<br/>　　楚风眠没有放过这三人的打算，在打退了他们三人之后，魔龙之躯又是冲着他们的方向追了过去。
<br/>　　两道龙查，无情的落下。
<br/>　　“快走！”
<br/>　　血魔王看着楚风眠过来，身躯连连闪烁，化为了一道血光，一把抓起了夜魔王，虚魔王二人，化为了一道血光，遁逃了十里之外。
<br/>　　而楚风眠的两道龙爪的轰击，也扑了一个空。
<br/>　　看着逃走的三人，楚风眠目光闪烁，考虑了一下，也没有再追，而是直接向着净土佛寺的方向飞去，第一个冲入到净土佛寺之中。
<br/>　　“这，龙摩罗的实力，怎么会变得如此恐怖了？”
<br/>　　一直是看到那一条魔龙，钻入到了净土佛寺之中，在场的许多魔族强者，才从惊讶之中反应过来，不可思议的说道。
<br/>　　龙摩罗，身为魔龙一族的强者，他的实力强横，自然人人皆知，在场不少人，都跟龙摩罗交过手。
<br/>　　但是龙摩罗虽然强横，可也只是比起蝎狮王等人，高上一头。
<br/>　　对抗血魔王等人，勉强以一敌二，应该已经是极限了。
<br/>　　以一敌三，换做之前，更是不可能的事情。
<br/>　　可今日居然是一招，就将血魔王，夜魔王等人，直接击败了。
<br/>　　“龙摩罗，不能招惹。”
<br/>　　许多魔族强者的心中，都明白了这一点，不过他们也没有停留，而是身形连连闪烁，不少人也接连催动起遁光，进入到了净土佛寺之中。
<br/>　　楚风眠刚刚跟夜魔王等人的战斗，只是一个小插曲，他们的目的，还是为了净土佛寺之中，血邪魔帝的宝藏而来。
<br/>　　至于刚刚的战斗，只是给他们提了个醒，让他们不要去轻易招惹楚风眠。
<br/>　　“没想到这龙摩罗的实力，如此恐怖，若是在净土佛寺中会中出手，只怕是我们都难以全身而退。”
<br/>　　另一边，血魔王等人身上的伤势，俨然是全部恢复，其中夜魔王，看着楚风眠离开的方向，不由的眉头紧锁了起来。
<br/>　　刚刚楚风眠表现出的实力，超出了他们的意料，哪怕是他们三人联手，想杀了楚风眠，都几乎没有可能。
<br/>　　“龙摩罗实力提升如此之快，只怕是得到了惊天的奇遇，我们能够杀了他，得到的好处会更多。”
<br/>　　血魔王舔了舔嘴角，充满贪婪的说道。
<br/>　　刚刚虽然吃了个亏，但是非但是没有打消了他心中的想法，还是让他更加兴奋了起来。
<br/>　　他的实力，便是以吞噬强者的精血提升，吞噬的精血实力越强，他的实力也会提升越大。
<br/>　　“这佛寺之中的危险不会少，那龙摩罗，反正一个人独自行动，我们总能抓住机会，等到他没有防备之时，在杀了他！”
<br/>　　虚魔王也是目光闪烁，充满杀意的说道。
<br/>　　三道身影，交谈了一阵，便是化为了一道遁光，向着净土佛寺之中飞去。
<br/>　　净土佛寺！
<br/>　　楚风眠第一个冲入到净土佛寺之中，在进入净土佛寺的一刻，楚风眠便是又化为了人形。
<br/>　　他刚刚施展出魔龙之身，也是怕对敌的时候，暴露身份来。
<br/>　　魔族强者之中，对龙族有所了解的武者，并不多，所以以魔龙身来施展力量，更是足以蒙蔽住他们。
<br/>　　“杀！杀！杀！所有人都得死！都得要跟我们陪葬！”
<br/>　　轰！
<br/>　　就在楚风眠刚刚步入到净土佛寺之中，还没有来得及观察周围的时刻。
<br/>　　从周围之中，便是浮现出十几名怨灵来，对着楚风眠突然袭击而来。
