　　叮铃铃……，叮铃铃……
<br/>　　桌子上的卫星电话吵个不听，让布克尔特感觉非常烦躁。
<br/>　　罗昊握着手枪，瞥了眼桌子上的卫星电话，眼睛又重新盯在发出莹绿色光芒的准信上，对布克尔特说道：“电话那头的人还真的很有毅力，你真的不打算接听吗？”
<br/>　　僵持不下，继续对峙下去，情况只会对战术中队越来越不利，战术中队的每一个人都知道，堵在外面的那些人不会因为打不开地下室的门就离开，对于布克尔特手中所掌握的这支P病毒，他们可以说是志在必得。
<br/>　　“这么低级的陷阱，你觉得我会上当？”布克尔特面色狰狞的对罗昊说道：“我们就这么耗着，看谁最后先先坚持不住！”
<br/>　　赫克托没有能打通布克尔特的电话，意识到情况的不妙，要求CiA的行动队从外部对脱氧钢板门采取措施，将脱氧钢板门打开。
<br/>　　毕竟是前海豹突击队队员，Royce很快就有了应对的办法。
<br/>　　想要从外部打开这扇脱氧钢板门，除了使用焊枪进行常规的切割外，还有一种更快速的办法，就是用钻孔机在脱氧钢板门上钻出几个孔，然后在孔中填塞塑胶炸药，实施爆破，将这扇脱氧钢板门炸开。
<br/>　　Royce通过无线电，跟比洛多取得联系，“比洛多将军，我现在需要一台钻孔机，还有五公斤，不，我想可能是十公斤的塑胶炸药！”
<br/>　　“东西会在一个小时内送到你们手中。”
<br/>　　比洛多说完放下电话，看着坐在自己对面沙发中的女儿，眼中闪过一抹歉意，但是那抹歉意也只是一闪而过而已。重新拿起电话，开始安排赫克托所交代的事情。
<br/>　　等到比洛多打完电话，塔尼特从沙发中站起身来，瞪着自己的父亲，气愤的说道：“爸爸，我真的对你太失望了！”
<br/>　　“塔尼特，我希望你能听我的解释。”比洛多组织了一下语言，说道：“是来自美国五角大楼的压力，我根本没有任何办法！”
<br/>　　“的确，这份压力很大。”塔尼特点了点头，眼睛依旧充满着失望之色，看着自己的父亲说道：“你可以迫于压力，取消之前的合作；但是你也不用帮着那些美国人去对付罗队长他们吧？”
<br/>　　“塔尼特，我没有选择……”
<br/>　　“够了！”
<br/>　　比洛多还想要解释什么，可是却被塔尼特给打断了，“你不需要再解释什么，你现在的行为说再多的解释都是徒劳的，我不想再听到你的解释！多行不义，你最好不要忘记，是他们在关键时刻救了我！其实说到底，你还是更倾向于跟美国政府合作，因为你根本就看不起他们！”
<br/>　　“塔尼特！”
<br/>　　塔尼特没去管比洛多，直接转身走出了房间。
<br/>　　比洛多看着被塔尼特重重甩上的门，跌坐在办公桌后面的椅子中，塔尼特没有说错，比起跟罗昊他们，比洛多更倾向于跟美国政府合作，毕竟美国才是世界第一，地处南美洲，如果比洛多想要坐稳总统的位置，跟美国人打好关系很重要，比洛多很清楚这一点。
<br/>　　……
<br/>　　安全屋中，战术中队跟布克尔特还在进行着对峙，时间已经过去了三十分钟。
<br/>　　这种对峙在精神和体力上都是一个极大的消耗，尤其是对于佟千雪来说，这种消耗更是已经接近了她的身体极限，尤其是地下室中氧气的消耗也是到了一个临界点。
<br/>　　“哥，佟医生的状态很不好，她恐怕坚持不了不少时间了。”齐凡对罗昊小声提醒道。
<br/>　　齐凡看出的问题，罗昊自然也是看出来了，但现在布克尔特摆明了态度是要跟自己死耗到底，等到外面的支援破门进入。
<br/>　　一个小时不到的时间，Royce需要的东西由直升机送到了安全屋。Royce使用探测仪对脱氧钢板门进行检测后，用记号笔在几个位置画上红叉，然后让人在红叉位置开始打孔，准备装填塑胶炸药。
<br/>　　听到从外面传来钻孔机在脱氧钢板门上钻孔的声音，布克尔特脸上露出了笑容，对罗昊说道：“看来胜负很快就要揭晓了！”
<br/>　　“哥，怎么办？”齐凡这个时候也显得有些焦急起来，对罗昊说道：“如果等到外面那些人破门而入，我们就彻底被动了。”
<br/>　　“没办法只能赌一把了！”
<br/>　　罗昊现在只能希望慕容飞教过佟千雪如果被人控制，面对营救队员时应该要怎么配合。如果现在被控制的人是云熙，罗昊根本就不需要有这方面的顾虑，话不要多，就是干！
<br/>　　因为云熙本身就对龙隐部队那套营救配合暗号非常熟悉，就如当年在“灵魂之门”中，她被巴朗手下的武装毒贩控制，就很好的跟罗昊完成了一次配合。
<br/>　　因为龙隐部队这个群体很容易遭到敌对势力的针对，队员家属往往是被报复的第一选择，所以从龙隐部队当年第一次出现队员家属遭到挟持的情况后，龙隐部队就做了很多相关的预案，其中就有让队员教授在紧急情况时，如何自救，以及如何配合营救队员进行营救。
<br/>　　罗昊尝试着用暗号跟佟千雪进行沟通，被布克尔特所控制的佟千雪偷偷对罗昊竖起一个“oK”的手势。
<br/>　　还好！
<br/>　　罗昊在心中有些庆幸，左手握着强光战术手电的食指，在强光战术手电上轻敲。
<br/>　　一！
<br/>　　二！
<br/>　　佟千雪紧紧盯着罗昊第三次抬起的左手食指，心中不免有些紧张，在家的时候，慕容飞在教佟千雪如果遭到绑架如何配合营救队员进行营救的时候，详细地进行过讲解，可是理论和实战毕竟是两回事。不过佟千雪自己也知道，现在她没有选择，如果她不能配合好罗昊的营救，她很有可能会死，不是死在布克尔特的手中，而是死在罗昊的枪下。
<br/>　　三！
<br/>　　罗昊抬起的左手手指敲下第三下的时候，按亮了手中的强光战术手电，而在强光战术手电亮起的刹那，佟千雪的头朝她的左侧偏开。
<br/>　　雪白刺眼的手电灯光直接打在佟千雪身后布克尔特的脸上。
