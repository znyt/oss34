　　“嗯？”萧逸眼尖，忽然看到前方有两道熟悉的身影。
<br/>　　“呵。”萧逸冷笑一声，快步走去。
<br/>　　......
<br/>　　“再不把钱和淬体丹交出来，信不信我们把你腿给打折了。”
<br/>　　萧杰和萧石正对着一个家族子弟拳打脚踢，勒索着。
<br/>　　“不给，这些淬体丹我要留着给我哥提升实力，我哥半个月后就要参加家族比武了。”
<br/>　　这个家族子弟约莫11岁左右，只有凡境二重的修为，哪里是萧杰萧石两人的对手。此刻已经被打得鼻青脸肿了，却始终不肯交出淬体丹和银两。
<br/>　　“既然你自己找不痛快，那就别怪我们了。”萧杰和萧石对视一眼，狞笑一声，同时催动了武魂力量，他们要动真格了。
<br/>　　萧杰和萧石两人总是欺负家族里实力较弱的子弟，抢夺他们的修炼资源。
<br/>　　偏偏两人还有萧若寒罩着，所以一般就算年纪比他们大，实力比他们强的子弟也没办法找他们报复。
<br/>　　“哎哟。”萧杰正打算用火烧藤将面前的子弟捆绑起来，猛地感到后背一疼，痛呼了一声。
<br/>　　“哪个不知死活的家伙，敢来管本少爷的事....”萧杰愤怒地转头，大骂一声。
<br/>　　然而，当他看到来人后，瞬间哑口无言。
<br/>　　“你说谁不知死活？”萧逸冷冷地看着萧杰。
<br/>　　“萧...萧逸...”萧杰和萧石同时不自觉地打了个冷颤。
<br/>　　“还愣着干嘛，走阿。”萧逸对一旁已经鼻青脸肿的家族子弟说道。
<br/>　　“额。”这个家族子弟愣了愣，意识到萧逸的意思后，立刻大喜，“谢谢萧逸少爷”，说罢，紧紧捂着怀中的淬体丹和银两，撒腿就跑。
<br/>　　萧杰萧石不甘地看了眼离开的家族子弟，却不敢去追。
<br/>　　“看什么看。”萧逸喝道，“老子让你们准备的钱准备好没有？”
<br/>　　“那个...”萧杰支支吾吾。
<br/>　　“我们正筹着钱呢。”萧石皮笑肉不笑地狡辩着，他原本就没打算真地还钱，只不过是敷衍罢了。
<br/>　　“既然你们自己找不痛快，那就别怪我了。”萧逸奉还了一句刚才他们欺负人时说的话。
<br/>　　‘咔咔’两声，萧逸一把将他们两人的肩膀给卸了下来。
<br/>　　以他现在凡境四重的修为，配上豹形，萧杰和萧石连反应过来的机会都没有。
<br/>　　“哎哟。”萧杰萧石痛呼一声，扶着自己脱臼的手臂。
<br/>　　“小废物，你想干什么！”萧杰痛而大呼。
<br/>　　“还敢顶嘴。”萧逸一巴掌扇了过去，劈头盖脸就是一顿揍。
<br/>　　两人被打趴在地上，但还是嘴硬道，“哼，小废物，打，有种你继续打，我就不信在这大庭广众之下，你还敢打死我们不成？”
<br/>　　萧石也威胁道，“小废物，等着吧，今日之仇，若寒表哥会十倍奉还给你。到时候，你会像死狗一样跪下来求我们。”
<br/>　　“找死。”萧逸双眼瞬间变冷，随手从地面捡起了块石头。
<br/>　　“萧若寒会不会找我算账我不知道，但起码，你们是没有那个机会看到了。我倒想看看，这石头砸下去，你们的脑袋会不会开花。”
<br/>　　“小废物，你敢？”萧杰和萧石叫嚣了一声，但其实还是一阵害怕，他们可不觉得自己的脑袋能比石头硬。
<br/>　　“你看我敢是不敢，萧若寒我都敢当着众长老执事的面扇他耳光，更何况你们两个？”
<br/>　　说罢，萧逸一石头朝着他们的脑袋砸下去。
<br/>　　“别...别阿，萧大爷，饶命阿。”萧杰立马求饶。
<br/>　　“萧...萧逸，我保证，十天之内，保证还钱。”萧石哆嗦着求饶。
<br/>　　“哼。”萧逸没有收回石头，但改变了方向，朝他们的手脚砸去。
<br/>　　“哎哟。”萧杰萧石痛呼一声。
<br/>　　这两人都是萧若寒的狗腿子，而且在欺负其他家族子弟时也经常下重手。萧逸自然不会对他们客气，使劲地揍着他们。
<br/>　　“哎哟...哎哟。”萧杰萧石两人的痛呼声此起彼伏。
<br/>　　“我再让你们说一遍，多久还钱。”
<br/>　　“八天，八天。”
<br/>　　萧逸一石头砸下去。
<br/>　　“哎哟，五天..五天。”
<br/>　　又是一石头下去。
<br/>　　“多少天。”
<br/>　　“三天，三天，保证三天。”
<br/>　　“哼，这还差不多。”萧逸扔掉了石头。
<br/>　　“滚吧。”萧逸喝道，“记住了，你们只有三天时间，别让我自己去找你们。否则，三天后，砸你们的就不是石头，而是刀子。”
<br/>　　“知道了知道了。”萧杰萧石如蒙大赦，连跑带滚地离开。
<br/>　　两人都是武者，被石头砸几下可没受多大的伤势，顶多让他们疼一会儿罢了。
<br/>　　“两个欺软怕硬的人渣。”萧逸呸了一口。
<br/>　　.....
<br/>　　另一边，那些原本还在对萧逸指指点点的家族子弟露出了像见了鬼一样的表情。
<br/>　　这还是那个废物萧逸吗？
<br/>　　这还是那个被人欺负了也不敢反抗的窝囊废？
<br/>　　萧逸竟然将萧杰和萧石两个恶棍收拾了？
<br/>　　“看什么看？”萧逸转过头剐了众人一眼，道，“怎么，你们也想尝一尝这石头的滋味？”
<br/>　　呼的一声，家族子弟们连忙逃散。
<br/>　　萧逸转过身，往萧家大门而去。
<br/>　　他还要去买药材和炼药炉。
<br/>　　......
<br/>　　紫云城，在整个北山郡都是小有名气的大城。
<br/>　　紫云城背靠陨星山脉。
<br/>　　而陨星山脉绵延千里，内有大量珍贵矿石。从各地前来的商人络绎不绝，车水马龙。
<br/>　　故紫云城的商业极为发达。
<br/>　　而陨星山脉内又有无数妖兽，可谓妖兽纵横，称霸一方。
<br/>　　故因此大量的武者成为了猎妖师，或为商人们保驾护航，或独自游走于山脉中谋取利益。
<br/>　　......
<br/>　　萧逸走了一段时间，随意找了间规模颇大的药材店，走了进去。
<br/>　　“客人，有什么可以帮到您的吗？”伙计见萧逸衣着华贵，显然是有钱人家的子弟，连忙谄媚着上前招呼。
<br/>　　“无心藤，八叶花，红首乌，六神草，灵参，这些药材可有？”萧逸淡淡地问道。
<br/>　　伙计愣了愣，道，“额，客人，这些都不是普通的药材吧....”
<br/>　　“有还是没有？”萧逸皱眉问道。
<br/>　　“有，当然有。”伙计连忙点头，道，“客人需要多少呢？对这些药材有其他要求吗？”
<br/>　　“嗯？”萧逸思索了一下，道，“前四种药材十年份以上，灵参百年份以上。数量嘛，各来五份吧。”
<br/>　　伙计愣了愣，随即为难道，“客人，前四种药材没问题，但百年份以上的灵参太过稀有了，这...”
<br/>　　“怎么，没有吗？”萧逸问道。
<br/>　　前四种药材是炼制淬体丹的主要药材，灵参则是炼制九转炼体丹的主要药材。
<br/>　　只要备齐了这些主要药材，其他辅助药材根本不是问题。
<br/>　　“有，公子您稍等。”不知为何，伙计忽然变得恭敬起来，连称呼都变了。
<br/>　　萧逸点点头。
<br/>　　....
<br/>　　药材店后堂，伙计对一位老者恭敬地说道，“掌柜，其他四种药材没有大问题，但百年份的灵参是您最近用来炼药的，所以小的来请示一番。”
<br/>　　老者作为这间药材店的掌柜之外，还是一位一品炼药师，在整个紫云城也算小有名气。
<br/>　　“灵参，还是百年份的。”老者思索了一下，对伙计道，“都卖给他吧，在价格上给他个折扣。”
<br/>　　“折扣？”伙计有些疑惑，问道，“掌柜，我们店可从来没有打过折扣阿。”
<br/>　　“你照做就是。”老者呵斥了一声。
<br/>　　“是。”伙计应答一声，退了出去。
<br/>　　.....
<br/>　　“公子。”伙计准备好药材，回到萧逸面前。
<br/>　　“这是您要的药材，各五份，总价五百两，本店给您打了个七折，只需三百五十两即可。”
<br/>　　“打折？”萧逸有些疑惑。
<br/>　　在出门前，萧逸就考虑过自己能买多少药材。自己从杨伏身上获得的钱，大概就是五百两左右。
<br/>　　一颗淬体丹价值为十两，但实则这只是在萧家内部的说法。
<br/>　　淬体丹单单材料就要十两，而且请炼药师炼制的话，还要给一笔不菲的酬劳。
<br/>　　外面卖的淬体丹，一颗起码要五十两。
<br/>　　前四种药材，各五份，大概就是二百两。而灵参一份大概就是六十两，五份三百两。
<br/>　　加起来五百两，伙计给的价格很公道。
<br/>　　只是，这药材店为什么要给自己打折呢？
<br/>　　萧逸点了点头，道，“你这有炼药炉卖吗？”
<br/>　　“有，当然有，公子请随我来。”伙计应答一声，恭敬地带着萧逸走向另一边。
<br/>　　这间药材店规模不小，除了各式药材外，还有炼药炉，以及成品丹药等等。
<br/>　　店铺卖的炼药炉有三种，伙计一一介绍起来。
<br/>　　“公子，这是精钢炼药炉，本城著名的铁匠亲手打造，售价五十两。”
<br/>　　“这是鎏金炼药炉，由我店的掌柜亲手所制，售价二百两。哦，对了，掌柜他可是一位尊贵的一品炼药师呢。”
<br/>　　“这是三旋火炉，本店的镇店之宝，售价八百两。”
<br/>　　“切。”萧逸心里暗暗撇嘴，这伙计的兜售本事还真是厉害。
<br/>　　他清楚得很，第一种精钢炼药炉，根本就是不入流的炼药师才会用的炼药炉。
<br/>　　第二种也一样，不过品质高上不少。
<br/>　　第三种倒是厉害，家族里的一品炼药师都是用这种三旋火炉，就是价格太贵了。
<br/>　　“给我来个鎏金炉吧。”萧逸道。
<br/>　　“好嘞。”伙计点点头，道，“仍旧给公子您打七折，公子还有其他需要的吗？”
<br/>　　“没了。”萧逸付过钱，将药材和鎏金炉拿好，头也不回地离开了药材店。
<br/>　　.....
<br/>　　萧逸刚离开不久，药店的掌柜走了出来，淡淡地看着萧逸离开的方向。
<br/>　　伙计疑惑道，“掌柜，那五份灵参可是您用来冲击二品炼药师所用的，为何....？”
<br/>　　掌柜自语道，“无心藤，八叶花，红首乌，六神草，这可都是炼制淬体丹必备的材料阿。这少年要么是个炼药师，要么就是背后有炼药师在指导着炼药。”
<br/>　　“百年份的灵参，起码是二品以上丹药的材料。那么，很可能这少年背后的起码一位二品炼药师，甚至三品炼药师。”
<br/>　　“记住了，如果这个少年下次还来买药材，尽量和他打好关系，最好能结下交情。”掌柜对伙计吩咐道。
