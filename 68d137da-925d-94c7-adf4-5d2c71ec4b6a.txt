　　第596章
<br/>　　程大雷的声音从屋子里传来，听上去颇有高手风范。隔着窗户的草上飞捂住了眼睛：我就知道，这几个帝国人不是程大雷的对手。
<br/>　　多亏自己没有搀和进这件事情去，不然，自己的情况怕是要比这几个帝国人更加糟糕。
<br/>　　只见钟牧山一伙歪三扭四倒在地上，每个人都狼狈不堪。除了耻辱，他们现在更多的是恐惧。就在刚刚的一瞬间，他们向七人同时向程大雷出手，每个人都拿出了自己的看家本领，结果……
<br/>　　结果程大雷连剑都没有动用，只是拳打脚踢，拎着他们的头发便一个个都丢出了屋子。过程，并不比赶几只猪困难更多。
<br/>　　事到如今，他们才大概明白自己与程大雷的差距，像一口深井，深不见底。
<br/>　　他们很显然没有实力摸出这口井有多深。
<br/>　　草上飞一开始没搀和这件事，但这个时候，他不能再犹豫。机会稍纵即逝，自己必须抓住。
<br/>　　“程当家在哪里，程当家在哪里。”草上飞提着刀，大吼着冲出屋子：“谁想害程当家，弟兄们与我杀呀。”
<br/>　　山寨里五十几号山贼都冲了出来，尤其是草上飞，更是一马当先，率先冲进程大雷所住的屋子。
<br/>　　“程当家，你没事吧，程当家……”
<br/>　　程大雷倒是吓了一跳，理了理袍子道：“我能有什么事。”
<br/>　　“没事就好，可是把我吓坏了。他们竟然想害您老人家，真的是吃了狼心豹子胆。”草上飞咬牙切齿：“程当家放心，他们一个也跑不了。”
<br/>　　程大雷从屋里出来，见小白狼也立在不远处。而院子当中，早就杀成了一团。黑天半夜的，也看不清楚，就只听见闹哄哄的喊杀声。
<br/>　　草上飞手下的弟兄，却也是雷声大雨点小，他们也不是钟牧山七人的对手。可钟牧山等人撞上程大雷这块大铁板，心里面已经怯了。乱杀乱砍，也分辨不出有多少马贼，一心只想着逃命要紧。
<br/>　　最后胡乱砍杀一阵，七人冲出了马匪的包围圈。一刻也不敢停留，宛若没投苍蝇一般乱窜乱逃。
<br/>　　钟牧山逃得气喘吁吁，一口气也不知跑出多远。最后停下脚步，转过头一看，自己与其他人都跑散了，身边只留了一名叫做莫小湖的小师弟。
<br/>　　“师兄，他们，他们去哪儿了？”莫小湖气喘吁吁的问道。
<br/>　　这个问题，也正是钟牧山自己想问的。但莫小湖已经问出口，他这个当师兄的也不能显得一点没有法子没有。可莫小湖问他，他又去该问谁。
<br/>　　转身四顾，周围黑漆漆一片，唯有荒草随风而动，隐隐还有各种不知名野兽的怒吼。身在此地，宛若坠入魔窟一般，说不得什么时候会有某种怪兽，撕破黑暗将自己一口吞了。
<br/>　　回想这次行动，千里迢迢来到这鸟不拉屎的地方，也的确如愿以偿同程大雷碰上了。可却也闹明白一件事，凭自己几人的本事，根本没办法在程大雷面前讨到便宜。
<br/>　　思前想后，心中憋闷不已，如此状况，自己有何面目再回书院。
<br/>　　哒哒哒
<br/>　　耳边忽然响起马蹄声，钟牧山和莫小湖一激灵，同时握紧了剑。
<br/>　　莫非那马匪追到这里来了。
<br/>　　远远有一队人向这边过来，当钟牧山看到对方时，对方也发现了他们两个。
<br/>　　这对人正是齐如海的商队。他受了野原火的委托，采购兵器送到草原上，这几日连轴转的忙碌，总算是将这件事办妥了。连夜过来，也是怕遇上路上的强人土匪。
<br/>　　“什么人。”
<br/>　　齐如海心中一惊，莫非真是怕什么来什么，当真遇到马匪了。
<br/>　　双方晃着了面，各自心里都松了一口气。齐如海脸上堆出笑容，道：“二位也是帝国人吧，这真是他乡遇故知，能遇到二位也是缘分。”
<br/>　　钟牧山和莫小湖心底也是松口气，在这样人生地不熟的地方，能遇到身着帝国服饰的同乡，心里总有几分亲切感。他自然看出齐如海是与戎族做生意的商队，对这些事他倒也不以为意，钟牧山毕竟没有正义感爆棚到眼底不容沙子的地步。
<br/>　　“看二位的样子，是遇上马匪了吧。”齐如海套着近乎：“这些杀千刀的马匪，就该千刀万剐。二位如果不嫌弃，可以与我们一道走，到时候大家一起回去。”
<br/>　　钟牧山两个也是筋疲力尽，况且他现在和李舜时几人失散，在草原上找得回来找不回来还是个问题。若与他们同行，路上也有个帮衬，趁机也能打听李舜时一伙的下落。
<br/>　　“诸位放心，若是路上遇到贼人捣乱，我们也能出一份力。”如钟牧山这等洁身自好的人，可不希望吃人家白食。
<br/>　　“好说，好说。”
<br/>　　双方结伴同行，半路上齐如海就将钟牧山二人盘了个底儿掉。钟牧山毕竟太嫩了，经过这一番坎坷，也没够令他们足够成熟。而齐如海是个生意人，机灵得恨不能拔根眼睫毛就能当哨吹。
<br/>　　一队人行了几日，路上倒也是顺风顺水，没有碰见马匪盗贼。等到这一日，到了野原火的营地。
<br/>　　齐如海亲自带人将兵器交付给野原火，足足能武装三千人的兵刃，每一件都是匠人精心打造，齐如海没有在里面弄虚作假。当然，他也没有弄虚作假，以次充好的胆量。
<br/>　　野原火对齐如海的表现很是满意，经过这第一次合作，双方已建立了信任。他将齐如海请到大帐之中，手下人已开始分发兵器。
<br/>　　得到这批兵器，他在草原上的势力竟登上一个新的台阶，面对自己将要做的事，也就更有自信了。
<br/>　　“我嘱托齐先生与我带来几个读过书的人，这件事办的如何？”野原火问道。
<br/>　　齐如海顿了顿，这件事他办得不怎么样，从帝国带货容易，带人就难了。
<br/>　　“启禀首领，我带了两个人来，只是他们愿不愿意辅佐大王，这个我还没把握。”
<br/>　　齐如海口中说的，自然就是钟牧山和莫小湖二人。如果不是完成这件差事，他怎么会无端端的发好心带上这二人。
<br/>(本章完)
