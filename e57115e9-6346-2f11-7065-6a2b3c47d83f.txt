　　现在林峰的作战计划就是，能够先远程武器轰炸就远程武器轰炸，实在不行再动刀动枪的干。
<br/>　　咻。
<br/>　　一发火箭弹射了过去，直接命中。
<br/>　　顿时那变异鳄鱼怒吼一声，而他不过身上的鳞甲破碎了一点。
<br/>　　“我去，这么硬的防御？”
<br/>　　林峰震惊了，这变异鳄鱼比之前那个变异野猪王的防御还要强大多了。火箭弹都没多大的效果。这简直比坦克上的反应装甲还要恐怖了。
<br/>　　一击未果。
<br/>　　变异鳄鱼已经怒冲而来，鳄鱼这个东西实际上爬起来的速度还是蛮快的。
<br/>　　眨眼间就是冲到了林峰面前，一个大嘴就是朝林峰咬来。
<br/>　　林峰立即是起身闪躲，躲掉了对方一击。
<br/>　　“千年僵尸！还有小妲己都给我出来！”
<br/>　　林峰放出了自己的帮手。
<br/>　　千年僵尸二话不说就是跳上去干变异鳄鱼了。
<br/>　　而小妲己还先买了个萌。
<br/>　　“先别卖萌了，这个时候卖萌可耻，快帮你主人对付这个鳄鱼。”
<br/>　　林峰道。
<br/>　　“知道了主人。”妲己说着也施展了自己的绝招。
<br/>　　妲己和千年僵尸一样都是六阶，而且最主要的是，妲己是远攻的，可以在旁边丢爱心。
<br/>　　这个爱心一旦命中，哪怕是这个变异鳄鱼也会出现精神恍惚的现象。
<br/>　　当然了，林峰是最主要的输出。
<br/>　　直接拔出七杀剑就是一刀怒斩而去。
<br/>　　狂龙出海！
<br/>　　青莲剑歌！
<br/>　　二话不说，林峰就是两刀先送了过去，这个变异鳄鱼，真的是全程武装，从头到尾巴都像是穿了一层反应装甲一样。
<br/>　　唯独他的腹部是个弱点，但是这鳄鱼一个爬行动物，重心很低，要让他露出肚皮，很难做到。
<br/>　　所以林峰只能对着对方眼珠子狂劈。
<br/>　　噗嗤。
<br/>　　两刀下去，直接在变异鳄鱼身上砍出了一个不深不浅的口子。
<br/>　　“这防御，真的无敌了！”
<br/>　　林峰心中一惊，这多亏自己斩灵刀还是个二星的武器，这若是拿着自己之前的那把武士刀，岂不是自己一刀下去，刀碎了，这家伙屁事没有？
<br/>　　变异鳄鱼彻底激怒了，尾巴一甩，就是把千年僵尸给砸飞了。
<br/>　　然后气势汹汹朝林峰咬来。
<br/>　　“我去！”
<br/>　　林峰看到那血盆大嘴心里就胆寒，这嘴巴，这牙齿，被咬一口，估计林峰直接要被一份为半了。
<br/>　　“对了！嘴巴！”
<br/>　　林峰眼睛一亮，二话不说就是两个汽油弹丢到对方嘴巴里。
<br/>　　哗啦啦。
<br/>　　瞬间，变异鳄鱼嘴巴里就燃烧起了熊熊烈火，刹那间林峰都闻到了一股烤肉味。
<br/>　　鳄鱼，虽然看起来恐怖，但实际上他的肉是可以吃的，而且味道很不错，末世之前就有烤鳄鱼肉，而且网上也有喂养的鳄鱼卖，一头好几千块钱。
<br/>　　所以这肉闻起来味道还挺香的。嘴巴里忽然着火，变异鳄鱼疼的是疯狂怒吼。
<br/>　　这就是自家后院了起火，他虽然有一层无敌的装甲，让他固若金汤，但是他身体内部可没有任何装甲，直接烧的他嘴巴里的肉皮开肉绽。
<br/>　　“杀！”
<br/>　　趁你病要你命，林峰是立即开始疯狂战斗。
<br/>　　刷刷刷。
<br/>　　一刀快过一刀，一刀强过一刀，对着这变异鳄鱼是展开了疾风暴雨般的猛烈强大攻势。
<br/>　　这变异鳄鱼也的确够顽强，哪怕是如此依然无比的残暴。
<br/>　　但他残暴归残暴，林峰比他更残暴。
<br/>　　而且最主要这变异鳄鱼，他攻击方式太单调了，一来就是用尾巴砸，二来就是用嘴巴咬，就这么两个。
<br/>　　他嘴巴一张，林峰就是汽油弹招呼对方。
<br/>　　几番下来，这变异鳄鱼整个嘴巴都烂了，整个脑袋都在冒烟，已经快要不行了，随后被林峰抓住机会一刀刺穿了嘴巴，从他脑袋里透出刀尖，死的不能再死。
<br/>　　林峰松了口气，抹了把头上的汗水，对付这个变异鳄鱼，还真的是费了九牛二虎之力，打了足足半个小时。
<br/>　　林峰收起妲己和千年僵尸，然后就把变异鳄鱼整个尸体收入物品栏。
<br/>　　七阶的怪物，绝对不低了，这东西尸体绝对是有科研价值的。
<br/>　　林峰最近就听说了，军部的科研部门急需要一批高等级的怪物尸体。
<br/>　　一来是研究变异原因，二来是研究新武器制造新材料。
<br/>　　毕竟世界都在变，人类的武器装备也必须要变一变了。
<br/>　　比如说这个鳄鱼，他的防御绝对可以无视子弹了。
<br/>　　把整个鳄鱼尸体放入物品栏。
<br/>　　林峰看了下自己的武器装备。
<br/>　　汽油弹就剩下最后一发。
<br/>　　火箭弹也就那么一发。
<br/>　　“需要补充弹药了。”
<br/>　　林峰继续下一个猎杀目标。
<br/>　　一头六阶的丧尸。
<br/>　　林峰赶到这里时。
<br/>　　这里居然还有幸存者。
<br/>　　一个穿着特警制服的六阶丧尸拿着个大锤子在那里砸门。
<br/>　　这是一个看起来像是仓库的小屋子里。
<br/>　　“啊，怎么办，我们要死了。”
<br/>　　“我不想死啊，这个时候若是有人救我，我愿意嫁给他。”
<br/>　　“我也是啊，我给他当小三都没意见。”
<br/>　　仓库内躲着四个女大学生，长得都非常好看。
<br/>　　此时此刻听到外面的丧尸声音吓得一个个都花容失色，抱在一起瑟瑟发抖。
<br/>　　林峰直接冲了出去，那六阶丧尸察觉到林峰顿时怒吼。
<br/>　　“叫你大爷。”
<br/>　　刷刷。
<br/>　　抬手林峰就是三枚麻痹飞镖。
<br/>　　瞬间这六阶丧尸身体一僵出现麻痹效果。
<br/>　　狂龙出海！
<br/>　　刷！
<br/>　　一刀切过，这六阶丧尸的脑袋直接从脖子上滚了下来，鲜血狂喷，无头尸体才是直挺挺倒下。
<br/>　　林峰敲了敲仓库门。
<br/>　　“丧尸我干掉了，你们可以出来了，沿着东边走大概三里地，就可以看到曙光基地里，那个地方还算安全。”
<br/>　　林峰挖走六阶丧尸的晶石然后就离去了。
<br/>　　随后仓库门打开。
<br/>　　四个女人看到那死了的丧尸，再看到林峰远去的背影。
<br/>　　“白马王子！他是我的白马王子！”
<br/>　　“好帅！好潇洒！”
<br/>　　“我发现我爱上他了！”
<br/>　　四个女大学生都眼中露出了崇拜的神色。
<br/>　　林峰接取的几个任务之中除了第一个七阶的变异鳄鱼让林峰颇为废了一些手段之后，其余的任务都是一刀斩的货色。
<br/>　　什么六阶丧尸五阶的怪物，尤其是没有盔甲鳞甲保护的丧尸，更好对付。
<br/>　　一个麻痹飞镖射过去，让对方产生麻痹，然后一刀秒杀。
