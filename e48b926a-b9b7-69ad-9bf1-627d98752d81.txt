　　对于柳长老亲手击毙白山的表现，慕清霄颇为满意。
<br/>　　其实一只蝼蚁的死活，对他来说根本无关紧要，可这只蝼蚁既然伤害了自己的女人，那他的下场注定只有死。
<br/>　　小医仙也完全没有在意，此时她的心思全都放在了慕清霄的身上。
<br/>　　回到当初两人居住的小阁楼中，慕清霄将窗户打开，随手一卷，其中的一些灰尘尽数被清理了干净。
<br/>　　随着时间的推移，夜幕降临。
<br/>　　坐在阁楼阳台上，搂着怀中的幽香玉/体，仰望着天上的星河，慕清霄眼中闪过一抹精芒。
<br/>　　“仙儿，按照我说的方法试试。”
<br/>　　说着，慕清霄就将《玄阴素女经》的心法路线传给了小医仙。
<br/>　　正所谓大道三千殊途同归，斗气与灵气并不冲突，只是能量的转换，或许厄难毒体并非如此难以解决。
<br/>　　察觉到脑海中凭空出现的神秘路线，小医仙虽有些困惑，却依旧是按照慕清霄说的照做，掌心向天，运转心法。
<br/>　　心法运转的瞬间，小医仙体内残存的毒气就涌动起来。
<br/>　　或许是因为慕清霄就在身边，小医仙并未害怕，就算毒气在经脉中乱窜，她也是纹丝不动。
<br/>　　良久，体内毒气随着心法的运转，在经脉中开始性质的转变，最后汇聚在了斗之气旋，也就是小腹下的丹田位置。
<br/>　　“清霄哥哥，毒气真的消失了，并且化作了其他能量。”
<br/>　　小医仙惊喜无比，她用心法转化的能量与斗气有着几分类似，但这股能量更凝实更浓郁，最重要的是连毒气都能转化。
<br/>　　这种心法闻所未闻，恐怕传说中的天阶斗技也不过如此吧？
<br/>　　如果慕清霄知道小医仙心中所想，估计会无语至极。
<br/>　　别说天阶斗阶，就算圣阶斗技，连给《玄阴素女经》提鞋的资格都没有。
<br/>　　差距犹如云泥，根本就没有可比性。
<br/>　　慕清霄抓起她纤细的胳膊，查探了下她体内的情况，仅仅将残存的毒气炼化就已经迈入筑基境界。
<br/>　　果然如他所预料的一样，《玄阴素女经》能够将毒气转化成天地间至寒灵气，让天火尊者去找魔核与菩提化体液倒是多此一举了。
<br/>　　不过，既然已经离开，慕清霄也不可能叫他回来。
<br/>　　“仙儿，我将你体内的封印开启一个缝隙，让身体最深处的毒气溢出，照刚才的方式继续转换。”
<br/>　　“嗯，好。”
<br/>　　乖巧的点了点小脑袋，小医仙静静的盘坐在星空下，任由慕清霄开启封印。
<br/>　　慕清霄食指与中指轻轻在她小腹位置一点，入手的是一股柔软的感觉，稳住心神，轻轻揭开封印。
<br/>　　小医仙颈部一红，某种闪过一抹羞涩。
<br/>　　旋即，绯红涌上绝美的脸颊，将她柔美至极的容颜染成一个红彤彤的苹果。
<br/>　　待封印揭开，毒气自封阴阳八卦印中溢出，小医仙急忙静下心神，按照心法路线上的修炼起来。
<br/>　　随着毒气不断的转换，她的气息就以一种极其恐怖的速度攀升起来。
<br/>　　半个小时左右，她的气息就从筑基一重初期攀升到了筑基九重巅峰。
<br/>　　对此，慕清霄并不担心，前期她的根基很牢固，而且她的境界本就是斗灵，相当于元婴修士。
<br/>　　现在，只是将体内全部的毒气都转化成至寒灵气，境界如坐火箭一样攀升实属正常。
<br/>　　转眼半个时辰过去，小医仙身躯一震，一股极强的气息自她体内席卷而出。
<br/>　　一颗金丹吞入腹，我命由我不由天。
<br/>　　金丹一重！
<br/>　　金丹二重！
<br/>　　金丹三重……
<br/>　　时间过的很快，随着时间的推移，小医仙的气息依旧渐渐的攀升着，只是速度没有先前恐怖。
<br/>　　“嘶嘶……”
<br/>　　就在这时，几道细微的嘶鸣声将慕清霄的思绪拉回现实。
<br/>　　只见他的袖口清微抖动了几下，随后一条极其绚丽的七彩小蛇钻了出来，金色蛇瞳望着小医仙的模样，闪过一抹震惊。
<br/>　　绚丽光彩一亮，七彩吞天蟒就化作一道红裙身影。

<br/>　　“你是如何办到的？”
<br/>　　望着小医仙此刻的状态，彩鳞美眸中闪烁着好奇之色。
<br/>　　慕清霄伸手轻轻将她揽入怀中，脸庞埋在她的颈部位置，一抹奇异的幽香就钻入他的鼻孔中。

<br/>　　彩鳞有些不敢置信的问道，五年从斗皇攀升至斗圣，简直是天方夜谭。
<br/>　　慕清霄温柔一笑，道：“事情非常的复杂，日后在说，除了实力，难道你心里就不想为夫吗？”
<br/>　　彩鳞美眸眨了眨，仔细打量了他一眼，柔情似水，道：“我们没有举办婚礼。”

<br/>　　说罢，慕清霄就将她轻轻抱起，迈步向阁楼中走去，不多时，卧室中就响起一阵阵令人面红耳赤的婉转低吟。
