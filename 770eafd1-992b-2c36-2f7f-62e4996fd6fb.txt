　　第57章
<br/>　　来也匆匆，去也匆匆，这就是闪电战的真谛。
<br/>　　全程不过一炷香时间，程大雷已经破门，放火，抢人，造成一片大乱后，扬长而去。
<br/>　　当卫兵赶到时，程大雷几人已经快马从小路离开落叶城。
<br/>　　快马离城二十里，见后面没有追兵赶来，程大雷才让马速慢下来，再全速狂奔下去，马会脱力而死。
<br/>　　柳芷和程大雷同乘一马，小小身躯依在程大雷怀里，随着马的前行而起伏。
<br/>　　“欧巴……”
<br/>　　“不瞒姑娘，某姓程，正是蛤蟆寨当家，之前是我骗了姑娘。”
<br/>　　“这……已看出来了。”
<br/>　　“姑娘果然冰雪聪明。”
<br/>　　“今日多谢大当家出手救了我，否则后果不堪设想，这份情一定要承大当家。”
<br/>　　“哈，柳姑娘严重，在下最是重义守诺，言出必践，江湖上有口皆碑的。”
<br/>　　“这……奴也看得出来。”
<br/>　　二人你一言我一语，话赶话就像针线，密密缝着。缝什么，嫁衣么！孟子云听在耳里，痛在心头。
<br/>　　本来是自己先去抱住小姐，可程大雷忒也无耻竟然将自己踹开，令小姐误以为是他救了小姐。
<br/>　　唉，小姐今年才十六岁，那个少女不怀春，既然误以为是程大雷救了她，说不得便春心萌动，起了相思。
<br/>　　可怜自己从小看着小姐长大，见小姐一日日出落得愈发标致，义父生前还提过将小姐许配给自己。
<br/>　　自己怎可看着小姐被这歹人蒙骗，一日一日越陷越深，必须让小姐及早悬崖勒马。
<br/>　　催马上前，接近程大雷的瘦马。
<br/>　　“小姐，受了惊吓，不如我们下马歇息片刻再走。”
<br/>　　“云哥哥，不必担心我，咱们还是听大当家的，早些回去蛤蟆寨。”
<br/>　　“这个……男女授受不亲，程当家一个外人，小姐要顾忌自己清誉啊。”
<br/>　　“程当家才不是外人呢，如果云哥哥觉得我清誉不好，就不要跟着我了。”
<br/>　　说着柳芷已板起脸，孟子云一怔，不知自己那句话说得不对，惹了柳芷。什么叫『程当家不是外人』，你们才见几面，他不是外人，我这青梅竹马，从小一起长大得倒成了外人！
<br/>　　远远地，就听得马上二人的谈话从夜幕里传来。
<br/>　　“大当家会不会嫌弃我曾沦落风尘……”
<br/>　　“哪里话，我一占山的强盗，姑娘不要看不起我才好。”
<br/>　　“我怎么会看不起程当家。”
<br/>　　“哈，小姐不必见外，对我直呼其名就可以，或者叫我雷雷，亲近的人都这么叫我。”
<br/>　　“雷雷，咱们还有多久到山寨？”秦蛮。
<br/>　　“滚。”
<br/>　　……
<br/>　　在当天夜里几人便回到蛤蟆寨，徐神机带着众人也刚回来不久，正给大家安排食宿。黄三元还要将他们登记在册，不过今夜天色以晚，只能粗略点个人数，明天再登记造册。
<br/>　　程大雷回到房间后，开始查看系统界面。他大闹黄家，收割一波恐惧值，如今已到一万之数。
<br/>　　在系统界面上，出现一座点将台的图像，可图像却是模糊的，看不清楚。
<br/>　　当程大雷将注意力集中在上面时，收到一道系统提示。
<br/>　　嘟，山寨人口不足100，无法获得点将台建设图纸。
<br/>　　程大雷又查看蛤蟆寨信息。
<br/>　　山寨：蛤蟆寨（初级山寨）
<br/>　　拥有者：程大雷
<br/>　　人口：99
<br/>　　技能：齐心、治愈
<br/>　　看到这里，程大雷道了一声呜呼哀哉。
<br/>　　就像前生玩的经营游戏一样，初级建筑所需条件简单，而高级建筑所需条件则困难些。点将台肯定是高级建筑，除了恐惧值要达到10000外，山寨人口还需突破100 。
<br/>　　山寨原先有六个人，又把柳家人拉到山寨，偏偏柳家人一共九十三个，加在一起就是凑不够一百之数。
<br/>　　差了最后一个人。
<br/>　　程大雷感觉自己有些悲催，心里也不知这最后一人该往何处寻，翻个身蒙着被子睡着了。
<br/>　　次日天明，程大雷早早就起床，他让徐神机敲锣，把柳家人聚到校场上。一是给他们登记名册，二也是看看他们情况怎样，有伤治伤，有病看病。最后，程大雷也要给大家讲讲话，比如『大家从四面八方为了一个共同目标走到一起』『虽非骨肉亲，落地为兄弟』之类的话，聚拢聚拢人心。
<br/>　　山寨人口多了，不能再和以前一样随便，领导艺术自己也要讲一下嘛。
<br/>　　可事情和程大雷想得不一样，铜锣敲得震天响，可柳家人却迟迟未来，直到日上三竿，柳家人才稀稀拉拉来到校场。
<br/>　　“我柳家世代忠良，怎可自贱沦落成山贼，愧对列祖列宗，愧对列祖列宗呐！”一白发老者捶胸顿足。
<br/>　　“落草是不可能落草的，这辈子都不可能落草的……”
<br/>　　“我们暂且在此地休整几日，等将妇孺安顿妥当，我们弟兄几个再投行伍，打下功名依旧光宗耀祖！”
<br/>　　“这里可是贼巢呐，贼巢！在这里待上片刻，此上一口贼饭，就沾染了贼气，一辈子也洗不清。”白发老者继续捶胸顿足。
<br/>　　“兀那小子，这几天你放心，我们可以保全你贼巢安全，但你想我们随你落草为寇，却是想也不要想！”
<br/>　　“饿了，哪里吃饭？”一个胖嘟嘟的小孩道。
<br/>　　徐神机、黄三元、秦蛮、林少羽、徐灵儿表情都阴沉下来，然后他们将目光都看向了程大雷。
<br/>　　大家都发现了问题所在。
<br/>　　蛤蟆寨原先只有六人，这次拉了柳家九十几人入伍，那么，六比九十三，这蛤蟆寨究竟是姓程还是姓柳？
<br/>　　当然，他们是没有占蛤蟆寨的意思，这不是他们有多善良，而是根本看不起蛤蟆寨，连蛤蟆寨都看不起，更何况是蛤蟆寨的程大雷。
<br/>　　其实如何解决这个问题，程大雷也不是没有办法。毕竟五千年文化熏着，厚黑学多少也懂一些。
<br/>　　程大雷应该陪着小心，挂着笑脸，先想办法安抚下他们的情绪，再在以后的日子中，拉拢一匹，孤立一匹，打击一匹，随着时间的推移慢慢建立他们对山寨的归属感。
<br/>　　毕竟程大雷需要这些人，他要完成任务，而想要山寨发展下去，总不能靠六个人撑着，所以眼前这几十号人其实很重要。
<br/>　　可程大雷今年才十八岁，他也青春年少，他也血气方刚。
<br/>　　你说我能惯着他们吗？
<br/>　　冥冥中似乎有个声音回答：能。
<br/>　　但程大雷这次的答案是：
<br/>　　不能。
<br/>
