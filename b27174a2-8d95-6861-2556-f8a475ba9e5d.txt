　　一众强者离去，只余萧逸独自坚守。
<br/>　　一头头洞玄境妖兽、先天境妖兽，蜂拥而来，凶残至极。
<br/>　　其下场，却是悉数死于萧逸有力的拳头之下。
<br/>　　不过短短时间，死在他手下的妖兽，已逾千之数。
<br/>　　紫炎屏障内，两头烈焰狂狮，疯狂地调动妖力抵消着紫炎。
<br/>　　它们作为狮类妖兽，天生有着一种王者般的霸气和高傲。
<br/>　　被一个明明实力不如它们的人类困着，对于它们来说，似是一种耻辱。
<br/>　　它们想不明白，一个人类施展的火焰，为何会如此厉害。
<br/>　　竟比它们这种天生控火强悍的狮类妖兽所操控的火焰更为强大。
<br/>　　甚至于，它们明显从这些紫炎中，闻到一股让它们惊颤的气息。
<br/>　　那是下位妖兽，遇到上位妖兽时所产生的惊惧感。
<br/>　　在它们眼中，漫天紫炎内，仿佛隐藏着一头比它们更为凶残，足以让它们匍匐跪下的狮类妖兽。
<br/>　　紫晶灵炎，乃是紫炎狮王的本命之火，自带紫炎狮王的特殊气息。
<br/>　　一般武者，根本感知不出这股气息；但妖兽，却凭着直觉，有着模糊的感觉。
<br/>　　当然，无论什么感觉，妖兽就是妖兽，凶残无比。
<br/>　　只要破掉紫炎屏障，它们会毫不犹豫地将萧逸撕碎，攻入绿光城，大开杀戒。
<br/>　　只是，它们暂时不会有这个机会。
<br/>　　灵气漩涡不灭，真气不消，紫炎便不会熄灭。
<br/>　　若是此刻叶铭等人没走，一定会惊讶地发现，那三百三十三丈的真气漩涡，在两头烈焰狂狮的疯狂消耗下，竟几乎没有缩减多少。
<br/>　　正常来说，像之前叶铭施展木龙降世，一次性消耗二百多丈真气，也仅仅是让得一头烈焰狂狮重伤。
<br/>　　那么，证明，二百多丈真气，绝对无法在烈焰狂狮的消耗下坚持多久。
<br/>　　可是，现在，三百三十三丈真气漩涡，却像极了没有被消耗过一样。
<br/>　　因为，叶铭的气泉，乃是长宽高，各三百丈。
<br/>　　一次性调动的二百多丈真气，几乎将气泉的真气都耗光了。
<br/>　　而萧逸，乃是气泉长宽高，各达三千三百三十三丈。
<br/>　　只不过，他如今才洞玄一重修为，气泉只填满了一成。
<br/>　　所以，长宽是三千三百三十三丈，高，则只有三百三十三丈。
<br/>　　这不是加法，而是乘法，是一个体积。
<br/>　　气泉中的真气量，乃是百倍于半空中的真气漩涡。
<br/>　　真气漩涡，一直与体内气泉有着联系。
<br/>　　无论消耗多少，萧逸体内气泉会马上补上。
<br/>　　这就导致了看上去，几乎没有消耗过一样。
<br/>　　也是因此，萧逸才有把握，可以困住这两头烈焰狂狮半天时间。
<br/>　　其实，如此庞大数量的真气，如果有一个集中点能将它们完全倾斜而出的话。
<br/>　　秒杀这两头烈焰狂狮轻而易举。
<br/>　　所谓集中点，便是武技。
<br/>　　确切来说，是地阶高级以上的武技。
<br/>　　可惜，萧逸没有。
<br/>　　他如今手头上最强的武技，便是百刃合一后的噬火百刃了。
<br/>　　但也不过是相当于地阶低级罢了。
<br/>　　地阶武技，每一个小等级间的差距，非常大。
<br/>　　这时，便看出武技的重要性了。
<br/>　　也能看出，武技，为何会作为全世界最珍贵的东西之一。
<br/>　　既然没有厉害的高级武技，萧逸便只能单纯地困住它们。
<br/>　　半天后，待得体内真气完全耗尽，他就会离去，到第二道防线的城市。
<br/>　　他并不担心体内真气耗尽会有危险。
<br/>　　因为，他还有着保命的底牌，易老给的剑气。
<br/>　　他甚至有把握，这道剑气之强，可以秒杀这两头烈焰狂狮。
<br/>　　只不过，这是自己现在保命能力最强的东西。
<br/>　　如无必要，他绝不会用。
<br/>　　拖住这些妖兽半天，已经足够第一道防线的所有城内人民撤离。
<br/>　　……
<br/>　　绿光城内，数百万居民，如热锅上的蚂蚁，慌乱地撤离着。
<br/>　　十八城城主，城卫兵，尽力疏散着。
<br/>　　“喂，那边的家伙，快走。”
<br/>　　那边，一个小家族的人，正忙乱地收拾着家族财物，打算装车运走。
<br/>　　“吗的，还在想这些身外之物，命都快没了。”
<br/>　　“别管他们了。”柳苍崖喝道，“他们自己找死，是他们的事。”
<br/>　　“给我们的时间，只有半天。”
<br/>　　“如果只是轻装撤离，城内全部居民都绝对可在半天内安全离开。”
<br/>　　“若他们还磨磨蹭蹭，舍不得财物，死了也怨不得人。”
<br/>　　“紫炎，用他的命为我们争取时间，那些家伙不珍惜，死了便死了。”
<br/>　　一众强者愤愤地骂了几句，没有再理那些还在收拾家财的人。
<br/>　　而是将心思放到疏散一心逃命的普通居民上。
<br/>　　……
<br/>　　半天时间，转瞬便过。
<br/>　　十八城城主的办事效率还是相当高的。
<br/>　　绿光城，以及周边另外5城的居民，超过九成以上都撤离了。
<br/>　　六座大城，空荡荡的，宛若死城。
<br/>　　弥漫在空气中的肃杀气息，还有远处飘来的浓郁血腥味。
<br/>　　无不表示着，情况尚未乐观，没有人可以掉以轻心。
<br/>　　要不了多久，妖兽就会攻入绿光城。
<br/>　　所有人，必须快速赶到第二防线的大城中。
<br/>　　……
<br/>　　“妖兽攻入绿光城，还有一段时间。”
<br/>　　柳苍崖等人，御空飞行在半空中，边赶路，边商议着后续的事宜。
<br/>　　“城内，已设置好各种障碍，还可以继续阻挡一阵时间。”
<br/>　　“足够所有人撤离进第二道防线的大城了。”
<br/>　　“现在看来，一切事情，可保无虞了。”宁浩等人说道。
<br/>　　“可惜，易霄大师他…”
<br/>　　“紫炎会回来的，我相信他。”柳苍崖等人坚定地说着，眼中尽是希冀。
<br/>　　“我也相信易兄，一定会逢凶化吉。”叶铭沉声道。
<br/>　　恰在此时，远方高空中，十数道人影，散发着恐怖气息，疾速飞来。
<br/>　　“好恐怖的气息，好强的气势。”一众强者顿时一惊。
<br/>　　嗖嗖嗖…
<br/>　　十数道人影，忽然停下了飞行。
<br/>　　其中，为首的是一个一脸冷酷的中年人，腰间别着一把血色刀刃。
<br/>　　“你们，可是东荒十八城的城主？”
<br/>　　中年人冷声问道。
<br/>　　“额，是的。”柳苍崖等人老实回答道。
<br/>　　“绿光城情况如何？”中年人言简意赅地问道。
<br/>　　柳苍崖等人脸色顿时一白，回答道，“绿光城已然失守。”
<br/>　　“猎妖殿执事，易霄执事，正独自守城，拖延着无数妖兽。”
<br/>　　“那里…”
<br/>　　柳苍崖还想说些什么，中年人以及十数个手下，已经身影一闪，消失在原地。
<br/>　　“刚才…那…那把血刀…绝对错不了，绿光城有救了。”柳苍崖先是一惊，而后一喜。
<br/>　　……
<br/>　　绿光城，城外战场。
<br/>　　一道身影，独自一人，鏖战无数妖兽。
<br/>　　人影自然是萧逸。
<br/>　　半天时间，他记不得自己挥动了多少拳，杀了多少妖兽。
<br/>　　只知，拳头上、衣衫上、脸庞上，尽是兽血。
<br/>　　他，几乎是在麻木地挥着拳头。
<br/>　　遍地的妖兽尸体，堆积得山般高。
<br/>　　远远看去，他仿佛成了一架无休止的杀戮机器。
<br/>　　鲜血、冰冷、坚毅，各种复杂的意味，几乎都集中在他身上。
<br/>　　半空中，那道三百三十三丈的真气漩涡，已经渐渐开始缩减。
<br/>　　不过数分钟，已经只剩下十来丈大小。
<br/>　　他体内气泉的真气，快要耗光了；自然无法再补充真气漩涡。
<br/>　　“要离开了。”萧逸咬了咬牙。
<br/>　　看着紫炎屏障内，两头散发着凶光的妖兽，他心里微微一惊。
<br/>　　他毫不意外，紫炎一散，这两头凶兽会疯了般追杀他。
<br/>　　他必须趁着真气漩涡还有数丈大小，紫炎还存在的时间，立刻逃离。
<br/>　　然而，正当他准备快速逃离时。
<br/>　　绿光城内，十数道人影御空而来。
<br/>　　为首的，是一个中年人，腰间别着一把血色刀刃。
<br/>　　中年人的速度极快，萧逸还未反应过来，他已经瞬间来到两头妖兽上空。
<br/>　　“压。”中年人冷喝一声。
<br/>　　澎湃无比的天地灵气，瞬间将两头烈焰狂狮禁锢。
<br/>　　“破玄境武者。”萧逸眉宇间一喜。
<br/>　　当看到中年人腰间的血刀时，顿时脸色一变，“血刀卫，北山郡王麾下，第一精锐。”
<br/>(本章完)
