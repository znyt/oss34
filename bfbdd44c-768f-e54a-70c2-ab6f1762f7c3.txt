　　“看来我们是要去这个‘代码小镇’逛逛了。”
<br/>　　罗昊看着投影布上的照片，说道：“我现在倒是对这个神秘的‘代码小镇’充满了好奇心。”
<br/>　　“‘代码小镇’有‘代码小镇’自己的规矩，所有的交通工具只能停放在小镇外，另外除了交易物品以外，其他物品一律不得带入其中，尤其是武器。”子午对着罗昊提醒道：“要不然就会遭到驱逐，被彻底剥夺进入‘代码小镇’的权利。在‘代码小镇’中不得行使暴力，但是出了‘代码小镇’，你就是扛着加特林对着别人扫射也不会有人来管你，而且每个人在进入‘代码小镇’的时候都会拿到一张卡片，这是一张在‘代码小镇’的通行证，凭借这张卡片，可以出入小镇所有的交易场所，但是每个人在小镇最多只能待七十二个小时，一旦超过时间，就会被强行驱逐。所以，如果在交易过程中得罪了不该得罪的，想要在小镇上躲着不出去，这也是行不通的。”
<br/>　　“难道就没有人尝试过武力闯入‘代码小镇’？”慕容飞看着子午问道。
<br/>　　“‘代码小镇’既然能存在到现在，被人当成源代码交易最公正、公平的地方，自然是有着它自己的规矩。”
<br/>　　子午微微一笑，对慕容飞说道：“你说得情况自然也是发生过的，但是凡是破坏了小镇规矩的人，全都死了。小镇上有一支传承自教廷的刺客组织，没有人知道他们一共有多少人，也没有人知道他们藏在什么地方，他们只负责清理那些破坏小镇规矩的人，而且他们杀人极少使用热兵器，基本都是弩箭、袖剑这些冷兵器为主。”
<br/>　　“刺客信条？”
<br/>　　慕容飞喜欢玩游戏，所以在听到子午的介绍后，立刻就在脑海中脑补出了那些维持“代码小镇”秩序的刺客形象。
<br/>　　“你可以这么理解他们。”
<br/>　　出乎慕容飞意料的是，子午竟然点头认同了自己的说法。
<br/>　　“我们什么时候出发？”李然对子午问道。
<br/>　　“我给你们预订了今天晚上的机票。”云熙说道：“你们坐飞机前往‘代码小镇’所在国家的首都，在那里你们可以拿到武器和补给。随后你们从那里出发，坐船前往‘代码小镇’，船靠岸后，你们会得到交通工具前往小镇。老师说，四天后就是‘代码小镇’一年一度的‘源代码’拍卖会，这是‘代码小镇’的年度盛会，是‘代码小镇’规模最大的交易会，世界知名软件公司的代表、武器公司的代表、自由黑客、地下军火商都会参加‘源代码拍卖会’，如果运气好能赶上这次‘源代码拍卖会’，我们或许能从那里得到一些有价值的东西。毕竟也是有人愿意用情报，来换取他们所需要的源代码的。”
<br/>　　当天晚上，罗昊、慕容飞、李毅峰、李然、周柏还有子午一共六个人，搭乘一架国航的航班飞往东欧。
<br/>　　飞机在进入巡航阶段后，子午就打开了自己所携带的笔记本电脑，在键盘上快速的敲打，编写着代码。
<br/>　　罗昊探头看了眼子午电脑屏幕上一组组自己根本看不懂的代码，不明白子午为什么要这么做。
<br/>　　“能用钱解决的事情就不叫事情。”子午看出了罗昊的不解，停下手上的工作，解释道：“在‘代码小镇’最不值钱的就是钱，有很多东西不是靠钱能买到的，而是交换！打个简单的比方，任何武器公司的设计源代码都是机密中的机密，是不可能对外出售的，那么如果有人想得到一家武器公司的源代码要怎么办呢？”
<br/>　　“交换！”罗昊立刻就明白了子午的意思，说道：“用相同价值的武器设计源代码进行交换，以物易物！”
<br/>　　“没错！”子午点头说道：“记住，在‘代码小镇’最不值钱的东西就是钱，源代码才是硬通货！”
<br/>　　因为是直达航班，经过9个多小时的飞行后，飞机在目的地机场降落。
<br/>　　罗昊打开之前在飞机上一直关闭的手机，手机上弹出一条未读短信，里面是一组开启机场储物柜的密码。利用密码打开了对应储物柜后，罗昊在储物柜中拿到了一把汽车的遥控钥匙。
<br/>　　拿着车钥匙，乘电梯进入停车场，站在停车场中，按下汽车开锁键，通过闪烁的车灯，一行人找到了那辆七人座的MPV。
<br/>　　在MPV中，放着四个大型旅行袋。
<br/>　　打开旅行袋，其中三个旅行袋中装着可以在拆开后，拼接成战术背心的特种战术背囊，剩下一个旅行袋中摆放着一捆捆扎好的欧元，全都是五百元面额的。
<br/>　　就这么一袋子钱，足够在京城三环以内买下一套高档公寓了。
<br/>　　慕容飞拿起一捆纸币，翻了翻，诧异地说道：“全都是真钱！？”
<br/>　　“不是真钱，你以为是首尾是真钱，中间全都是白纸吗？”子午轻笑一声，说道：“如果你这样做的话，根本进不去‘代码小镇’。”
<br/>　　“子午教官，这么多钱我们能在‘代码小镇’买到什么？”周柏对子午问道。
<br/>　　“两套‘永恒之蓝’级别的网络病毒源代码。”子午回答道。
<br/>　　“就这样？”
<br/>　　周柏显然是对子午的回答感到意外。
<br/>　　“不然呢？”子午看了眼周柏，说道：“你还想凭着这一袋子钱去把F22的设计源代码买回来不成？我早说过，在‘代码小镇’最不值钱的东西就是钱。”
<br/>　　“那也就是说，我们这次去‘代码小镇’就是见见世面的。”
<br/>　　罗昊拉开一个特种战术背囊的拉链，从里面拿出一把MPX冲锋枪，拉开枪栓，空扣扳机，说道：“如果想要得到我们需要的东西，还是要靠这个东西。”
<br/>　　“强抢？”慕容飞皱了皱眉，故作不屑地说道：“这可不符合我们的身份。”
<br/>　　“作为地下军火商，难道这不是最简单明了的方式吗？”罗昊把MPX冲锋枪塞进特种战术背囊中，眨巴着眼睛对慕容飞问道。
