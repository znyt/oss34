　　第657章
<br/>　　城主府的前庭后院是截然不同的两种存在。前面大庭人来人往，话里话外聊一些帝国大势，总是有几分杀伐之气。
<br/>　　但后宅截然相反，在苏樱的管理下一切井然有序，有四时不谢之花，八节长青之草。让这塞外凉州有了些江南水乡的温软气质。
<br/>　　苏樱平日里也无事可做，她虽然有些商业才能，又有明察秋毫的属性，可平素也不做事。于是便整理一座院子，委实说是有些大材小用。
<br/>　　程大雷最近与苏樱见面的机会越来越少了，一来是的确有事忙碌，二来，是程大雷不太愿意说出口的一个愿意：他不是很想见到苏樱。
<br/>　　进入正厅，便有柳芷接过大氅，将佩剑与帽钗接了。程大雷长舒了个懒腰，刚刚坐下，便有丫鬟端了茶过来。之后，再乖巧的退到角落去。
<br/>　　苏樱这时候过来，身披素采锦衣裳，头戴玉钗，脚踏花鞋，过来时向程大雷款款行了一礼。
<br/>　　程大雷隐隐觉得有些头疼，无可奈何的挺直腰板，也拱手还礼。
<br/>　　今天的苏樱已经不是曾经的苏樱，这倒也不算有错。程大雷毕竟也不是当初青牛山那个小山贼。如今他统治着两座城，十几万人，三五万兵马。养移体，居移气，苏樱讲究一些仪容仪表也是没有问题的事。
<br/>　　但问题是，程大雷并不是一个讲究仪容仪表的人。
<br/>　　即便与徐神机等人相处，程大雷也是很随便的。但面对苏樱也必须正经起来，毕竟不是苏樱做错了什么，程大雷也不好说什么，于是也只好一板一眼的与她假客气。
<br/>　　这对程大雷来说，实在不是什么愉快的事情。他和苏樱见面的机会越来越少，并不是没有这方面原因。
<br/>　　其实，苏樱也有些困扰，虽然人心隔肚皮，但一个人心中什么情绪她还是能感受到的。为什么自己已经做对了所有事情，程大雷还是不开心。
<br/>　　程大雷与苏樱隔着茶桌而坐，时间一点点流逝，只有一缕檀香缓缓燃烧着。
<br/>　　“那个……近来没什么事吧？”
<br/>　　“启禀老爷，诸事无忧，你在外面安心做事，不用挂念。”
<br/>　　程大雷的头越来越疼，挥挥手让丫鬟下去，等屋里只有两个人时才开口道：“那个……你其实对我直呼其名就好，这样叫太生分了。”
<br/>　　苏樱微微低头，道：“本该如此称呼，方显得敬重，让外人听见也不好。”
<br/>　　程大雷长长吐出一口气，振作一下精神：“杨小妹最近怎样？”
<br/>　　杨国忠加入蛤蟆城后，便开始处理城中的内政，而杨小妹则住在城主府，平时与苏樱一起，也算是为苏樱解解闷。
<br/>　　提起杨小妹，苏樱的话才渐渐多起来。听得出来，她对杨小妹很是喜欢。杨小妹本身模样可人，但说实在的，帝国之大，也并不缺美女。苏樱可心杨小妹，倒不是单单因为这一点。杨小妹看上去不是很聪明，没有什么威胁性，但平素做人做事很让人舒服，与苏樱短短相处几日，便令苏樱觉得与她亲近。
<br/>　　程大雷倒抽一口凉气，这样小的年纪，便如此会做人。这要是再长几岁，那还了得。看苏樱提起杨小妹滔滔不绝的模样，明显是已经被这小姑娘忽悠住了。
<br/>　　程大雷暗暗觉得有些可怖了，未来这姑娘会行到哪一步，当真是不敢预料。
<br/>　　他又与苏樱聊了一阵，约过了半个时辰，程大雷感觉完成任务，起身与苏樱告辞，迈步离了屋子。
<br/>　　走出屋外，方才觉得浑身轻松，程大雷松了口气。这一关目前是过去了，可还有晚上，那同样是自己需要完成的任务。
<br/>　　举头望天，程大雷颇感无奈，这如果是现代，日子过成这个样子，两个人就该离了。
<br/>　　别了苏樱，程大雷又去找樊梨花聊了一阵，最近无有要紧之事，只能用这些事打发一下时光。
<br/>　　所有感受都是相互的，程大雷如此，苏樱何尝不知。程大雷走后，她回到闺房，颇感有些无趣。
<br/>　　“姐姐。”
<br/>　　一颗小脑袋透过门探进来，走进屋子时带进一阵香风。见到杨小妹时，苏樱脸上添了抹会心的笑容。
<br/>　　“你快过来，瞧瞧着一脑袋汗，定是贪玩胡闹去了。”
<br/>　　拽过来杨小妹，用手帕将她脸上的汗珠抹去，杨小妹一动不动，模样乖巧可爱。
<br/>　　毕竟也是会察言观色的人物，她低低的声音问：“姐姐，是大当家惹你不开心么？”
<br/>　　苏樱苦笑一声：“他那里会惹我不开心，他做事还是不错的。”
<br/>　　说到这里，苏樱心里有些无语。程大雷的确做事不坏，也不会三言两语便板起脸训人，总是客客气气，也算是温柔体贴。但问题便是太客气了，平日见不到程大雷，见到他便是吵架的机会都没有。
<br/>　　独守香闺，总是有几分冷清。苏樱一腔心思无人述说，见杨小妹虽然年幼，却也是通事明理，便与她唠叨了几句。
<br/>　　“姐姐为何不陪着大当家？”杨小妹道：“常在他身边走动，或许会好一些？”
<br/>　　苏樱仔细想想，觉得她的话也有道理，最后摆摆头：“我一个妇道人家，怎么明白他要做的大事，还是不敢打扰他做事的。”
<br/>　　杨小妹顿了顿，道：“或许姐姐就是太懂事了，心中想要什么，总是要说出口才好。”
<br/>　　苏樱一楞，忽有种一语点醒梦中人的感觉。自思量，她与程大雷之间也有番精力，当初她冷冷淡淡，程大雷倒是上赶着与自己说话。如今是越客气，反倒是越加生疏了。
<br/>　　苏樱皱起眉头，自己这位相公，平素做人做事就是有些奇怪，日子久了，却也算是习以为常。真莫非，他倒是一位敬酒不吃吃罚酒的主儿。
<br/>　　程大雷在后面陪着樊梨花，这位姑娘更加抑郁。她本是马上的女武将，被关进这城主府做了压寨夫人，心情自然不开心。于是话里话外便挑程大雷的不是，如此，程大雷倒是能与她多说几句。
<br/>　　正在此时，程大雷忽然打了几声喷嚏，揉揉鼻子，心中不知发生了怎样的事。
<br/>(本章完)
