　　“这小子真是富得流油，一住就是天级房间，就算是我们，都没有这个待遇。”
<br/>　　就在通天楼外，几名武者盯着楚风眠进入了通天楼，一个个眼神都是无比嫉妒。
<br/>　　“哼，这小子连一千万圣品灵石眼睛都不眨的就能拿出来，这点财富算什么。”
<br/>　　又一名武者盯着楚风眠，眼神之中露出了几分冰冷道。
<br/>　　“不过可惜，这小子居然敢去招惹少主，再多的财富，以后也是没命花了。”
<br/>　　“以少主的身份，要杀此人，哪里这么费事，依我看直接冲进去杀了他就是，就凭通天阁，也敢不给我们六道魔门的面子？”
<br/>　　又一名武者阴狠的开口道，有些不耐烦。
<br/>　　“闭嘴，少主的命令，你们都没听到吗？”
<br/>　　那看起来为首的一名六道魔门弟子，冷哼一声道。
<br/>　　“你们几个，盯紧了这小子，只要是这小子出来，就去告诉我，我先去禀告少主。”
<br/>　　说着，这名六道魔门弟子便是身躯一闪离开了。
<br/>　　而剩下的几名六道魔门弟子，还在通天楼外，盯着楚风眠。
<br/>　　归墟城内，一座名为万财楼的酒楼之内，一名年轻男子，正坐在主座上，慢慢的喝着酒。
<br/>　　他刚刚喝完，旁边的几名俏丽女子，便是急忙将酒杯倒满，随后放下酒壶，给那年轻男子捶腿。
<br/>　　“你们说那小子，进了通天楼？”
<br/>　　这年轻男子，正是那站魔宗的少宗主，纣绝阴。
<br/>　　他听着下方六道魔门弟子的汇报，嘴角露出几分冷笑道。
<br/>　　“看来这小子，是想要参悟那几件破东西的玄机？就那么几个破烂，这小子居然花费那么大的价格去买，简直是脑子进水了。”
<br/>　　“让他慢慢参悟，反正本少爷有的是时间跟他耗。”
<br/>　　纣绝阴冷笑一声。
<br/>　　突然看向旁边另外一名六道魔门弟子开口问道。
<br/>　　“那老家伙呢？”
<br/>　　“那老家伙径直离开了归墟城，由战狠长老亲自带人去抓了，应该很快就能够抓到了。”
<br/>　　那六道魔门弟子恭敬的回答道。
<br/>　　“哼，小家伙，敢不给我面子，等我抓了他，一定要好好的折磨他！”
<br/>　　纣绝阴眼神之中闪出几分阴狠，这纣绝阴心胸狭隘，只要是招惹过他的人，他都要一一报复回来。
<br/>　　那老者跟楚风眠，可都是当着众人的面，驳了他面子的人，这仇怨他可一直记得。
<br/>　　就在这交谈之际，一道身影，突然出现在了房间之内。
<br/>　　纣绝阴看到这到来的人影，笑着说道。
<br/>　　“原来是战狠长老回来了，快座，那老家伙抓住了吗？”
<br/>　　“老夫无能，让那老家伙给跑了。”
<br/>　　这战狠长老，就是一直站在纣绝阴身旁的老者，如今回来，直接跪在了地上。
<br/>　　“什么？跑了？”
<br/>　　纣绝阴一拍桌子，直接站了起来。
<br/>　　那老者的手上，可是有着足足一千万圣品灵石啊，整个站魔宗的宝库内，都没有一千万圣品灵石这么多。
<br/>　　纣绝阴都想好了得到这一千万圣品灵石之后要怎么花了，可现在居然跑了？
<br/>　　他盯着战狠长老，刚想要训斥，可最终还是压抑住了心中的怒火，冷哼说道。
<br/>　　“怎么回事？是有人抢先一步，把那老家伙抓走了？”
<br/>　　知道那老者身上有着大量财富的人，不止是他们一方，在纣绝阴看来，应该是有人，抢先一步了。
<br/>　　“不，那老者在离开了归墟城后，就突然间的消失了，就连我，都没有看出那老者是如何离开的。”
<br/>　　战狠长老回答道。
<br/>　　“突然消失了？就连你都看不出来？”
<br/>　　听到这话，纣绝阴的眼神之中露出几分惊讶。
<br/>　　他也不是傻子，眼前这战狠长老，可是一名七劫古帝，在战魔宗都都算是一位高手了。
<br/>　　所以他才会对着战狠长老，如此客气，可是那老者，可以在一名古帝的面前逃走？
<br/>　　“难道那老家伙，是什么大人物不成？”
<br/>　　纣绝阴目光闪烁，这事，他也不信战狠长老等人敢欺骗他。
<br/>　　毕竟盯着那老者的人，可不只是他们一方，如果欺骗，一查就可以查得出来。
<br/>　　这纣绝阴想了一下，还是对着战狠长老开口道。
<br/>　　“起来吧，看来这老家伙，没那么简单。”
<br/>　　纣绝阴突然想到。
<br/>　　“那老家伙卖的东西，难道真的是什么宝物？”
<br/>　　如果说那老者，真的是一名强者的话，那他卖的三件宝物，也许可是真的。
<br/>　　虽然纣绝阴也见过，可是纣绝阴也不敢保证，他可以看出其中的玄机来。
<br/>　　楚风眠既然是敢花一千万圣品灵石的天价，去买下这三件宝物，现在看来，也许是纣绝阴当时想错了。
<br/>　　这三件宝物，绝不会都是废物，只是其中的玄机，他没有看出。
<br/>　　纣绝阴急忙的对着一名六道魔门弟子追问道。
<br/>　　“那小子呢？还在通天楼里？
<br/>　　“回禀少主，是。”
<br/>　　那六道魔门弟子急忙回应道。
<br/>　　“走，我们也去通天楼，好好守着那小子去！”
<br/>　　纣绝阴眼神冰冷无比。
<br/>　　只要是杀了楚风眠，不管是楚风眠身上的财富，还是那三件宝物，都将是他的。
<br/>　　一群六道魔门弟子，在纣绝阴带领下，急忙离开，前往了通天楼。

<br/>　　楚风眠看了一眼，还是有些失望。
