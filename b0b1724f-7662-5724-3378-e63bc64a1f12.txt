　　第221章
<br/>　　拂尘轻卷，便挡住两柄铁枪，刘阿吉身子从御撵上跃下，以身体拦在御撵前。
<br/>　　“保护陛下！”
<br/>　　刘阿吉高喝，以柔克刚，以一敌二，莫看他是个太监，竟然丝毫不落下风。
<br/>　　长枪七叠劲，扎向刘阿吉胸口，刘阿吉身体倾倒，紧跟着另一柄枪扎向他的小腹。他的身体宛若提线木偶般，突然窜起，又躲过一道枪锋。
<br/>　　脚下如生清风，长袍轻扬，在两柄枪的攻击下，刘阿吉若是在刀尖上行走，步履轻盈，飘飘若仙。
<br/>　　战至半酣，刘阿吉飞身撤离战团，下令道：“活捉他们。”
<br/>　　程大雷此时已过了，手中宝剑出窍：“刘公公保护陛下，我来应付贼人。”
<br/>　　刘阿吉战到现在，其实已经有些应付不来，他看了一眼程大雷，却是没想到程大雷竟有如此勇气。
<br/>　　“好，靠你了。”
<br/>　　“贼人呐，受死吧。”
<br/>　　说时迟，那时快，两个刺客已经被禁卫军团团围住，纵然他们有插翅之能也难以逃脱。
<br/>　　程大雷此刻没有去刺杀杨龙停，而是迎向两个贼人。
<br/>　　关键是因为，他刚才已经认出，这两个刺客竟是林冲和林少羽。
<br/>　　的确二人脸上罩着黑纱，可看到他们的枪法程大雷已经觉得事情有些不对，又用系统查看了一下二人信息，顿时吓了一跳。
<br/>　　他们若是刺杀杨龙停，程大雷是很容易理解的，可这二人要杀的竟然是明帝。
<br/>　　这二人为什么要这么做，况且今天看到布防的兵力，程大雷的计划都取消了，这两个憨货来刺杀，岂不是来送死么。
<br/>　　看二人的劲头，怕真的有送死的想法。
<br/>　　“不要放箭，刘公公要活捉。”
<br/>　　程大雷冲了过去，口中大喝，蹂身避过林少羽的一枪。一年不见，林少羽的枪法变好了，这一枪当真好快。
<br/>　　程大雷以一敌二，不停向二人挤眼，那意思在说：走啊，走啊。
<br/>　　今日二人已被团团围住，有程大雷暗中保护或许能争得一线生机，可二人却是越打越急，非要杀了程大雷不可。
<br/>　　在二人的枪围中旋身，此刻身无彩凤双飞翼的本事使出，大家只见到满脸大胡子的程大雷从一地跃到一地，身形翩翩起舞，排除他长相这件事，真可称得上潇洒二字。
<br/>　　刘阿吉也怔了怔，这个莽汉的身手丝毫不弱于自己。
<br/>　　百里胜本以为程大雷上前就是送死，看到这一幕时一下睁大眼睛：他竟有如何好的身手！
<br/>　　程大雷是自己苦自己知道，身无彩凤双飞翼，心有灵犀一点通，这功夫是绝世级别的，以程大雷现在顶级的实力，撑不到太长时间。
<br/>　　若在打下去，被林冲或林少羽一枪捅了，死在自己兄弟手中，此生岂不太过悲催。
<br/>　　而林冲见一直杀不死程大雷，心中已经有些着急，再如何下去，非死在这里不可。
<br/>　　“昏君，去死吧！”
<br/>　　手中的铁枪突然投出，射向御撵。御撵前有不少禁卫，可看到这柄迅猛飞来的铁枪时，出于求生本能下意识躲开。而刘阿吉此刻也是鞭长莫及，救不得明帝。
<br/>　　突然，一道身影拦在御撵前，用掌硬生生的接住铁枪。
<br/>　　心有灵犀一点通。
<br/>　　特么的……力气真的好大啊！
<br/>　　所有人发出一声惊呼，随之冷汗哗哗直落。就连明帝也从御撵内探出个脑袋，看到程大雷立在面前，以后背冲着自己，手中紧紧握着那铁枪。
<br/>　　“狗贼，再不投降，我就要用五雷法炸你们了。”程大雷咆哮一声。
<br/>　　林少羽和林冲一怔，这台词好熟呐，再看看这眼睛，这鼻子，排除掉脸上的大胡子后，岂不就是，岂不就是……
<br/>　　“大……”
<br/>　　“嘿，狗贼，你们不要想逃，更不要想往东逃，让俺老牛生擒你们！”程大雷放下铁枪扑了过来，咬着牙比划着嘴型：走啊。
<br/>　　二人呆住，却是不知道程大雷为何会出现在这里，程大雷也不明白二人为何要刺杀明帝，此时此刻，先脱身再说。
<br/>　　双方一边假打一边往外溜，打得真可谓难分难解。
<br/>　　“莫要误伤了牛校尉！”刘阿吉高声喝道。
<br/>　　混战中，二人突破了重围，自然是向东逃去。
<br/>　　“狗贼，不要逃，再与我大战三百回合！”程大雷大吼。
<br/>　　百里胜将他扶起来，道：“牛校尉不要着急，大王子已经去追了，大王子的本事，你可能不知道，贼人绝对逃不了。”
<br/>　　程大雷看到李乐天已经上马，手中长剑一挥：“与我追！”
<br/>　　“呀，贼人不要逃啊！”
<br/>　　程大雷腾地跃起来，猛地将李乐天从马上撞下来，自己骑着马向前追去。
<br/>　　“大王子，大王子……”
<br/>　　有人慌张将李乐天扶起，李乐天被摔得鼻青脸肿，摇摇头道：“我没事，我没事……”
<br/>　　目光看着程大雷远去的方向，李乐天纳闷道：“牛校尉和贼人这么大仇么！”
<br/>　　林少羽和林冲向前驰马，程大雷紧紧缀在后面，再后面是追捕的禁卫军，碍于程大雷在中间隔着，后方的人无法射箭。
<br/>　　前方忽然出现一队牛群，呼嘞嘞跑过来，将双方隔开，卷起烟尘动天，等牛群散去的时候，前方已经没有了两个刺客的身影，当然，程大雷也消失不见了。
<br/>　　这牛群本来是程大雷给自己准备的，如今却是给林少羽二人用了。
<br/>　　一路奔跑，三人出现在深山中一座小庙内，把马拴在院内，三个人聚在大殿中。
<br/>　　“大当家，真的是你么？”林少羽现在还有些不敢相信：“大当家，你没死！”
<br/>　　“没死，没死，这不活着好好的么。”程大雷平复着呼吸。
<br/>　　林少羽过来重重击打着程大雷的肩膀，激动的热泪盈眶，林冲和程大雷关系远些，但毕竟是生死场中闯过来的兄弟，此时表情也很激动。
<br/>　　“轻点轻点。”程大雷揉着发疼的肩膀：“你们不是也没死么，你们是怎么回事，为何要刺杀明帝？”
<br/>　　林少羽和林冲对视一眼，二人脸上的表情渐渐变得凝重，林少羽缓缓开口。
<br/>　　“家父林问天。”
<br/>
