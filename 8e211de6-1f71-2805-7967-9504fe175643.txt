        &lt;h3&gt;第三卷第一百四十五章  家事&lt;/h3&gt;
<br/>
<br/>        潘青松睡了两天，张函来他都不知道。
<br/>
<br/>        大家知道潘青松很累，便也不打扰他，看着这么多人都没有吵醒他，可知潘青松有多辛苦，于是大家说话尽量压低声音。
<br/>
<br/>        兄弟姐妹们又因为陈子澜聚在了一起。
<br/>
<br/>        这一次，没有了冯睿，这一次大家也没有心情在一起吃饭，他们都坐在病房里悄悄的说着要陈子澜好好养身体的话。
<br/>
<br/>        他们相互鼓励，相互安慰，人生漫长，彼此都不容易，在时间的长河中，亲情将他们凝结在一起，他们是一个整体，却又是独立的个体，每一个个体随着时间的推移，随着社会的发展在做着不同的变化。
<br/>
<br/>        大姐陈子慧是最安稳的一家，日子过得平平谈谈、普普通通，不缺吃不少穿，比上不足比下有余。
<br/>
<br/>        二哥陈子建最是悲凉，原本是陈家骄傲的他，因为娶妻不淑，落得无家可归，好的是老家还有房子，令人欣慰的是，二哥虽然请了长假，但是二哥所著的书已经在他们业界掀起了轩然大，波。
<br/>
<br/>        二哥原本在业界就是名人，如今更是稳坐第一把椅子，也算是为他的事业出了一份力。
<br/>
<br/>        三姐陈子澜因为工作负伤，得在床上躺个一年半载的。
<br/>
<br/>        四姐陈子乔在西城顺风顺水，还是一如既往地掌握着经济大权。
<br/>
<br/>        五姐陈子悦已经是西省知名的企业家，王浩楠除了周末会到西城之外还是在他的原单位教书，五姐一直保持着她那要死不活的婚姻。
<br/>
<br/>        就像是五姐自己说的，王浩楠到西城了那就是孩子他爹，不到西城了他就是一个外人。
<br/>
<br/>        六哥陈子寒还是一如既往地缺席，不过陈子昂相信六哥一直在，而甸城的谷强就是她的六哥。
<br/>
<br/>        七哥陈子霖随着陈子昂到了北京，是陈子昂不可多得的助手，兄妹俩齐心协力。
<br/>
<br/>        陈子霖的土特产不但在网上有销售，在西城几个旅游景点都有门店，村里很多人都重新盖了新楼房，他们那一块因为移民搬迁，很多在山里的人都和他们住在了一起，陈子霖带动了更多人加入到他事业中来，他的销售已经形成了一套完整的体系。
<br/>
<br/>        父母亲身体不错，偶尔就是有个头疼脑热的，吃点药就好了。陈家兄弟姐妹的日子随着社会的发展也都好过起来了，相互间如果有个什么事情也都能帮衬着。
<br/>
<br/>        这次三姐负伤，虽然说有单位报销，但是兄弟姐妹们每人都拿出来好几万，五姐陈子悦早早的雇了俩个护工，让三姐住的是vip的病房。
<br/>
<br/>        原来连陈子昂上学的钱都拿不出来的人们，现在出行都是小汽车。
<br/>
<br/>        潘青松醒来的时候发现陈家兄妹站的站坐的坐，围了一屋子，他才惊觉他睡过头了。
<br/>
<br/>        “都来了啊。”潘青松打了个哈欠，他实在是太困了，回来将陈子澜交给陈子悦他倒头就睡。
<br/>
<br/>        “这几天辛苦你了。”陈子建看着胡子拉碴的潘青松说。
<br/>
<br/>        “只要老婆好好的，我辛苦点算什么。”潘青松走到陈子澜的床边替陈子澜拢了拢头发。
<br/>
<br/>        “我陪大家出去吃饭吧？”潘青松看着这么一屋子人呆在病房也不是个事儿。
<br/>
<br/>        “算了，我们都不吃饭了，还有事情，你好好陪着子澜，需要我们帮忙的就说。”陈子慧说。
<br/>
<br/>        子澜这个样子，大家也没有心思吃饭
<br/>
<br/>        大家听大姐这样说，便都说有事情，准备离开，潘青松最后留了二哥一小会，也不知道和二哥说了啥，二哥有些心不在焉。
<br/>
<br/>        大姐要回泉城，问二哥要不要一起走，二哥有点犹豫，陈子霖对二哥说：“你和大姐一起回去，你要是不想上班了，家里的事情你就多照应点，有你在家陪着爸妈，我在北京呆着也能放心，再说这边也没有你啥东西！”
<br/>
<br/>        四姐陈子乔责怪弟弟不应该这样和自己的哥哥说话：“子霖，你怎么能用这种口气和你二哥说话？”
<br/>
<br/>        陈子霖说：“难道你还要他和冯睿死灰复燃？冯睿那女人什么缺德的事情都能干得出来，说不定哪一天就遭了报应，二哥就不要再和她来往了。”
<br/>
<br/>        “再怎么说，她也是你的二嫂，给陈家也生了孩子。”陈子乔生气了。
<br/>
<br/>        “四姐，你都不知道冯睿对子昂做过的事情。”陈子霖想着反正二哥他们离婚了，说出来也不怕了。
<br/>
<br/>        “七哥。”陈子昂打断了陈子霖的话：“二哥想回去就回去，不想回去就不要勉强。”
<br/>
<br/>        “我和大姐一起走，你们几个保重。”陈子建咬咬牙，冯睿已经和他离婚了，虽然有了孩子，但是孩子现在也大了，他似乎也是心有余而力不足。
<br/>
<br/>        原来，他也劝过冯睿收手，可是冯睿不但不听，还要和他离婚，嫌弃他窝囊，他也算是尽力了吧。
<br/>
<br/>        现在的陈子建似乎是六根清净了。
<br/>
<br/>        潘青松也是叮咛他最近不要和冯睿有过多的牵扯，他本来于心不忍，想去看看冯睿，可是被自己的弟弟一顿骂有些抹不开面子，干脆就随着大姐一起回老家算了。
<br/>
<br/>        潘青松送走了大家，回头看着陈子澜，躺在床上的陈子澜一脸温柔。
<br/>
<br/>        “老婆，辛苦你了。”潘青松不知道该如何安慰自己的老婆，如果能调换身份，他希望能替她去遭那份罪。
<br/>
<br/>        陈子澜看了一眼护工。
<br/>
<br/>        潘青松对护工说:“大姐，我想和老婆说说悄悄话。”
<br/>
<br/>        护工微微一笑顺手带上们离开房间。
<br/>
<br/>        “老婆，你说。”潘青松坐在床边的凳子上，轻轻的按摩着陈子澜的手，老婆的一个眼神，他就知道老婆想什么。
<br/>
<br/>        “我和郑培去了一趟甸村，原本档案里记录着几年前那里被清洗过一次，再也没有不好的事情，但是我们发现了一帮孤儿，郑培闻到了那个院子里的味道异于别的地方。”陈子澜皱着眉头说。
<br/>
<br/>        潘青松：“你们去甸村有人知道吗？”
<br/>
<br/>        “就我们四个人知道。我们是临时找的司机。”陈子澜沉默了一下：“我们四个人中不会有问题。”
<br/>
<br/>        “按照这样的说法，你们这次确实是车祸。”潘青松心疼得不行，他可爱而活泼的老婆啊。
<br/>
<br/>        “子澜，现在你什么都不要想，养好身体是最重要的，工作的事情就放下吧，等你恢复了再说。”
<br/>
<br/>        “也只能如此了，对了，张函来找你了，被大家赶走了。”陈子澜笑着说“没有想到你和张函还有来往。”
<br/>
<br/>        “我们是工作上的事情，张函在我们圈子里是出名的六亲不认，对待工作一丝不苟，原本他现在的这个身份是不用亲自过问的，但是有些事情他还亲力亲为，说真的我还很佩服他。”潘青松难得的严肃起来。
<br/>
<br/>        “还有你佩服的人，真是难得。”陈子澜微微一笑并没有对张函评头论足。
<br/>
<br/>        岁月变迁，历经沧桑，人们都随着社会的发展在前进着，每个人的出身和所处的环境都是不一样的，我们能做的就是彼此尊重。
