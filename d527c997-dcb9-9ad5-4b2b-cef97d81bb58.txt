　　停下改装SUV后，罗昊拿着武器推开车门，猫着腰，绕过汽车，在李毅峰M134转管机枪的掩护下，朝着右侧那栋别墅推进过去。
<br/>　　“哒哒哒哒哒哒哒哒哒哒……”
<br/>　　M134转管机枪还在对着别墅倾泻着弹药，仿佛是要将这栋装修豪华的别墅扫成马蜂窝一般。
<br/>　　藏在别墅中的影子部队队员依旧死死地被压制着，不敢探出头来，更别提开枪反击。
<br/>　　“嗡……”
<br/>　　改装SUV上M134转管机枪突然停止了咆哮，子弹箱中的子弹打完了，六根通红的枪管还在快速转动着。
<br/>　　李毅峰松开M134转管机枪的发射键，摘下背后侧边的ACE-GAP39SB短突击步枪，推开车门离开了改装SUV。
<br/>　　别墅中前一刻还被M134转管机枪压制的透不过气来的影子部队队员，听到别墅外的枪声停止，没有子弹再飞进别墅，顿时松了一口气，将M4A1自动步枪对着别墅外打出一串串点射，子弹打在改装SUV上，发出“铿铿”的声响，防弹玻璃上被子弹打得多出了好几个白色的印痕。
<br/>　　靠在门口墙壁上的罗昊摸出一颗震爆弹，对李毅峰喊道：“闪光弹！”随后拔掉保险销，拆掉保险夹，朝着别墅内扔去，并且迅速撇开脑袋。
<br/>　　“哐！”
<br/>　　落在别墅客厅实木地板上的震爆弹发出刺眼的白光和能刺破人耳膜的噪音，强光和噪音散开的瞬间，罗昊对李毅峰使了个眼色，吼道：“上！”
<br/>　　两个人一前一后冲进别墅。
<br/>　　别墅中，受到震爆弹影响的影子部队队员还没有从刚才强光和噪音的影响中恢复过来，朦胧间看到有人冲进别墅，这些影子部队队员用力甩着脑袋，跌跌撞撞的寻找客厅中的家具作为掩体，试图进行反击；可是罗昊和李毅峰根本不会给他们反击的机会，急促而又带有节奏感的点射声响起，别墅客厅中的影子部队队员中弹倒地。
<br/>　　一名胸口中了两颗子弹，但是还没有断气的影子部队队员费力从手雷包中摸出一颗手雷，用牙齿咬住保险销上的圆环，看着客厅中的罗昊和李毅峰，露出沾满鲜血的牙齿，勾起一个狰狞的冷笑，然后拼尽最后的力气拉出了保险销，并且用力按压保险夹，使得保险夹弹出，手一松，沾着鲜血的手雷从他手中滚落在地板上，在客厅地板上“骨碌碌”的滚动着。
<br/>　　“注意手雷！”
<br/>　　罗昊看到在地板上滚动的手雷，顿时大惊失色，大吼一声，跟李毅峰两个人奋力一跃从只剩下边框的落地窗中跳了出去。
<br/>　　“轰！”
<br/>　　两个人刚从别墅客厅中跳出去，别墅中的手雷就爆炸开来，一团夹杂着火光的硝烟从别墅中喷涌出来，炙热的气浪朝着别墅外的两个人席卷而来，实木地板碎片，还用墙壁上掉落下来的水泥碎屑，面砖碎块掉落在两个人身上。
<br/>　　过了一会儿，罗昊才挣扎着从地上爬了起来，扭头看了眼身后被手雷炸得一片狼藉的别墅客厅，把李毅峰从地上拉起来，问道：“没事吧？”
<br/>　　“死不了！”李毅峰抖掉身上的东西，骂咧道：“妈的，差点就挂了。”
<br/>　　“后面我们得小心些。”罗昊对李毅峰叮嘱道：“从刚才的事情可以看出，这里的影子部队队员跟我们之前接触的不一样，他们不怕死，就算是死，也要拉上垫背的。”
<br/>　　李毅峰认同的点了点头，手中握着ACE-GAP39SB跟在罗昊身边那重新返回到别墅中，此时别墅客厅就像是被龙卷风席卷过一样，满目苍夷，满地狼藉，尤其脚下的实木地板更是像被牛拉着犁耙犁过一样，大片地板被手雷炸得翻飞出去，露出实木地板下的水泥地面。
<br/>　　别墅客厅中的影子部队队员此时已经没有了活口，他们绝对不可能逃过刚才那场爆炸。
<br/>　　罗昊和李毅峰对别墅进行搜索起来，罗昊去了别墅二楼和楼顶的直升机停机坪，李毅峰则是留在底楼客厅进行搜索。
<br/>　　二楼有很多房间，有些是卧室有些是健身室，罗昊把房间门一间一间踹开进行搜索，但是房间中并没有看到什么人，有些房间地板上还扔着比基尼和使用过的安全套。
<br/>　　罗昊走到走廊尽头最后一间房间门口，还没有把房门踹开，罗昊就闻到了从房间内飘散出来的血腥味，闻到血腥味后，罗昊瞬间变得谨慎起来，他伸手拧动了一下房门门锁，发现房间被锁住了，往后退了两步，罗昊抬脚就朝着房门踹去；可是出乎罗昊意料的是，他这一脚下去，居然没有能把房门给踹开！
<br/>　　而当罗昊一脚踹在房门上的时候，房间内很明显的有人活动的声音传来。
<br/>　　罗昊退后两步，举起G36C对准门锁的方向，朝着房间内的人员喊话道：“联合行动特种部队，里面是谁？三秒钟内不回答，直接击毙！”
<br/>　　隔了差不多两秒钟的时间，就在罗昊准备扣动扳机，准备隔着门板给里面的人员来一个扫射的时候，房间内传来了声音，“哥，是你吗？”
<br/>　　“小凡！？”
<br/>　　罗昊先是一惊，然后心中狂喜，对着房间内齐凡喊道：“小凡，里面除了你还有谁？”
<br/>　　“还有宿鸟、野牛、蜂鸟和几个‘雷鸣’特种部队的队员”齐凡回答道。
<br/>　　“退开门后三米！”
<br/>　　罗昊拔出手枪，对着上了锁的门锁就是两枪，将门锁打烂，然后一脚把房门踹开。
<br/>　　当房门被踹开的刹那，罗昊被自己眼前所看到的画面吃了一惊。这个面积在三十多平米的房间，就是一个刑房。房间没有窗户，唯一的光亮来自房间天花板上那盏白炽灯，四周墙壁上挂着各种各样的刑具。
<br/>　　齐凡、梅庆伟、林晓阳还有那几个“雷鸣”特种部队队员被关押在一道铁栅栏背后，他们身上或多或少都带着一些伤口，身上也是血迹斑斑，不过看他们的精神状态还不错，至少没有人重伤昏迷。
<br/>　　而在房顶上则是吊着一个被扒去了外衣和裤子，身上只剩下一条内裤的“雷鸣”特种部队队员。
