自从知道夏夕绾成了陆太太后，夏振国和李玉兰消沉阴霾了一段时间，现在夏妍妍金针封穴大爆，这两个人像是起死回生了，又跳了出来。
<br/>
<br/>        双双很不喜欢这两个人，所以当即不客气的出声道，“夏伯父，夏伯母，我看你们就是黄鼠狼给鸡拜年没安好心，今天是夏妍妍表彰的日子，你们去找夏妍妍就好了，你们拉着绾绾去接受电视台的采访是几个意思，你们想借机羞辱绾绾吧，我看你们欺人太甚了！”
<br/>
<br/>        李玉兰趾高气扬的哼了一声，“小姑娘，话可不能说，今天电视台的采访可是全网直播，我们好心邀请绾绾去露脸，毕竟绾绾刚从乡下回来，应该还没有被电视台采访过也没有机会享受这份荣誉吧，所以，你们就不要给脸不要脸了。”
<br/>
<br/>        双双冷冷的睨了李玉兰一眼，然后拉了一下夏夕绾的衣袖，“绾绾，我们不要听她放p，不要去！”
<br/>
<br/>        被放屁的李玉兰面色一变，“你！”
<br/>
<br/>        夏夕绾安抚的看了双双一眼，然后澄亮的眸子落在了李玉兰的脸上，她勾唇道，“你们真的邀请我去？”
<br/>
<br/>        “当然了，绾绾，你不会不敢去吧？”李玉兰竭力的想要夏夕绾过去，这也是夏妍妍的意思，所以她不惜用了激将法。
<br/>
<br/>        夏振国今天心情很好，他最骄傲最疼爱的女儿夏妍妍果然没有让他失望，这么快就翻身了。
<br/>
<br/>        金针封穴可是上古针法，夏振国一生学医，至今连金针都没能掌控，更别提传说里的金针封穴了，夏妍妍的医术这么高，果然是有天赋，还愁没有前程和未来么？
<br/>
<br/>        夏振国一锤定音道，“绾绾，你现在就跟我们一起过去接受采访吧。”
<br/>
<br/>        “好啊，”夏夕绾一口答应了，“我跟你们过去，不过，你们千万别后悔。”
<br/>
<br/>        看着夏夕绾意味深长的样子，李玉兰轻蔑的嗤了一声，一个乡下回来的废才，不知道用什么手段把陆寒霆给迷得团团转，她就以为自己能上天了？
<br/>
<br/>        她女儿可是夏妍妍，她就不信这个夏夕绾能在电视台采访的时候玩出什么花样来！
<br/>
<br/>        “绾绾，我怕他们欺负你，我跟你一起过去。”双双劝不住夏夕绾就要跟她一起过去。
<br/>
<br/>        夏夕绾点头，“好啊，一起去，请你免费看戏。”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾和双双来到了表彰大会，先召开表彰大会然后再在影棚里进行电视台的采访，阵容十分的强大，全部都是网络同步直播。
<br/>
<br/>        夏夕绾一眼就看到了人群中央的夏妍妍，今天的夏妍妍可是主角，她化着精致的裸妆，身上一件粉色连衣裙，婀娜娇柔。
<br/>
<br/>        大家将她簇拥在了中央，迫不及待的想对她进行采访，艳羡和赞美已经将她给包围了。
<br/>
<br/>        这时夏妍妍看到了夏夕绾，她当即提着裙摆走了过来，整个人就像是骄傲的小孔雀，按捺不住的兴奋和炫耀，“绾绾，你来啦？刚才我让爸妈去叫你，我还以为你不会来参加电视台的采访呢。”
<br/>
<br/>        夏夕绾勾起了红唇，“夏妍妍，你都这么诚心的邀请我了，我肯定不会让你失望的。”
