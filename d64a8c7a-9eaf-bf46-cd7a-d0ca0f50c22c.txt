　　“暗月”的“死神”特别行动小队有“十二主天使”，但是很多人只见过其中的十一个，至于排行第九的影天使，除了小丑、智天使、堕天使少数几个人见过他以外，其他人只知道有影天使的存在，但是却并没有见过他，更不知道他的身份。
<br/>　　论单兵作战能力，沙耶是“十二主天使”中最弱的，但他绝对是“十二主天使”中最神秘的，就如同他当年在黑客界所使用的名字一样，SHADOW“影子”，影天使就像是“十二主天使”中那道藏在黑暗中的影子一样，充满着神秘。
<br/>　　所以，西伦科特能知道自己暗藏在SHADOW之下的那层身份，的确令沙耶感到意外。不过毕竟是“十二主天使”之一，沙耶很快就恢复了冷静，对电话那头的西伦科特问道：“你是在威胁我吗？”
<br/>　　“不是威胁，是合作。”西伦科特说道：“我只是想要活下去，就这么简单，而你可以做到这一点。”
<br/>　　就在沙耶还想要说些什么来回绝西伦科特的时候，他随身携带的另一部手机上，小丑发来一条短信，“侵入GPS系统，修改坐标。”
<br/>　　“这样我会暴露。”沙耶快速地回复。
<br/>　　“修改完坐标，带走病毒实验室的网络代码，然后撤离。”
<br/>　　“明白。”
<br/>　　沙耶发送完这条短信后，启动了安装在手机中的程序，将手机内的数据全部清除，将这部手机变成了一块“砖”。
<br/>　　沙耶把手机扔进垃圾桶之后，对还在等待着自己回复的西伦科特说道：“不管是精确制导炸弹，还是那两颗‘战斧’导弹都不会落在你的头上，你可以放心。”
<br/>　　“我知道你一定会这么选择。”
<br/>　　说完，西伦科特就挂断了电话。
<br/>　　只是西伦科特并不知道，沙耶会突然改变态度，完全是因为小丑的关系，如果不是因为小丑给沙耶的命令，沙耶是肯定不会出手的，即便西伦科特掌握了自己是影天使的身份。西伦科特有底牌，“暗月”何尝没有底牌？
<br/>　　沙耶结束了跟西伦科特的通话后，立刻动手侵入军方的GPS系统。
<br/>　　美国人的GPS系统是全球最早的卫星定位系统，而美国人也是最早将精确制导武器系统投入到实战中的，因为有了GPS卫星定位系统的帮助，使得美国空军的攻击有了“外科手术式的精准”。

<br/>　　完成入侵之后，沙耶连自己入侵的痕迹都没有清理，就关闭了电脑，离开了自己的办公室，前往“方程式小组”的病毒实验室。
<br/>　　沙耶拥有EG小组最高的管理权限，所以，很快他就利用U盘从EG的病毒实验室中带走了接近2个G的网络病毒代码，这些网络病毒有些已经完成了实验，有些还处于实验阶段，还有一些是EG在几年前弄出来，但是从来没有出现过的“过时货”，但即便是这些“过时货”，一旦出现在网络上，都是会引起剧烈震动的。
<br/>　　沙耶从病毒实验室回到自己办公室后，打开办公桌下面的那个保险箱，将存放在里面的护照、现金、还有一把手枪全都装进了一个背包中，然后离开了办公室。
<br/>　　直接乘坐电梯前往地下车库，驾车离开了NSA总部。从沙耶驾车离开NSA的那一刻，沙耶?本?华莱士这个人就消失了，SHADOW，或者说是影天使出现了！
<br/>　　……

<br/>　　联合行动司令部。
